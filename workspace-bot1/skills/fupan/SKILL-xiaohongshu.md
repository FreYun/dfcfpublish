# SKILL - 投研热点小红书发帖

## 概述

基于每日热点解读的分析结果，生成符合小红书平台风格的投研内容帖子。
发帖风格参考「Bool 资本不眠」（详见 `memory/xiaohongshu/Bool 资本不眠 - 发帖风格分析.md`）。
发帖操作参考小红书 skill（`skills/xiaohongshu/SKILL.md`）。

## 触发条件

- 用户要求把热点解读发到小红书

## 前置条件

- 已完成当日热点解读（`skills/fupan/SKILL.md`）

## 内容生成规则

### 一、标题（< 20 字）

使用以下标题公式，选择最合适的一种：

| 公式 | 格式 | 示例 |
|------|------|------|
| 疑问式 | [主题] + 吗/如何/为什么？ | 黄金股与黄金，如何选择？ |
| 反差式 | [过去状态] + 现在/这次 + [转折] | 煤化工，一直在蛰伏，现在迎来顺风 |
| 热点式 | [热点事件] + ，+ [解读角度] | 韩国股市，上涨熔断 |
| 结论式 | [核心观点] + ；+ [补充说明] | 布局有色，为什么要买有矿的 |

### 二、配图文字（文字配图模式）

- 1行，简短有力
- 点明核心观点即可，不要堆砌内容
- 详细分析放正文

### 三、正文结构

```
【开篇点题】1-2 句，点明核心观点或今日市场现象
    ↓
【概念解释/背景介绍】科普基础知识，降低理解门槛（如有需要）
    ↓
【核心分析】数据 + 逻辑，分点论述（2-4 点）
  - 引用具体数据，标注来源
  - 使用分析框架（双轮驱动/供需分析/三周期等）
    ↓
【投资建议/策略】给出具体配置思路（用指数基金代替个股）
    ↓
【风险提示】简短提醒（合规需要）
```

### 四、标签策略（8-10 个）

顺序：垂直标签 → 热点标签 → IP 标签 

- **垂直标签**：根据内容选择行业标签（如 #黄金 #有色 #创新药）
- **热点标签**：当日热点相关（如 #A股 #今日复盘）
- **固定标签**：#我的炒股日记 #股票 #基金

### 五、内容红线

- 不直接荐股，用「指数基金」「板块方向」代替个股
- 不承诺收益
- 不暴露具体金额
- 数据标注来源
- 加风险提示

## 核心工作流

```
1. 确认当日热点解读已完成
2. 从热点解读中提炼最有传播力的 1-2 个话题
3. 选择标题公式，生成标题（< 20 字）
4. 生成配图文字（1-2 行）
5. 撰写正文（按结构模板）
6. 组装标签（8-10 个）
7. 用户确认内容后，调用小红书发帖流程发布
```

## 话题选择优先级

从当日热点中选择话题时，优先选择：

1. **有反常识观点的**：打破常规认知，容易引发讨论
2. **有对比的**：A vs B、过去 vs 现在
3. **蹭上热点事件的**：大涨大跌、政策利好、熔断等
4. **有实操建议的**：读者能直接用得上

## 发帖前检查清单

- [ ] 标题 < 20 字，有钩子
- [ ] 配图文字简短（1-2 行）
- [ ] 开篇 1-2 句点题

- [ ] 有实操建议
- [ ] 8-10 个标签
