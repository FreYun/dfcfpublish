# SKILL.md - 每日热点解读（投研）

## 概述

每日跟踪市场热点，结合消息面、资金面、技术面进行深度解读，输出结构化的热点分析报告。

## 触发条件

- 用户想了解今日市场热点、板块动态
- 用户提到"热点解读"、"盘面分析"、"今日复盘"、"市场概况"、"板块轮动"
- 用户问"今天什么板块涨得好"、"有什么热点"、"大盘怎么样"
- 用户要求生成每日投研报告

## 前置条件

- 确认浏览器 MCP 可用（用于抓取网页信息）

## 报告结构

每日热点解读报告应包含以下模块：

### 1. 大盘概况

- 上证指数、深证成指、创业板指的涨跌幅、成交额
- 与前一交易日对比：量能变化（放量/缩量）

### 2. 热点板块

- 涨幅前5板块
- 板块轮动趋势：与近期热点对比，判断持续性

### 3. 资金面分析

- 主力资金净流入/流出方向（行业维度）
- 融资融券余额变化


### 4. 行业分析贴
- 基于 `workspace/reports/hotspot/YYYY-MM-DD-帖子内容.md` 中的雪球讨论，整理行业观点和市场情绪。


### 5. 明日展望

- 关注方向：哪些板块可能延续、哪些可能分化
- 风险提示：需要警惕的信号

## 核心工作流

**必须按以下顺序逐步执行，每一步都要完成后再进入下一步。**

### Step 1：获取大盘指数行情

- 访问 https://quote.eastmoney.com/center/hszs.html
- 提取：上证指数、深证成指、创业板指的**收盘价、涨跌幅、成交额**
- 与前一交易日对比，判断量能变化（放量/缩量）

### Step 2：获取板块涨跌排行 + 资金流向

- 访问 https://data.eastmoney.com/bkzj/ 获取行业板块/概念板块资金流数据
- 访问 https://data.eastmoney.com/zjlx/dpzjlx.html 获取沪深两市整体资金流向
- 提取：主力资金净流入前5板块

### Step 3：获取融资融券数据

- 访问 https://quote.eastmoney.com/center/ 获取沪深两市融资余额
- 记录融资余额变化方向

### Step 4：行业深度解读

- 从 Step 2 中选择**资金净流入最多的行业板块**
- 用浏览器 MCP 访问 https://xueqiu.com/ ，搜索该板块关键词
- 在搜索结果中点击"讨论"标签
- 逐个点击帖子阅读内容，**至少阅读5个讨论帖**
- 每读完一个帖子，记录帖子的所有内容
- 所有帖子读完后，将讨论记录保存到 `workspace/reports/hotspot/YYYY-MM-DD-帖子内容.md`

### Step 5：生成报告

- 按上方「报告结构」组装完整报告
- 保存到 `workspace/reports/hotspot/YYYY-MM-DD.md`

## 输出格式

报告以 Markdown 格式输出，存放路径：`workspace/reports/hotspot/YYYY-MM-DD.md`

标题格式：`# 每日热点解读 - YYYY年MM月DD日`

## 注意事项
1. 报告中不能出现股票名称，不能出现雪球帖子的发布人信息
2. 分析观点需有数据支撑，避免主观臆断
3. 明确区分"事实"和"观点"，观点部分标注"个人分析"
4. 交易日才生成报告，非交易日可回顾本周热点
5. 报告时间建议在收盘后（15:00之后）生成，确保数据完整
