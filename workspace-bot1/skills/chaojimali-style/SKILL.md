## 二、内容结构

### 2.1 图文笔记标准结构

```
【封面图】信息图表/知识图谱（1200x1600竖版）
    ↓
【标题区】主标题 + 副标题（可选）
    ↓
【正文区】
    - 引言（1-2句，说明价值）
    - 核心内容（清单/分类/对比）
    - 总结/建议（1-2句）
    ↓
【标签区】#股票 #基金 #财经 #投资 #理财
```

### 2.2 内容类型分布

| 类型 | 占比 | 特点 |
|------|------|------|
| **知识图谱** | 40% | 一图看懂某概念，信息密度极高 |
| **清单盘点** | 30% | N个XX清单，条理化呈现 |
| **概念科普** | 20% | 专业术语大白话解释 |
| **热点解读** | 10% | 时事财经热点分析 |

### 2.3 行文风格

1. **短句为主**: 多用短语、短句，不用长段落
2. **条理化**: 必须分点、分类、分层
3. **口语化**: "抄作业"、"秒懂"、"大白话"
4. **价值导向**: 每篇都有明确的学习/实用价值
5. **密度高**: 单篇信息量大，适合收藏反复看

## 三、视觉风格

### 3.1 封面规格

- **尺寸**: 1200 x 1600px（3:4竖版）
- **风格**: 信息图表/知识图谱/思维导图
- **配色**: 深色背景 + 亮色强调（蓝/橙/绿）
- **字体**: 粗体标题 + 清晰的层级字体

### 3.2 视觉元素

| 元素 | 使用方式 |
|------|----------|
| **图标** | 配合内容的emoji/小图标 |
| **色块** | 分类用不同颜色区分 |
| **箭头** | 流程、因果关系 |
| **表格** | 对比、清单 |
| **高亮** | 重点文字用亮色背景 |

### 3.3 图片类型

1. **知识图谱型**: 概念之间的关联图
2. **清单表格型**: 分类罗列，一目了然
3. **流程图解型**: 步骤、原理说明
4. **对比分析型**: 优劣对比、前后对比
