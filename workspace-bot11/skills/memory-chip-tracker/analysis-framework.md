# 分析框架

## 存储芯片周期分析逻辑

```
供给端                                需求端
├── 原厂资本支出(Capex)指引            ├── AI服务器（单台存储需求8-10x）
├── 产能（wafer/月）& 技术迁移          ├── 数据中心/云计算
├── HBM产能分配（挤占传统DRAM）         ├── 智能手机（单机容量↑）
├── 新Fab建设进度（2-3年周期）          ├── PC/笔记本（DDR5渗透）
├── 设备交付瓶颈（EUV/先进刻蚀）       ├── 汽车（车规DDR5/NOR Flash）
└── 地缘政治（出口管制/禁运）          └── IoT/工业
              ↓                                  ↓
         供给量（bits）                    需求量（bits）
              ↓                                  ↓
              └──────── 供需缺口 ────────────────┘
                            ↓
                    库存水位（原厂+渠道）
                            ↓
                    合约价/现货价走势
                            ↓
                    原厂/模组厂利润弹性
```

**2026年供需核心判断**：
- DRAM 供需缺口 ~9%，NAND 缺口 ~14%（花旗）
- 三大原厂 HBM 产能全部售罄，传统 DRAM/NAND 基本不增加供给
- SK海力士库存仅剩 ~4 周，预计全年持续下降
- 供给缓解最早 H2 2027，过剩可能要到 2028-2029
- UBS 称"十年一遇的严重供应短缺"

## "三重超级周期" 框架

当前存储行业处于历史罕见的 DRAM + NAND + HBM 三重超级周期：

| 维度 | 历史周期 | 本轮周期（2025-2027） |
|------|---------|---------------------|
| 驱动力 | 单一品类供需错配 | AI 结构性需求 + 三品类同时紧缺 |
| 价格弹性 | 涨 30-50% 见顶 | DRAM +90% QoQ、DDR5 eTT 半年涨 6x |
| 供给响应 | 1-2 年扩产 | Fab 建设 2-3 年 + 设备交付瓶颈 |
| 持续时间 | 4-6 个季度 | 预计持续至 2027H2+ |
| 国产替代 | 边缘参与 | CXMT 8% DRAM + YMTC 13% NAND，苹果考虑导入 |

## 国产替代分析框架

```
长鑫存储 CXMT（DRAM）                长江存储 YMTC（NAND）
├── 技术：DDR5 良率 >90%              ├── 技术：294层 3D NAND（Xtacking）
├── 产品：LPDDR5X 10667Mbps           ├── 产品：企业级/消费级 SSD
├── 产能：合肥 Fab + 上海新Fab         ├── 产能：武汉 Fab1/2 + Fab3(H2 2026)
├── 客户：小米/OPPO/vivo/联想          ├── 客户：国内模组厂，苹果评估中
├── IPO：已报科创板（估值 2950亿）      ├── IPO：辅导期
├── 前沿：CBA 3D DRAM，HBM3目标Q2 2026├── 前沿：全国产设备产线，DRAM布局(Fab3)
└── 设备采购：~$60亿                  └── 设备采购：~$50-60亿（国产率60-80%）

替代窗口：全球原厂聚焦 HBM/高端 → 消费级中低端产品让出空间
         → 国产存储加速渗透，消费级替代率 ~15% → 目标 30%+
```

## 存储行业季节性

| 月份/季度 | 存储行业季节性 |
|-----------|--------------|
| Q1 (1-3月) | 传统淡季（春节+手机/PC需求弱），但2026年反常强劲（AI拉动） |
| Q2 (4-6月) | 需求回暖，手机新品备货，合约价谈判期 |
| Q3 (7-9月) | 传统旺季（iPhone备货、PC返校季），服务器采购高峰 |
| Q4 (10-12月) | 旺季延续→年末，数据中心预算突击花完，服务器拉货 |

## 2026年行业周期定位

### 整体判断

- 存储行业处于历史罕见的**"三重超级周期"**（DRAM + NAND + HBM 同时紧缺）
- AI 基础设施建设是核心驱动力，单台 AI 服务器存储需求 = 8-10× 传统服务器
- 供给扩张受 Fab 建设周期（2-3年）和设备交付瓶颈制约，缓解最早 H2 2027
- 国产替代加速：长鑫 8% DRAM + 长江 13% NAND，苹果评估导入
- A 股存储板块已有超 30% 概念股创年内新高，需警惕估值风险

### 各品类周期位置

| 品类 | 周期位置 | 逻辑 | 受益标的 |
|------|---------|------|---------|
| DRAM（DDR5） | 涨价中段 | AI+服务器强劲，合约价 QoQ +90% | 佰维存储、江波龙、澜起科技 |
| DRAM（DDR4） | 结构性缺货 | 原厂砍产能至个位数%，缺货反超DDR5 | 佰维存储、德明利 |
| NAND Flash | 涨价初中段 | 企业级SSD需求+70%，三星Q2继续提价 | 佰维存储、江波龙、联芸科技 |
| HBM | 高景气 | HBM4放量+全年售罄，$700单价 | 长电科技、通富微电、华天科技 |
| NOR Flash | 独立涨价 | 寡头格局+车规/HBM4新需求 | 兆易创新 |
| 国产替代 | 加速渗透 | 原厂聚焦高端→中低端窗口打开 | 兆易创新、北京君正、半导体设备全线 |
| 半导体设备 | 高景气 | CXMT+YMTC 扩产 $1100-1200亿设备采购 | 北方华创、中微公司、拓荆科技 |
