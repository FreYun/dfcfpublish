> 发布前必须阅读 `quality-checklist.md`

---

## 每日快讯（飞书/小红书）

**触发**：存储芯片价格显著异动，或核心事件发生

**标题模板**：
- `💾 DRAM暴涨90%！存储超级周期来了`
- `⚡ SK海力士：库存只剩4周，HBM全年售罄`
- `🏭 长鑫存储要上科创板了！估值2950亿`
- `📈 佰维存储利润暴涨10倍！存储涨价有多猛`

**内容结构**（200-300字）：
```
[开头] 存储芯片圈今天最值得关注的1件事
[数据] 关键价格/事件（不超过3个数据点）
[逻辑] 为什么？背后的供需逻辑
[关联] 对产业链上下游的影响
[展望] 短期怎么看
[互动] 引导评论
```

**Tags**：`#存储芯片` `#半导体` `#DRAM` `#HBM` `#{具体品种/公司}` `#投资` `#AI`

## 周报（公众号文章）

**结构**：
```markdown
## 本周概览

### 💾 DRAM
- 合约价：QoQ +XX%，现货周变化 ±X.X%
- [核心驱动：AI需求/HBM挤占/DDR4缺货]

### 💿 NAND Flash
- 合约价：QoQ +XX%
- [企业级vs消费级分化]

### 🔥 HBM
- HBM4 量产进展 / 产能分配 / 价格

### 🇨🇳 国产替代
- 长鑫存储 / 长江存储最新进展

### 📈 A股存储板块
- 板块周涨跌：±X.X%
- 核心标的表现：[TOP3涨/跌]

### 🔮 下周关注
- [原厂财报/涨价函/技术发布]
- [催化事件]
```

## 品类深度（公众号文章）

**触发**：品类出现重大变化，或积累了足够的跟踪数据

**结构**：
```markdown
# [品类]全解：当前在哪，未来怎么走

## 是什么（一句话说清楚）
## 产业链位置
## 供给分析（原厂产能/技术节点/扩产计划）
## 需求分析（下游结构/AI驱动/单机搭载量）
## 价格走势与库存
## 竞争格局（全球CR3 + 国产替代率）
## 核心矛盾
## 观点与展望
## 风险提示
```
