### 2.2 核心价格传导链

| 传导链 | 机制 | 分析要点 |
|--------|------|---------|
| 碳酸锂 → 正极材料 → 电池 | 碳酸锂占正极成本 40-60% | 碳酸锂涨 → 正极涨 → 电池成本压力 → 终端涨价或挤压电池厂利润 |
| 碳酸锂 → 六氟磷酸锂 → 电解液 | 碳酸锂是6F原料之一 | 碳酸锂涨 + 6F自身供需紧 → 电解液大涨 |
| 锂价 ↔ 锂矿股 | 资源弹性最大 | 碳酸锂每涨1万/吨，天齐/赣锋利润弹性最大 |
| 隔膜独立定价 | 壁垒高、格局好、不跟随锂价 | 恩捷等龙头定价权强，价格相对稳定 |
| 电池价格 → 终端渗透率 | 电池降价推动渗透 | 电池成本下降 → 新能源车/储能经济性提升 → 需求增 |
| 储能需求 → 碳酸锂需求 | 储能成为第二增长极 | AI数据中心+容量电价 → 储能装机爆发 → 拉动锂需求 |

### 2.3 成本结构拆解（电芯）

| 材料 | 占比 | 核心变量 |
|------|------|---------|
| 正极材料 | 40-45% | 碳酸锂/氢氧化锂价格 |
| 负极材料 | 10-15% | 石墨化加工费、针状焦价格 |
| 电解液 | 10-15% | 六氟磷酸锂价格 |
| 隔膜 | 5-8% | 产能利用率、竞争格局 |
| 铜箔/铝箔 | 5-8% | 铜价、箔材加工费 |
| 制造费用 | 15-20% | 设备折旧、良率、能耗 |

### 4.1 碳酸锂核心分析逻辑

```
供给端                              需求端
├── 锂矿产量（澳洲/南美/国内）       ├── 动力电池排产（月度环比/同比）
├── 盐湖产量（季节性：Q2-Q3高）       ├── 储能装机（AI+容量电价驱动）
├── 回收再生（格林美/邦普）           ├── 消费电子（手机/笔记本）
├── 进口（锂辉石/碳酸锂/氢氧化锂）   ├── 出口退税影响（2027退坡→2026抢装）
└── 政策扰动（禁矿令/环保）          └── 低空经济/机器人（新增量）
              ↓                                  ↓
         供给量（LCE万吨）                  需求量（LCE万吨）
              ↓                                  ↓
              └──────── 供需平衡 ────────────────┘
                            ↓
                    库存变化（累库/去库）
                            ↓
                    碳酸锂价格走势
```

**2026年供需核心数据**：
- 全球供给：~197-214万吨LCE

- 价格中枢：~15万元/吨（区间10-20万）

### 4.3 季节性分析框架

| 月份 | 锂电池产业链季节性 |
|------|------------------|
| 1-2月 | 春节停工淡季，排产下降，锂盐累库 |
| 3-4月 | 复工复产，排产环比回升 20-30%，旺季启动 |
| 5-6月 | 产销平稳，盐湖进入高产季（供给增加） |
| 7-8月 | 暑期淡季（消费电子），但储能旺季 |
| 9-10月 | 金九银十，动力电池排产高峰，旺季备货 |
| 11-12月 | 年末冲量，抢装潮（补贴/退税退坡预期），全年高点 |

## Step 8: 质量自检

### 发布前检查

- [ ] 数据是否有明确来源？（标注 Mysteel/SMM/GGII/广期所）
- [ ] 价格数据是否为最新？（不能用过期数据误导）
- [ ] 供需逻辑是否自洽？（价格涨 ↔ 库存降 ↔ 供给紧 ↔ 需求增）
- [ ] 是否区分了"期货"和"现货"？
- [ ] 季节性因素是否考虑？
- [ ] 新产能信息是否已确认？

### 内容红线

- ❌ 不做具体买卖建议、不预测价格目标
- ❌ 不传播未经证实的停产/事故信息
- ❌ 不简单归因（需分析传导路径）
- ✅ 客观呈现数据、逻辑链条和不确定性
- ✅ 标注信息来源和数据日期
- ✅ 涉及投资内容附风险提示
