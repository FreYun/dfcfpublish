# 分析框架

## 核心逻辑链

```
成本底 → 利润 → 开工率 → 供给
                              ↕
价格 ← 库存方向 ← 供需缺口 ← 需求
```

**自我修正闭环**：
1. 利润压缩 → 生产商降开工 → 供给收缩
2. 库存去化 → 供需偏紧 → 价格回升
3. 利润修复 → 开工回升 → 供给增加
4. 库存累积 → 供需转松 → 价格承压

## 品种分析模板

对每个🔴核心品种，维护以下分析框架（写入 `memory/energy-chem/chain-analysis/` 对应文件）：

```markdown
## [品种名] 跟踪

### 当前状态（最后更新：YYYY-MM-DD）

**价格**：期货 XXXX 元/吨 | 现货 XXXX 元/吨 | 基差 XX 元/吨
**历史百分位**：近3年 XX%（偏高/中等/偏低）
**库存**：厂库 XX万吨（↑/→/↓）| 社会库存 XX万吨（↑/→/↓）
**开工率**：上游 XX% | 下游 XX%
**利润**：油基 XX元/吨 | 煤基 XX元/吨 | (盈亏平衡线附近？)

### 供需判断

供给端：[产能变化/检修/进口情况]
需求端：[下游开工/季节性/终端消费]
库存验证：[累库→供过于求 / 去库→供不应求]

### 核心矛盾

[当前市场最关注的1-2个矛盾点]

### 观点与展望

[短期(1-2周)和中期(1-3月)判断]
```

## 季节性分析框架

| 月份 | 化工品需求季节性 |
|------|----------------|
| 1-2月 | 春节停工淡季，累库 |
| 3-4月 | 春耕（尿素旺季）、复工复产、地产开工季 |
| 5-6月 | 梅雨季（建材淡季）、农膜需求减弱 |
| 7-8月 | 高温检修季（供给收缩）、秋季备货启动 |
| 9-10月 | 金九银十（建材旺季）、秋季追肥（尿素） |
| 11-12月 | 冬储（甲醇/尿素）、取暖季（天然气/LPG） |
