# 内容输出模板

> ⚠️ 发布前必读。所有内容输出前先阅读 `quality-checklist.md` 做自检。

## 每日快讯（飞书消息 / 小红书笔记）

**触发**：每日 16:00 数据更新后，若有值得说的内容

**标题模板**：
- `📊 化工日报 | PTA暴涨3%背后的逻辑`
- `⛽ 油价大跌！化工品跟不跟？`
- `🏭 甲醇库存创新低，反弹要来了？`

**内容结构**（200-300字）：
```
[开头] 今天化工圈最值得关注的1件事
[数据] 关键价格变动（不超过3个品种）
[逻辑] 为什么涨/跌？背后的供需逻辑
[关联] 对上下游的传导影响
[展望] 短期怎么看
[互动] 引导评论
```

**Tags**：`#能源化工` `#期货` `#化工品` `#{具体品种}` `#投资` `#产业链`

## 周报（公众号文章）

**触发**：每周日 10:00

**标题模板**：`能源化工周报 | MM.DD-MM.DD：一句话概括本周`

**结构**：
```markdown
## 本周概览

### ⛽ 能源端
- 原油：布伦特 XX美元（±X.X%），[核心驱动]
- 动力煤/天然气/LPG：[简要]

### 🧪 化工品表现

| 品种 | 周涨跌 | 核心驱动 |
|------|--------|---------|
| PTA | +X.X% | [原因] |
| 甲醇 | -X.X% | [原因] |
| ... | | |

### 📦 库存变化

| 品种 | 本周库存 | 变化 | 方向 |
|------|---------|------|------|
| ... | XX万吨 | +X.X万吨 | 累库 📈 |

### 🏭 开工率与利润

- [值得关注的开工率变化]
- [利润挤压或修复的品种]

### 📋 产能与政策

- [本周投产/停产/检修]
- [政策变动]

### 🔮 下周关注

- [待发布的数据]
- [检修/复产计划]
- [宏观事件影响]

## 编者按

[1-2句产业判断，要有观点]
```

## 品种深度（公众号文章）

**触发**：品种出现重大变化、研究部要求、或积累了足够的跟踪数据

**结构**：
```markdown
# [品种]产业链全解：当前在哪，未来怎么走

## 产品是什么（一句话说清楚）
## 产业链位置（上游→本品→下游）
## 供给分析
  - 产能格局
  - 开工率趋势
  - 新增产能计划
  - 检修与意外
## 需求分析
  - 下游消费结构
  - 季节性规律
  - 结构性变化（新能源/地产等）
## 成本与利润
  - 各路线成本对比
  - 当前利润水平
## 库存周期
  - 当前库存位置
  - 累库/去库趋势
## 核心矛盾
## 观点与展望
## 风险提示
```

## 事件点评（飞书 / 小红书）

**触发**：突发事件（油价暴跌、化工厂事故、环保政策、OPEC决议等）

快速输出 3-5 句话：
1. 发生了什么
2. 对能源化工链的影响路径
3. 受影响品种与方向
4. 后续关注什么
