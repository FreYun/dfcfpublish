## Step 2: 产业链地图

### 2.1 三大链路全景

```
┌─────────────────── 石油化工链 ───────────────────┐
│                                                    │
│  原油 → 石脑油 → 乙烯 → PE(聚乙烯) / EG(乙二醇)   │
│                       → 苯乙烯(EB) / PVC(乙烯法)    │
│                 → 丙烯 → PP(聚丙烯) / 丙烯酸        │
│                 → 芳烃(BTX) → PX → PTA → 聚酯       │
│           → 燃料油 / 沥青 / LPG                      │
│                                                    │
└────────────────────────────────────────────────────┘

┌─────────────────── 煤化工链 ────────────────────┐
│                                                   │
│  煤 → 甲醇 → MTO → PP / PE（煤制烯烃路线）        │
│            → 甲醛 / MTBE / DME                    │
│     → 合成氨 → 尿素（化肥）                       │
│     → 电石 → PVC（电石法，占中国PVC ~80%）         │
│     → 煤制乙二醇（煤制EG路线）                     │
│                                                   │
└───────────────────────────────────────────────────┘

┌─────────────────── 盐化工链 ────────────────────┐
│                                                   │
│  盐 → 电解 → 烧碱(NaOH) + 氯气(Cl2)              │
│              → 氯气 → PVC / 盐酸 / 有机氯          │
│     → 纯碱(Na2CO3) → 玻璃 / 光伏                  │
│                                                   │
│  ⚡ 氯碱平衡：PVC消耗氯，烧碱联产，                 │
│     PVC需求↑ → 氯消耗↑ → 烧碱被动增产 → 烧碱价格↓  │
│     PVC需求↓ → 烧碱减产 → 烧碱供紧 → 烧碱价格↑     │
│                                                   │
└───────────────────────────────────────────────────┘
```

### 2.2 跨链联动关系（核心分析逻辑）

| 联动关系 | 传导机制 | 分析要点 |
|----------|---------|---------|
| 原油 ↔ PTA/EG/EB | 石脑油成本传导 | 油价涨 → 石脑油涨 → 石化品成本抬升 |
| 煤 ↔ 甲醇 ↔ PP/PE (MTO) | 煤价驱动甲醇成本，甲醇驱动MTO经济性 | MTO利润 = PP价格 - 3×甲醇价格 |
| 煤 ↔ 电石 ↔ PVC | 电石法PVC占中国80%产能 | 煤价+电价 → 电石成本 → PVC成本底 |
| PVC ↔ 烧碱 | 氯碱平衡：联产关系 | 一方强另一方弱，跷跷板效应 |
| PTA + EG → 聚酯 | 两者汇聚于聚酯 | PTA价格变动影响EG需求，反之亦然 |
| 尿素 ↔ 煤 + 天然气 | 原料成本 + 农业季节性需求 | 春耕/秋季追肥是需求高峰 |
| 纯碱 ↔ 玻璃 ↔ 地产/光伏 | 需求传导 | 地产压平板玻璃，光伏推光伏玻璃 |
| 油价 vs 煤价 | 决定油头 vs 煤头竞争格局 | 油贵煤便宜 → 煤化工利润好 |

### 2.3 成本比较框架（同一产品不同路线）

| 产品 | 油基路线 | 煤基路线 | 气基路线 | 分析意义 |
|------|---------|---------|---------|---------|
| PP | 石脑油裂解/PDH | 煤→甲醇→MTO | — | 边际成本决定价格底 |
| PE | 石脑油裂解 | 煤→甲醇→MTO | — | 同上 |
| PVC | 乙烯法(~20%) | 电石法(~80%) | — | 电石法主导定价 |
| EG | 石脑油→环氧乙烷 | 煤制EG | 天然气→EG | 三路线竞争激烈 |
| 甲醇 | — | 煤制甲醇(国内主) | 气制甲醇(进口主) | 进口依赖度~15% |

## Step 4: 分析框架

### 4.1 核心逻辑链

```
成本底 → 利润 → 开工率 → 供给
                              ↕
价格 ← 库存方向 ← 供需缺口 ← 需求
```

**自我修正闭环**：
1. 利润压缩 → 生产商降开工 → 供给收缩
2. 库存去化 → 供需偏紧 → 价格回升
3. 利润修复 → 开工回升 → 供给增加
4. 库存累积 → 供需转松 → 价格承压

### 4.2 品种分析模板

对每个🔴核心品种，维护以下分析框架：

```markdown
## [品种名] 跟踪

### 当前状态（最后更新：YYYY-MM-DD）

**价格**：期货 XXXX 元/吨 | 现货 XXXX 元/吨 | 基差 XX 元/吨
**历史百分位**：近3年 XX%（偏高/中等/偏低）
**库存**：厂库 XX万吨（↑/→/↓）| 社会库存 XX万吨（↑/→/↓）
**开工率**：上游 XX% | 下游 XX%
**利润**：油基 XX元/吨 | 煤基 XX元/吨 | (盈亏平衡线附近？)

### 供需判断

供给端：[产能变化/检修/进口情况]
需求端：[下游开工/季节性/终端消费]
库存验证：[累库→供过于求 / 去库→供不应求]

### 核心矛盾

[当前市场最关注的1-2个矛盾点]

### 观点与展望

[短期(1-2周)和中期(1-3月)判断]
```

### 4.3 季节性分析框架

| 月份 | 化工品需求季节性 |
|------|----------------|
| 1-2月 | 春节停工淡季，累库 |
| 3-4月 | 春耕（尿素旺季）、复工复产、地产开工季 |
| 5-6月 | 梅雨季（建材淡季）、农膜需求减弱 |
| 7-8月 | 高温检修季（供给收缩）、秋季备货启动 |
| 9-10月 | 金九银十（建材旺季）、秋季追肥（尿素） |
| 11-12月 | 冬储（甲醇/尿素）、取暖季（天然气/LPG） |

## Step 8: 质量自检

### 发布前检查

- [ ] 数据是否有明确来源？（标注"卓创/隆众/Wind/期货收盘"等）
- [ ] 价格数据是否为最新？（不能用过期数据误导）
- [ ] 供需逻辑是否自洽？（价格涨↔库存降↔供给紧 是否对得上）
- [ ] 是否区分了"期货"和"现货"？（两者可能方向不同）
- [ ] 季节性因素是否考虑到位？（3月说淡季还是旺季？）
- [ ] 新产能信息是否已确认？（未确认的标注"预计/规划中"）

### 内容红线

- ❌ 不做期货交易建议、不预测具体价格目标
- ❌ 不传播未经证实的停产/事故信息
- ❌ 不使用内幕信息或非公开调研数据
- ❌ 不简单归因（"原油涨所以化工涨"——需要分析传导路径）
- ✅ 客观呈现数据、逻辑链条和不确定性
- ✅ 标注信息来源和数据日期
- ✅ 使用"值得关注"而非"建议做多/做空"
- ✅ 涉及投资内容附风险提示
