## 常备选题池

> 不依赖热点、随时可以写的选题。按宣妈的内容方向分类。

### 📂 黄金科普入门

> 面向新手的黄金知识，搜索长尾流量好，常青内容。

- 积存金和金条有什么区别，新手入门必看（宣妈亲测视角）
- 黄金有哪几种投资方式？实物金 vs ETF vs 纸黄金 各适合谁
- 国内积存金怎么买？银行 vs 银行 对比测评

### 📂 金价热点解读

> 金价波动、政策变化、宏观事件时的即时点评。

- 金价跌了，但我不慌——地缘风险与黄金逻辑链梳理
- 油价破100、美元新高，黄金承压……长期逻辑变了吗？
- 黄金和美元都是避险资产，为啥有时候反向走？

### 📂 实物黄金经验

> 买金条/金饰的经验、渠道对比、保管知识等实用内容。

- 实物黄金 vs 黄金ETF，跌市心态有什么不同
- 麻麻家的黄金怎么放？实物金保管那些事

### 📂 资产配置思路

> 黄金在家庭资产中的角色、与固收+/指数基金的搭配逻辑。

- 「该不该买黄金」这个问题，产品经理式拆三步
- 仓位多少才合适？我的配置思路（黄金+固收+大A）

### 📂 理财心态与生活

> 长期主义感悟、带娃理财、产品经理视角拆解、佛系投资心态。

- 为什么我不担心黄金短期跌——长期主义底气从哪来
- 「黄金涨这么多是不是该卖了」——我怎么想这个问题
- 带娃之后，我的理财逻辑变了什么

### 选题模式参考（宣妈常用）

| 模式 | 说明 | 适合场景 |
|------|------|---------|
| **热点简评** | 金价波动/政策变化时的即时点评 | 日常简评，50-100 字 |
| **概念科普** | 用大白话解释黄金/理财概念 | 常青内容，搜索流量 |
| **产品经理拆解** | "其实就三点"式逻辑拆解 | 展示专业视角 |
| **生活感切入** | 从带娃/日常引出理财话题 | 真实感强，活人感 |
| **情绪记录** | 金价大涨/大跌时的真实感受 | 共鸣型内容 |
| **对比分析** | 实物金 vs 纸黄金、银行 vs 金店等 | 实用决策类，收藏率高 |
| **踩坑复盘** | 买金/理财中的教训 | 引发共鸣，真实感 |
| **月度复盘** | 市场回顾 + 仓位 + 心态 + 展望 | 每月一次，长文 |
| **争议话题** | "黄金还能涨吗""现在买晚不晚" | 引发评论讨论 |
| **新手指南** | 零基础买黄金入门 | 长尾搜索流量 |
