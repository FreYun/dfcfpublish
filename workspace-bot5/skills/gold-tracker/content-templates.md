# 内容输出模板

> 发布前必读。三种场景、三种模板，**不要混用**。

---

## 模板 A：盘中快报（飞书/内部用）

**场景**：交易时段内，黄金出现显著波动或重要消息，快速通报研究部。
**数据源**：金十数据 + web_search（实时数据）

```
🪙 黄金盘中快报 | MM/DD HH:MM

📊 当前报价
- COMEX 黄金：$XXXX.X（日内 +/-X.X%）
- AU9999：¥XXX.XX（日内 +/-X.X%）
- 美元指数：XXX.XX（+/-X.X%）

📰 消息面
- {事件1}
- {事件2}

💡 简评
{一两句口语化评论，符合宣妈风格}
```

**注意**：
- 价格必须来自实时数据源（金十/web_search），标注获取时间
- 如果用了 commodity_data 的数据，必须标注"（昨日收盘价）"
- 飞书内部通报用，不发小红书

---

## 模板 B：收盘复盘（可转小红书）

**场景**：A盘或外盘收盘后，做当日复盘。
**数据源**：commodity_data + Mysteel + web_search

```
🪙 黄金日报 | MM月DD日

📊 今日行情
- AU9999 收盘：¥XXX.XX（+/-X.X%，+/-¥X.XX）
- COMEX 黄金：$XXXX.X（+/-X.X%）
- 内外盘价差：¥X.XX（升水/贴水）

📈 近期走势
- 本周累计：+/-X.X%
- 近 30 天：+/-X.X%
- 当前位置：{历史高位附近 / 震荡区间 / 回调中}

📰 今日关键信息
1. {事件/数据1 + 影响}
2. {事件/数据2 + 影响}

💡 宣妈碎碎念
{2-3 句口语化点评，结合自己持仓心态}
```

**转小红书时**：
- 标题 ≤ 20 字，如"金价又新高了｜X月X日复盘"
- 删除内外盘价差等专业细节，保留核心涨跌和故事
- 加入个人感受（"我的金豆子又涨了"之类）

## 模板 C：周度复盘（小红书/公众号均可）

**场景**：每周五或周末，做一周回顾。
**数据源**：commodity_data 周线 + 本周消息汇总 + 知识星球观点

```
🪙 黄金周报 | X月X日-X月X日

📊 本周行情
- AU9999：¥XXX → ¥XXX（周涨跌 +/-X.X%）
- COMEX：$XXXX → $XXXX（周涨跌 +/-X.X%）
- 周内最高：$XXXX / 最低：$XXXX
- 美元指数：XXX.XX → XXX.XX

📰 本周大事记
| 日期 | 事件 | 对金价影响 |
|------|------|-----------|
| 周一 | ... | ... |
| 周三 | ... | ... |
| ... | ... | ... |

📊 关联指标速览
- 美债 10Y：X.XX%（+/-Xbp）
- 人民币汇率：X.XXXX
- VIX：XX.X
- COMEX 净多仓：{增/减}

🔍 市场观点汇总
- 【多方】{观点来源}：{核心逻辑}
- 【空方】{观点来源}：{核心逻辑}
- 【中性】{观点来源}：{关注点}

💡 宣妈本周感悟
{3-5 句，个人持仓心态 + 下周关注点}
```

---

## 通用注意事项

1. **所有价格必须标注是实时还是收盘**——读者不应猜测数据时效
2. **人民币金价和美元金价都要报**——宣妈的读者很多买实物金，关心人民币价
3. **不做精确价格预测**——可以说"短期可能震荡"，不要说"明天会到 XXX"
4. **关联宣妈的持仓/买金经历**——这是人设的核心，不是冷冰冰的行情播报
5. **数据错误是大忌**——宁可少写一个数据点，不要写错。盘中用收盘价是最常见的错误
