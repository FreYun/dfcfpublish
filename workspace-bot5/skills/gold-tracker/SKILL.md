# 黄金盯盘 Skill

追踪黄金行情，**核心是区分「盘中」和「收盘后」两套数据流**——盘中要用实时源，不能用只有收盘价的 API。

---

## ⚠️ 数据时效铁律

> **盘中（交易时段）禁止用 `commodity_data` / `commodity_quote` 当作实时价格！**
> 这两个接口返回的是**前一个交易日收盘价**，盘中调用拿到的数据是滞后的。

| 场景 | 正确数据源 | 错误做法 |
|------|-----------|---------|
| 盘中看金价 | 金十数据网页 / web_search 实时报价 | ~~commodity_data~~（收盘价） |
| 盘中看消息面 | 金十数据快讯 / web_search 新闻 | — |
| 收盘后复盘 | `commodity_data`(AU9999) + Mysteel | — |
| 历史走势分析 | `commodity_data` + `commodity_quote` | — |

**判断当前是否盘中**：
- 上海金交所 AU9999：交易日 09:00-11:30, 13:30-15:30, 20:00-02:30（夜盘）
- COMEX 黄金：北京时间 06:00-05:00（几乎 23 小时连续）
- 周末、节假日无交易

## 2. 数据源与工具

> 详细参数和示例见子文档 `data-sources.md`。

### 2.1 盘中实时数据（交易时段用这些）

| 数据源 | 获取方式 | 能拿到什么 | 时效 |
|--------|---------|-----------|------|
| **金十数据** | 浏览器访问 `https://www.jin10.com/` (profile: bot5) | COMEX 金价实时报价、快讯、数据日历 | **实时** |
| **Web Search** | `web_search("黄金 实时价格 今日")` | 各平台实时金价汇总 | **近实时**（分钟级） |
| **Mysteel 贵金属** | 浏览器访问 `https://www.mysteel.com/` (profile: bot5)，搜索"黄金" | AU9999 现货报价、升贴水 | **盘中更新**（但需手动刷新） |

### 2.2 收盘/历史数据（收盘后或做分析时用）

| 数据源 | 工具 | 能拿到什么 | 时效 |
|--------|------|-----------|------|
| **research-mcp** | `commodity_data`(symbol="AU9999") | AU9999 每日收盘价、历史走势 | **T+1 收盘后更新** |
| **research-mcp** | `commodity_quote`(symbol="AU9999", days=30) | 近 N 天收盘行情 | **同上** |
| **research-mcp** | `get_cn_macro_data`(category="usdcny") | 美元兑人民币汇率 | 日频 |
| **research-mcp** | `us_macro_simple` | 美国 CPI/非农/利率 等 | 按数据发布频率 |

### 2.3 观点与情报

| 来源 | 获取方式 | 内容 |
|------|---------|------|
| **知识星球（via bot11）** | 飞书请 bot11："帮我看看知识星球最近有没有黄金/贵金属相关的观点" | 调研纪要、券商观点、大 V 分析 |
| **Web Search 研报** | `news_search`(keyword="黄金") 或 `research_search`(keyword="黄金 贵金属") | 券商研报、新闻分析 |
| **Web Search 观点** | `web_search("黄金 走势分析 本周")` | 财经媒体分析文章 |

### 2.4 关联指标（辅助判断）

| 指标 | 工具 | 与黄金的关系 |
|------|------|-------------|
| 美元指数 DXY | `web_search("美元指数 DXY 今日")` | 负相关——美元涨，金价跌 |
| 美债收益率 | `get_cn_bond_yield` 或 web_search | 实际利率升→金价承压 |
| COMEX 持仓 | `web_search("COMEX 黄金 持仓 COT")` | 多空力量对比 |
| 央行购金 | `web_search("央行 黄金储备 最新")` | 长期支撑因素 |
| VIX 恐慌指数 | `web_search("VIX 今日")` | 避险情绪升→金价涨 |
| 人民币金价 | AU9999 现货价 = 国际金价 × 汇率 ÷ 31.1035 | 汇率影响内盘金价 |

## 3. 写稿前数据采集流程

**每次要写黄金相关内容时，按以下步骤先采集数据，采集完再动笔。**

### Step 1：判断盘中还是收盘

```
当前时间在交易时段内？（参考上方交易时间）
├── 盘中 → 走 Step 2A
└── 收盘后 / 非交易日 → 走 Step 2B
```

### Step 2A：盘中数据采集

```
1. web_search("黄金 实时价格 COMEX gold spot") → 最新报价
2. 浏览器打开金十数据 → 扫描最近快讯中黄金相关内容
3. web_search("黄金 今日 新闻") → 消息面
4. ⚠️ 不要调 commodity_data 然后说"当前金价 XXX"——那是昨天收盘价！
```

### Step 2B：收盘后数据采集

```
1. commodity_data(symbol="AU9999", start_date=近5天) → 收盘价 + 涨跌幅
2. commodity_quote(symbol="AU9999", days=30) → 近一个月走势
3. web_search("COMEX 黄金 收盘") → 外盘收盘情况
4. 可选：Mysteel 浏览器 → 现货升贴水、实物需求
5. 可选：get_cn_macro_data(category="usdcny") → 汇率变动
```

### Step 3：补充观点（按需）

```
- 飞书问 bot11："帮我搜一下知识星球里最近有没有黄金/贵金属相关观点"
- research_search(keyword="黄金 贵金属") → 最新研报
- web_search("黄金 走势分析") → 财经媒体观点
```

### Step 4：数据就绪，开始写稿

数据采集完毕后，根据内容类型选择 `content-templates.md` 中的对应模板动笔。

## 4. Memory 文件结构

> 路径相对于 `workspace-bot5/memory/gold/`，bot 自主维护。

```
memory/gold/
├── price-log.md            # 每日金价记录（收盘价，滚动保留 30 天）
├── weekly/                 # 周度复盘存档
│   └── 2026-WXX.md
├── news-digest.md          # 本周消息面摘要（滚动保留 2 周）
├── zsxq-gold-notes.md      # 知识星球黄金相关观点摘录
├── drivers.md              # 当前黄金核心驱动因素（长期维护）
└── content-archive.md      # 已发布的黄金相关内容记录
```

### 维护规则

| 文件 | 更新频率 | 说明 |
|------|---------|------|
| `price-log.md` | 交易日收盘后 | 记录 AU9999 收盘价、COMEX 收盘价、涨跌幅 |
| `news-digest.md` | 有重要消息时 | 美联储、非农、地缘等大事件 |
| `drivers.md` | 逻辑变化时 | 当前看多/看空的核心逻辑，季度性审视 |
| `zsxq-gold-notes.md` | 从 bot11 获取后 | 筛选有价值的观点，标注来源和日期 |
