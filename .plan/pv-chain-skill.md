# PV Chain Skill 实现计划

## 概述

创建光伏全产业链研究 skill `pv-chain`，放在 `workspace/skills/pv-chain/`（通用skill）。

## 核心设计思路

光伏 ≠ TMT。TMT是四个并行行业比景气差，光伏是**一条垂直产业链上的产能周期博弈**。

核心经济学框架：**产能周期理论**（capacity cycle）
- 高利润 → 扩产投资 → 产能释放 → 供过于求 → 价格战 → 亏损出清 → 供需再平衡 → 高利润
- 光伏2023-2025恰好处于"产能释放→价格战→出清"阶段，这是分析的核心背景

## SKILL.md 主流程设计

与TMT的关键区别：

1. **Step 1 不是定价环境，是产能周期定位** — 光伏的α来自周期位置判断，不是利率/风偏
2. **Step 2 不是四方向并行景气，是链上利润分配推导** — 利润在链上环节间流动（硅料暴利→电池亏损→组件微利），推导当前利润向哪个环节转移
3. **Step 3 和TMT类似** — 盈利验证
4. **增加技术路线判断** — PERC→TOPCon→HJT→BC 的技术迭代会重新洗牌竞争格局
5. **增加全球装机需求推导** — 光伏需求 = f(组件价格, 电价, 碳价, 政策补贴)
