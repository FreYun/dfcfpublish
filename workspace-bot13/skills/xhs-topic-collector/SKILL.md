# 美股话题收集器

> 美股留子 Alex 的素材积累系统。
> 目标：每天美股收盘后，有料可写、有数据可说。

---

## 一、素材分类

| 类型 | 说明 | 频率 | 示例 |
|------|------|------|------|
| 📈 **指数复盘** | 美股三大指数走势+原因 | 每日 | 纳指大涨2%创X月新高 |
| 🇨🇳 **中概异动** | 热门中概股异动解读 | 每日 | 小鹏暴涨15%，交付量超预期 |
| 🏛️ **Fed动态** | 联储官员讲话/政策信号 | 事件驱动 | 鲍威尔暗示暂停加息 |
| 📊 **宏观数据** | 非农/CPI/GDP等重磅数据 | 事件驱动 | 3月CPI超预期，降息预期降温 |
| 📅 **财经日历** | 本周重要事件预告 | 周度 | 本周FOMC+非农数据 |
| 💡 **市场热梗** | wallstreetbets/推特热聊 | 不定期 | Meme股暴动、某只股票热搜 |
| ☕ **留子日常** | 时差/看盘/咖啡相关 | 穿插 | 又是一个3am看盘的夜 |

## 三、SOP：素材积累流程

### Step 1: 美股收盘后 30 分钟内

**获取隔夜美股数据：**
1. 调用 `research-mcp` 查询：
   - 三大指数收盘数据（道指、标普500、纳指）
   - 热门中概股异动
2. 记录核心数据：收盘点位、涨跌%、成交量、关键原因

**产出：**
```
日期: 2026-04-02
📈 指数复盘:
- 道指: xxx (+x.xx%) - 原因: xxx
- 标普: xxx (+x.xx%) - 原因: xxx  
- 纳指: xxx (+x.xx%) - 原因: xxx

🇨🇳 中概异动:
- xxx: +xx% (原因)
- xxx: -xx% (原因)
```

### Step 2: 检查当日宏观事件

**是否有重磅数据/事件？**
- CPI/PPI/非农/FOMC/鲍威尔讲话
- 若有 → 立即查询数据，写快评
- 若无 → 准备常规复盘

**产出：**
```
🏛️ Fed动态 / 📊 宏观数据:
- 事件: xxx
- 数据: xxx (预期x.x vs 实际x.x)
- 市场反应: xxx
- Alex解读角度: xxx
```

### Step 3: 浏览市场情绪

**快速扫一眼：**
- Twitter/X 热聊话题
- 雪球社区情绪
- 是否有突发消息

**产出：**
```
💡 市场热梗:
- xxx股票 trending
- xxx meme流行
- 情绪: 贪婪/恐惧/麻木
```

### Step 4: 整理到素材库

**写入 `topic-library.md`：**
```markdown
## 2026-04-02 素材

### 今日可用选题
1. [指数复盘] 纳指创X月新高，科技股rally能否持续？
2. [中概] 小鹏财报超预期，中概造车新势力集体异动
3. [Fed] 鲍威尔讲话释放XX信号，6月降息预期升温

### 数据速记
- 道指: xxx, 标普: xxx, 纳指: xxx
- VIX: xx.x (恐慌指数)
- 10Y美债: x.xx%

### 待开发灵感
- [ ] 留子时差看盘的心路历程
- [ ] Fed暂停加息历史回顾
- [ ] 中概股vs美股大盘走势对比
```

## 六、输出标准

**素材合格标准：**
1. **有数据** — 不是"涨了"，是"涨了多少、为什么"
2. **有角度** — Alex怎么看这件事
3. **有时效** — 收盘后24小时内可用
4. **可展开** — 200字能说清楚，也能扩展到500字

**不合格素材：**
- ❌ "今天美股涨了"（太泛）
- ❌ "特斯拉又跌了"（单股，超出Alex范围）
- ❌ "感觉要崩了"（情绪化，无数据）
