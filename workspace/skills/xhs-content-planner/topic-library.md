## 灵感池（日常浏览随手记）

> 刷小红书、看雪球、读新闻、跟用户聊天时，发现好素材就往这里扔。
> 格式：`💡 选题简述 | 来源：xxx | 日期：YYYY-MM-DD`
> 用掉的标记 ✅，超过 30 天没用的定期清理。

_示例：_
- 💡 金价连续5天新高，可以写一篇"普通人要不要买黄金"的科普 | 来源：雪球热帖 | 日期：2026-03-12
  > 角度：从"我为什么配了25%黄金"的个人视角切入，不荐股，分享逻辑
- 💡 小红书刷到一篇"买基金vs买盲盒"的对比很火 | 来源：小红书首页推荐 | 日期：2026-03-12
  > 角度：可以模仿这个对比形式，写自己领域的版本

## 常备选题池

> 不依赖热点、随时可以写的选题。按 agent 自身的内容方向分类。
> 分类标签由 agent 根据自己的人设和擅长领域自定义。

### 📂 分类说明

agent 首次使用时，根据 IDENTITY.md 和 SOUL.md 中的擅长领域，创建自己的分类。例如：
- 投资理财类 agent → "入门科普""策略分享""情绪记录""热点解读"
- 美食类 agent → "食谱分享""探店记录""厨房好物""季节限定"
- 技术类 agent → "工具推荐""踩坑记录""概念科普""项目复盘"

每个分类下用 checkbox 管理选题，发过的标记 ✅ + 日期。

### 选题模板参考（agent 按需选用）

| 模式 | 说明 | 适合 |
|------|------|------|
| **踩坑复盘** | 分享真实犯过的错，引发共鸣 | 有真实经历时 |
| **概念科普** | 用大白话解释一个概念 | 常青内容，搜索流量好 |
| **情绪记录** | 记录某一刻的真实感受 | 行情波动/生活事件时 |
| **方法论分享** | 分享自己的做事方法和习惯 | 展示专业性 |
| **热点解读** | 从自身视角解读当下热点 | 蹭热点 + 个人观点 |
| **对比分析** | A vs B 我怎么选的 | 实用决策类，收藏率高 |
| **争议话题** | 提一个有争议的问题 | 引发评论和讨论 |
| **数据盘点** | 阶段性数据总结 | 月度/季度节点 |
| **新手指南** | 零基础入门指南 | 长尾搜索流量 |
| **日常碎片** | 生活中的小事、感悟 | 真实感强，活人感 |
| **季节时令** | 跟随季节的话题 | 自然流量，应景 |
| **系列连载** | 固定系列的某一期 | 粉丝粘性，长期运营 |

## 维护规则

1. **灵感池随时写**：看到好素材立刻记，不要等到选题时再找
2. **常备池按需补**：发现某个分类选题快用完了，集中补充一批
3. **发布后立即更新**：标记 ✅ + 日期 + 效果备注
4. **每周清理一次**：灵感池中超 30 天未用且失去时效性的归档或删除
5. **效果好的选题可复用**：换个角度、更新数据，隔 1-2 个月重新写
