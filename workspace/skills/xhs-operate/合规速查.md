# 合规速查（印务局打回时才读）

> 印务局打回帖子时，根据违规类型找到对应条目，修改后重新投稿。

---

## 金融类（最高频）

| 违规表现 | 修改方向 |
|---------|---------|
| 绝对化收益（"必涨"、"稳赚"、"XX%收益"） | 改为"可能"、"我的判断是"、"历史上曾经" |
| 明确投资建议（"现在买入"、"建议配置"） | 改为"我在关注"、"如果我来做选择"、"仅供参考" |
| 缺少风险提示 | 正文末尾加"市场有风险，以上仅为个人观点，不构成投资建议" |
| 炫富（晒具体收益金额） | 删除具体数字，改为感性描述 |
| 提及个股代码/名称（部分账号限制） | 替换为行业描述，如"某新能源龙头" |

## 引流类

| 违规表现 | 修改方向 |
|---------|---------|
| 引导加微信/私信/"主页有联系方式" | 删除所有引流话术 |
| 提及竞品平台（微博、抖音、B站等） | 删除或替换为"其他平台" |
| 二维码、链接 | 删除 |

## 内容规范类

| 违规表现 | 修改方向 |
|---------|---------|
| 标题含"最"、"第一"、"NO.1"等极限词 | 改为相对描述 |
| 未标注 AI 生成（如内容明显 AI 味） | 正文末尾加"以上内容辅助 AI 整理" |
| 暴露幕后身份（提到"研究部"、"boss"、"雇主"） | 统一以账号人设第一人称写，删除幕后信息 |
| 正文含 # 话题标签（应用 tags 参数） | 正文中的 `#话题` 移到 frontmatter `tags` 字段 |

---

印务局打回时会附带具体违规类型，对照上表直接修改，不需要通读全文。
