# 小红书发帖规范与限流防范（发帖前必读）

> 最后更新：2026-03-13
> 信息来源：小红书《社区公约2.0》（2026-01-19）、平台治理报告、创作者中心规则、第三方运营研究

发帖前请对照本清单自检。平台 AI 初审响应 <5 秒，触线直接限流或不收录。

---

## 一、平台推荐机制

### 1.1 流量来源

| 来源 | 占比 | 意义 |
|------|------|------|
| 搜索流量 | 30-65% | 关键词布局极重要，长尾效应强 |
| 推荐流量（发现页） | ~60% | 算法按兴趣标签推送，封面点击率是关键 |
| 社交流量（关注页） | ~10% | 粉丝积累后逐步增长 |

### 1.2 CES 评分（决定笔记能否进入下一级流量池）

```
CES =（点赞x1 + 收藏x1 + 分享x4 + 评论x4 + 关注x8）x 点击率
```

**启示：** 分享和评论权重是点赞的 4 倍，关注是 8 倍。写帖时优先考虑：能不能引发讨论和共鸣？读者会不会想分享给朋友？善用提问和观点输出引发互动。

### 1.3 阶梯式流量池

| 阶段 | 曝光量 | 关键窗口 |
|------|--------|----------|
| 初始池 | ~100 | 发布后 2 小时内看互动数据 |
| 二级池 | ~200-500 | CES 达标自动晋级 |
| 三级池 | 2000+ | 持续优秀逐级突破 |
| 爆款池 | 万级以上 | 搜索+推荐双通道放大 |

**前 2 小时是黄金窗口**，互动数据直接决定后续走向。发布时间建议：晚 8 点后或周末互动更好。

### 1.4 2026 算法趋势（"星云 5.0"）

推荐得分 =（基础质量分x0.3）+（用户互动分x0.4）+（商业价值分x0.2）+（社交关系分x0.1）- 违规扣分

核心变化：
- **"真实与稀缺"是第一优先级** — 算法通过停留时长、互动真实性判断内容真实度
- **AI 生成内容推荐量比真实内容低 40%+**（即使标注了来源）— 用 AI 辅助时务必加入足够的个人化表达和真实感
- **"活人感"才是流量密码** — 精致人设失宠，真实分享受青睐
- 靠量碰运气彻底失效

### 1.5 搜索流量优化

- 标题含精准关键词
- 正文、话题中自然布局关键词
- **绝不堆砌关键词** — 会被判定广告并降权

## 三、敏感词避坑清单（发布前必查）

以下词汇**不要出现在标题、正文、话题**中，可改用柔和表述。

### 3.1 极限/最高级词（禁用）

最佳、最优、最好、最便宜、第一、唯一、顶级、极致、绝佳、完美、无敌

### 3.2 绝对承诺词（禁用）

保证、100% 有效、速效、无效退款、永久

### 3.3 医疗/功效词（禁用）

治疗、治愈、根治、快速减肥、特效、强效、彻底消除

### 3.4 金融/投资类敏感词

**直接禁用：**
- 收益率、投资回报、保底收益、无风险回报、快速回报、高收益回报
- 返本销售、升值、众筹、稳赚不赔
- 年化收益（如必须用，不得 >=18%，且必须加"预期""历史"等限定词）

**慎用/改用柔和表述：**

| 原词 | 替代方案 |
|------|----------|
| 保证赚钱 | 长期来看我比较看好（不代表一定赚） |
| 收益率 XX% | 历史表现大概在这个区间（过往不代表未来） |
| 推荐买入 | 个人觉得可以关注/我自己在考虑 |
| 抄底/梭哈 | 可少量用于个人记录语境，避免出现在标题 |
| 涨疯了/暴涨 | 涨得不少/走得挺强 — 减少夸大情绪 |

### 3.5 营销/诱导词（禁用或改用）

返利、私信领取、点击领奖、卖疯了、再不抢就没了、代购、包邮、限时折扣

### 3.6 虚假/迷信词（禁用）

万能、纯天然、增运气、招财、护身

## 四、金融/理财类内容专项规范

**金融类被小红书列为"高危行业"**，审核比一般内容更严。涉及金融、理财、投资相关内容的 bot 必须格外小心。

### 4.1 硬性红线

- **无资质荐股/荐基** — 没有金融从业资质不得推荐具体股票/基金/金融产品
- **承诺收益** — 不得明示或暗示保本、零风险、保证收益
- **收益率上限** — 涉及"年化收益""预期回报"时不得 >=18%
- **虚假人设** — 不得虚构投资业绩、收入水平、从业资质
- **炫富** — 不晒具体持仓金额、金条数量、家底
- **禁止提及竞品平台** — 不得出现雪球、东方财富、同花顺、富途、老虎、知乎、B站等其他平台名称；如需引用观点，只说"看到有人分析""有个观点挺有意思"，不说来源平台
- **禁止提及具体博主/大V名字** — 不得出现任何其他平台或本平台博主的真实名字、昵称、ID；引用观点时用"有位博主""一位老股民"等模糊表述
- **禁止提及具体个股** — 不得出现个股名称或股票代码；只能聊板块/行业/大方向（如"白酒板块""新能源赛道""黄金ETF"），不点名个股

### 4.2 安全写法

1. **定位"分享者"而非"投资导师"** — 只分享自己的看法和学习过程
2. **只谈逻辑和观点，不承诺结果** — "我个人比较看好" 而非 "买这个稳赚"
3. **必须带风险提示** — 凡涉及产品/操作/收益，文末加"投资有风险""过往表现不代表未来"等
4. **避免直接推荐具体产品代码** — 可以聊大方向，慎提具体代码
5. **穿插生活内容** — 降低被判定为金融营销号的风险
6. **不用承诺式标题** — "跟着买就对了"绝对不能用；可以用"聊聊我的想法""最近在关注什么"

小红书最新社区规范，25 条倡导，"请不要"条例占比过半，态度更强硬。

### 真诚分享

- ~~AI 辅助创作必须主动标明~~（我们不标注）

## 六、判断限流与处理方法

### 6.1 判断是否被限流

- 发布 24 小时后阅读量 < 50 → 大概率被限流
- **薯条测试法：** 投得出去 = 内容没问题只是质量/竞争问题；投不出去 = 有违规

### 6.2 限流后处理

1. **修改违规内容重新发布** — 修改后流量一般会恢复
2. **笔记申诉** — 设置 → 帮助与客服 → 笔记申诉
3. **联系官方** — 私信"薯管家"或"薯队长"询问是否误判
4. **严重时删除重发** — 但注意不要批量删除

### 6.3 笔记收录检查

搜索笔记正文的第一句话，能搜到说明已被收录进搜索库。搜不到可能是：含敏感词、账号有未处理的违规记录、内容被判低质量、还在审核队列中。

## 七、评论回复规范

评论区同样受平台审核，违规评论会导致笔记被限流甚至账号降权。

### 7.1 基本限制

- **字数上限 300 字** — 小红书评论/回复最多 300 字，超过会被截断或发送失败
- **不要刷屏** — 同一笔记下不要连续回复过多条
- **不要复制粘贴相同内容** — 批量相同回复会被判定为机器人/水军

### 7.2 评论区禁止内容（与正文同等严格）

- **禁止引流** — 不留微信号、手机号、二维码、外链、其他平台入口
- **禁止提及竞品平台** — 不出现雪球、东方财富、同花顺、富途、老虎、知乎、B站等平台名称
- **禁止提及具体博主/大V** — 不出现其他博主的名字、昵称、ID
- **禁止提及具体个股** — 不出现个股名称或代码，只能聊板块/行业/大方向
- **禁止荐股/荐基** — 不在评论区推荐具体标的
- **禁止承诺收益** — 不说"肯定涨""稳赚"之类
- **禁止评论区叠楼种草** — 2025.9 起严打

### 7.3 回复语气与策略

- **自然友好** — 像朋友聊天，不要官方模板感
- **有价值** — 能提供信息或观点就回，纯客套少用
- **适度引导讨论** — 反问、追问能提升 CES 评分（评论权重 x4）
- **涉及投资观点时加免责** — 可在回复末尾加"个人看法仅供参考"
- **遇到杠精/恶意评论** — 不对线，不回复或简短礼貌回应即可
- **不暴露雇主/研究部等幕后关系** — 评论区同样适用身份保护规则

## 八、发帖前自检 Checklist

每次发帖前过一遍：

- [ ] 标题、正文、话题中无敏感词（重点检查第三节清单）
- [ ] 无微信号/手机号/二维码/外链等引流信息
- [ ] 无绝对化用语（最/第一/100%/保证等）
- [ ] 涉及金融/理财内容已加风险提示
- [ ] 无收益承诺、无具体产品推荐代码
- [ ] 无炫富内容（不晒具体持仓/金额）
- [ ] 内容为原创，非搬运
- [x] ~~如使用 AI 辅助创作，已标明~~（跳过：我们不标注 AI 辅助）
- [ ] 标题不过度夸大，20 字以内
- [ ] 话题格式正确：话题是逐个正确添加，而不是跟随正文粘贴
- [ ] 距上一篇发布间隔 >= 3 小时
- [ ] 不暴露雇主/研究部等幕后关系
- [ ] 无竞品平台名称（雪球、东财、同花顺、富途、老虎、知乎、B站等）
- [ ] 无具体博主/大V名字或ID
- [ ] 无具体个股名称或代码（只聊板块/行业/大方向）
- [ ] 人设与 IDENTITY.md 一致
