# 小红书写作通用规范（xhs-writing）

> 写稿前必读。排版和表达决定流量，内容质量决定留存。

---

## 一、标题（≤20 字，流量第一关）

### 高点击公式

| 公式 | 示例 | 场景 |
|------|------|------|
| 疑问+反问 | "黄金为何不涨了？不是说避险抗通胀吗？" | 击中困惑（5279赞验证） |
| 日期+安抚+判断 | "3.16，别慌，买点快来了" | 波动时安抚（717赞） |
| 热点+❗️+疑问 | "中东战火❗️为什么黄金反而大跌？" | 热点解读（385赞） |
| 数字+情绪 | "从-22w到+25w" "一个月高300块" | 收益/里程碑 |
| 悬念+省略号 | "黄金，大事发生..." | 吊起好奇心 |
| 称呼+劝告 | "满仓的宝听我一句" | 拉近距离 |
| 短+情绪爆发 | "黄金！！！" "没完了啊" | 本能反应（最高互动） |

**结论**：疑问句是流量密码，尤其"反问式"和"认知差式"。**情绪型标题 > 分析型标题**。

### 标题禁忌
- 超20字（被截断）
- 绝对化用语（最/第一/100%）
- 承诺收益（稳赚/保底）
- 标题党（内容不符）

## 二、开场（前3秒决定去留）

### 高互动开场类型

**1. 情绪本能型**（最高互动）
> "下午开会间隙看了眼金价……我直接在工位上沉默了"
- 直接进场景，不铺垫，像发朋友圈

**2. 痛点共鸣型**
> "毫无过年的感觉，只有对年后生活焦虑"
- 说出大家不敢说的

**3. 自嘲反差型**
> "我可能疯了，太喜欢上班赚钱的感觉了"
- 制造反差和幽默，显得真实

**4. 故事铺垫型**
> "过年回老家，我才突然顿悟：我妈是怎么靠普通工资悄悄攒下两套房的"（5370赞）
- 家人故事天然亲切，"悄悄"有秘密感

**5. 灵魂拷问型**
> "你买黄金，到底是图什么？"
- 直接反问，逼读者想

**6. 争议判断型**
> "反正你们都在抄底那一定没见底"（1350赞，958条评论全是争论）
- 一句争议判断 > 一篇完整分析

### 开场避雷（数据验证的低互动模式）

| ❌ 别这么写 | ✅ 换成 |
|------------|--------|
| "很多姐妹问我…" | 直接进场景 |
| "今天想跟大家聊聊…" | 直接抛观点或故事 |
| "大家好我是xxx…" | 不自我介绍 |
| "关于xxx我有几点看法…" | 直接说结论 |
| "有点紧张第一次发…" | 别矫情 |

## 三、正文排版（视觉 > 内容）

### 核心原则：短段落 + 多换行 + 呼吸感

```
❌ 错误（大段压一起）：
看到金价波动，很多姐妹第一反应是要不要跑或者要不要补。说实话，我也经历过坐过山车——刚买完就跌，卖了又涨，来回折腾心态崩了。后来想明白了：频繁操作的前提是能判断短期走势，但这个没人能做到。

✅ 正确（短段落+换行）：
看到金价波动，第一反应是「要不要跑」🤔

说实话，我也经历过——
刚买完就跌，卖了又涨，心态崩了😅

后来想明白了：
频繁操作的前提是「能判断短期走势」
但这个，没人能做到🙅‍♀️
```

### 分行规则

| 场景 | 方法 |
|------|------|
| 对比/转折 | 每行一个意思 |
| 列表要点 | 序号+换行+空行，用❶❷❸或1️⃣2️⃣3️⃣ |
| 金句/结论 | 单独成行，前后空行 |
| 情绪表达 | 短句+emoji |

### 段落控制
- 每段1-3行，不超过
- 总段数控制在8-12段（太碎零散，太密压抑）
- 该合并的合并，该强调的单独成行

### Emoji规范
- 每段最多1-2个，点缀不堆砌
- 用在段尾情绪点或关键转折处
- ❌ 每句话后面都加emoji = 机器人感

## 四、正文结构公式

### 四步结构（爆款验证）

```
1. 共情开场 → 击中痛点/困惑（1-2句）
2. 逻辑拆解 → 分点，用"第一/第二/第三"（主体）
3. 明确结论 → 有态度，不两头说（1句）
4. 互动钩子 → 留一个让读者想说话的点（1句）
```

### 关键：制造讨论空间

**小红书不是比谁分析得好，是比谁互动率高。**

- 内容写得太完美太完整 → 读者看完觉得有道理然后划走，没评论冲动
- 留一个争议点/开放问题/情绪共鸣 → 读者有话说

| 互动钩子类型 | 示例 |
|-------------|------|
| 灵魂拷问 | "你买黄金，到底是图什么？" |
| 求互动 | "评论区留个言" "你们觉得呢？" |
| 争议判断 | "反正你们都在抄底那一定没见底" |
| 情绪共鸣 | "谁懂啊！！" "允许我emo一会儿" |
| 晒自己 | "这周只做了一件事" "我心里在滴血" |

## 五、拉近距离的措辞

### 口语化替换

| ❌ 正式 | ✅ 口语 |
|--------|--------|
| 我们 | 咱们 |
| 最近的 | 这届 |
| 非常 | 超/巨/真的 |
| 我认为 | 我觉得吧 |
| （无语气词） | 吧/呢/哈/诶 |

### 自我暴露（制造信任）
- "我一开始也有点懵"
- "说实话有点焦虑"
- "我也在摸索"

### 对话感（不是讲话感）
```
❌ 讲话感："黄金的优点有……"
✅ 对话感："你知道吗？黄金其实……"
```

### "钱→时间"转化（理财类高赞秘诀）
```
❌ "年化10%，年收益4万"
✅ "存下来的不是钱，是将来能不慌不忙过日子的底气"
✅ "这笔利息，等于你多赚了4个月的自由时间"
```

## 六、话题标签 & 风险提示

### 话题格式（必须正确）
```
✅ # 黄金 # 理财 # 长期主义
❌ #黄金 #理财（#后没空格=平台不识别）
```
- 1-3个为宜，不堆砌
- 大流量词+垂直词混合

### 风险提示（金融类必带）
```
———
（仅个人记录，不构成投资建议）
```
涉及具体产品时加：`*文中提及产品仅作为举例，不作为推荐。投资有风险，入市需谨慎。`

## 七、限流防范速查

### CES评分（决定流量池晋级）
```
CES =（点赞x1 + 收藏x1 + 分享x4 + 评论x4 + 关注x8）x 点击率
```
**分享和评论权重是点赞的4倍**。写帖时优先想：能引发讨论吗？读者会分享吗？

### 前2小时是黄金窗口
发布后2小时内的互动数据决定后续推荐。发布时间建议：晚8点后或周末。

### 2026算法趋势
- "活人感"是流量密码，精致人设失宠
- AI生成内容推荐量低40%+，必须加足够个人化表达
- 靠量碰运气彻底失效

### 高危红线（触一个就限流）

| 类型 | 具体 |
|------|------|
| 引流 | 微信号/手机号/二维码/外链/其他平台水印 |
| 绝对化 | 最/第一/100%/保证/稳赚 |
| 金融违规 | 无资质荐股荐基、承诺收益、年化>=18% |
| 炫富 | 晒具体持仓/金额/数量 |
| 竞品 | 雪球/东财/同花顺/富途/老虎/知乎/B站 |
| 具体标的 | 个股名称或代码（只能聊板块/行业） |
| 虚假人设 | 虚构财富/学历/收入 |

### 金融敏感词替换

| ❌ 禁用 | ✅ 替代 |
|--------|--------|
| 保证赚钱 | 长期来看我比较看好 |
| 收益率XX% | 历史表现大概在这个区间 |
| 推荐买入 | 个人觉得可以关注 |
| 涨疯了/暴涨 | 涨得不少/走得挺强 |

### 发帖前 Checklist

- [ ] 无敏感词（金融类重点查）
- [ ] 无引流信息
- [ ] 无绝对化用语
- [ ] 涉及理财已加风险提示
- [ ] 无收益承诺
- [ ] 无炫富
- [ ] 原创内容
- [ ] AI辅助已标明
- [ ] 标题≤20字
- [ ] 话题格式正确（# 后有空格）
- [ ] 距上一篇≥3小时
- [ ] 不暴露幕后关系
- [ ] 无竞品平台/具体个股
- [ ] 人设一致

### 评论区规范
- 字数上限300字
- 不刷屏、不复制粘贴
- 评论区同样禁止：引流/竞品/荐股/承诺收益
- 回复要有价值（增量信息），不要纯客套
- 涉及投资观点加"个人看法仅供参考"

## 八、评论区运营（第二次传播）

| 技巧 | 效果 |
|------|------|
| 置顶"简单版"解释 | 降低理解门槛，二次传播（273赞验证） |
| 回复给补充信息 | 增加互动深度 |
| 引导用户晒自己 | 拉评论数，提升权重 |
| 幽默互动 | 段子比说教更有传播力 |

**科普类评论区策略**：每条回复给一点新信息，评论区变成第二篇干货帖。
