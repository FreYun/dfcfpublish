# 基金推荐规则

## 目标

这个步骤现在由本地 Python 脚本承担：**先调用脚本拿基金筛选结果，再继续后面的写作。**

主 bot / agent 只需要：

1. 先确定 2-3 个重点关注方向或主题
2. 调用 `fund_selector_llm.py`
3. 等脚本返回 JSON 结果
4. 把返回结果写成推荐文案

脚本内部会自己读取本地底库（位于 skill 目录下）：

- `东财基金.csv`
- `YYYYMM-指数基金池.xlsx`
- `YYYYMM-权益基金池.xlsx`

---

## 调用方式

多个主题：

```bash
python skills/daily-market-recap/fund_selector_llm.py --topics 科技,新能源,医药
```

单个主题：

```bash
python skills/daily-market-recap/fund_selector_llm.py --topic 创新药
```

调试模式：

```bash
python skills/daily-market-recap/fund_selector_llm.py --topic 创新药 --debug
```

---

## 内部规则

脚本内部不是让 agent 手工挑基金，而是分两步：

1. **代码召回候选**：
   - 从 3 个本地基金文件中召回候选
   - 自动分层、去重、控制候选规模
2. **模型最终筛选**：
   - 调用大模型理解主题/板块
   - 根据优先级和补位规则选出最终基金

所以 agent 在这里的正确动作是：
- **运行脚本**
- **读取输出**
- **根据输出写成表格格式的基金推荐**

而不是自己在表格里翻基金。

### 输出格式

基金推荐在 04 后市展望中以表格形式呈现，紧跟在对应板块分析段落下方：

```
相关基金：

| 代码 | 基金名称 | 基金经理 |
|------|---------|---------|
| 008327 | 东财通信C | 姚楠燕 |
| 021894 | 易方达半导体设备ETF联接C | xxx |
```

- 用"相关基金："引出，后接 markdown 表格
- 表格三列：代码、基金名称、基金经理
- 每个板块的基金紧跟该板块分析，不要集中堆到文末

---

## 使用要求

- 优先调用这个 Python 脚本，不要手工翻基金池
- 在脚本输出之前，不要先写基金推荐
- 如果脚本返回空结果或报错，先处理这一步，再继续后面的写作
- 不要绕过这个脚本直接去读 CSV/XLSX 手工筛基金

---

## 基金池月度更新

基金池数据来自天天基金研究部，每月初更新。使用自动下载脚本更新：

```bash
# 下载当月基金池（--bot 传入你自己的 bot ID）
python skills/daily-market-recap/download_fund_pools.py --bot <你的botID>

# 指定月份
python skills/daily-market-recap/download_fund_pools.py --bot <你的botID> --month 202605
```

脚本会自动下载 xlsx 到 skill 目录并删除旧月份文件。

**首次使用或登录过期时**，脚本会自动截取微信登录二维码通过飞书发给研究部，扫码后继续下载。也可手动触发登录：

```bash
python skills/daily-market-recap/download_fund_pools.py --bot <你的botID> --login
```

数据来源：`https://docs.qq.com/s/KEims9pfBxkMG6-ujR4wYq`（天天基金研究部-基金池汇总）
