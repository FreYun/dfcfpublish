# 券商观点获取（03/04 写作前必做）

> 写 03 盘面解读和 04 后市展望之前，除了行情数据，还需要主动搜索当日券商的市场解读和策略观点。这一步与 [行情数据获取](行情数据获取.md) 并列，都是写稿前的必做功课。

---

## 为什么要做这一步

研究部传入的新闻链接侧重于**消息面事实**（政策、事件、数据发布），但 03/04 需要的是**分析和观点**。券商策略团队每天都会发布收评和策略报告，包含：

- 对当日行情的专业解读（比你从数据推导更有深度）
- 对后市的展望和判断（多家券商的共识/分歧本身就是有价值的信号）
- 板块轮动逻辑和产业催化线索
- 技术面和资金面的专业分析

引入这些观点，能让文章从"数据+新闻的机械拼凑"升级为"有专业支撑的市场分析"。

---

## 搜索流程

### 第一步：用 research-mcp 搜索券商当日观点

```
# 搜索当日 A 股策略/收评
npx mcporter call 'research-mcp.research_search(query: "A股 策略 收评 YYYY-MM-DD", top_k: 10)'

# 搜索当日后市展望
npx mcporter call 'research-mcp.research_search(query: "A股 后市 展望 观点", top_k: 10)'

# 搜索券商盘后解读
npx mcporter call 'research-mcp.research_search(query: "券商 盘后 解读 策略", top_k: 10)'

# 如果当日有重大事件，针对性搜索券商解读
npx mcporter call 'research-mcp.research_search(query: "具体事件关键词 券商 解读", top_k: 5)'
```

> 灵活组合关键词，多搜几轮，确保覆盖面足够。

### 第二步：提炼和整理

从搜索结果中提取以下信息，记在草稿中供写稿使用：

```
【券商观点汇总（草稿，不发布）】

1. 大势判断
   - XX证券：认为……（看多/看空/震荡，核心理由）
   - YY证券：认为……
   → 共识点：……
   → 分歧点：……

2. 板块观点
   - XX证券看好：……板块（理由）
   - YY证券看好：……板块（理由）
   → 多家共同看好的方向：……

3. 风险提示
   - 多家券商共同提及的风险点：……

4. 值得引用的精彩观点
   - "原文引述"（XX证券 策略团队）
   - "原文引述"（YY研究员 某某某）
```

## 如何融入写作

### 03 盘面解读中的使用

- 解释板块强弱时，可引入券商对板块逻辑的专业分析
- 解释资金行为时，可引用券商对资金面的判断
- **不要大段转述**，用一两句话点出券商的核心判断，自然融入你的分析逻辑

### 04 后市展望中的使用

- **短期技术面**：引入券商对支撑位/压力位的判断，与自己的判断互相印证或对比
- **外部/内部环境**：引入券商对宏观和政策面的专业解读
- **重点关注方向**：多家券商共同看好的方向，说服力更强；如有分歧，可以呈现不同声音
- **风控提示**：券商提及的风险点是重要参考

### 引用方式

遵循 [引用来源管理.md](引用来源管理.md) 的规则：

- 明确来源的观点：正文中归因（"中信建投策略团队认为……""国泰君安指出……"）
- 多家共识：列出 1-2 家具名券商，再用"等"收尾（"中信建投、国泰君安等多家券商策略团队均认为……"），不要笼统说"多家券商"而不点名
- 不确定来源的：用自己的话表达，不虚假归因
- 写稿后附【引用来源（不发布）】清单

---

## 注意事项

1. **当日观点优先**：只搜索和引用当天发布的券商观点，隔天的策略报告时效性大打折扣
2. **不要全盘照搬**：券商观点是参考素材，不是抄写对象。提炼核心判断，用自己的话和人设风格表达
3. **多方交叉验证**：至少参考 2-3 家券商的观点，避免被单一券商的偏见带偏
4. **保持独立判断**：如果券商观点与你从数据面得出的判断矛盾，应该分析矛盾原因，而不是盲目跟随
5. **搜索要高效**：整个券商观点搜索环节控制在合理时间内完成，不要无限发散
