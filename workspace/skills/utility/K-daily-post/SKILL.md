# 老K发帖 SOP

## 触发条件

> ⚠️ 本 skill 仅供 **cron 定时任务 / 心跳巡检 / 自主判断**触发。研究部直接说「去发帖」「投稿」等指令时，**不走本 skill**，直接按研究部要求投稿即可。

- 心跳巡检到了选题推荐时间段（10:00 / 20:00）
- 自主判断有值得写的热点

---

## 五步流程

```
问热点(bot1) → 自研(2站+) → 推选题(飞书) → [研究部选题] → 写稿+做图 → [研究部终审] → 投稿
```

### Step 1：问热点

向 bot1 获取当日盘面信息和热点板块。

```
send_message(to: "bot1", content: "今日热点和盘面梳理", trace: [{
  agent: "bot7",
  session_id: "<当前session>",
  reply_channel: "feishu",
  reply_to: "<研究部ou>",
  reply_account: "bot7"
}])
```

**从 bot1 回复中提取**：
- 大盘指数走势、成交量
- 当日热门板块 & 资金流向
- 核心主线 & 异动方向
- 风险提示

**产出**：热点摘要记录到 `memory/YYYY-MM-DD.md`。

---

### Step 2：自主研究

基于 bot1 的热点信息，选 1-2 个与自身关注领域（AI/芯片/CPO/具身智能/有色/新能源）契合的方向深挖。

**必须做到**：
- 至少访问 **2 个站点**（雪球、东方财富、同花顺、巨潮、36kr 等）
- 调用已装备的研究技能：`deepresearch` / `sector-pulse` / `flow-watch` / `market-environment-analysis` 等
- 形成独立判断，找到预期差或数据亮点

**产出**：研究笔记存入 `memory/research/` 或当日日记。

---

### Step 3：推选题给研究部

整理 **2-3 个选题方案**，通过飞书发送给研究部。

**每个选题包含**：

| 字段 | 要求 |
|------|------|
| 选题标题 | 8-18 字，符合 dae-fly-style 标题公式 |
| 核心论点 | 1-2 句话，这篇帖子要传达什么判断 |
| 内容类型 | 日评 / 主题分析 / 热点解读 |
| 数据亮点 | 支撑论点的关键数字（成交额、涨幅、资金流向等） |

**飞书推送格式示例**：

```
📝 今日选题推荐（3月26日）

1️⃣ 【半导体设备走到哪里了？】
   论点：国产替代进入加速期，设备端订单确认拐点
   类型：主题分析
   亮点：北方华创Q1订单同比+45%，中微公司海外收入占比首破30%

2️⃣ 【AI电力：被忽略的算力瓶颈】
   论点：数据中心扩张推动灵活电力需求，储能是下一个爆发点
   类型：热点解读
   亮点：全国数据中心用电量同比+28%，储能装机目标上调

3️⃣ 【3月26日 逻辑记录！】
   论点：市场风格切换信号明显，资金从AI向周期轮动
   类型：日评
   亮点：有色板块成交占比创月内新高，北向净流入XX亿
```

**然后等待研究部选定或修改。**

---

### Step 4：写稿 + 做图

研究部审核通过后，按选定主题执行。

#### 4a. 写稿

- 遵循 `skills/dae-fly-style/SKILL.md` 检查清单
- 正文结构：开头钩子 → [一R] → [二R] → [三R](可选) → [打卡R] → [加一R](可选) → 声明 + 标签
- 字数 500-1200
- 反审查表达：高风险词替换（🈶色、📈、💰、基J、投Z 等）
- 结尾加免责声明 + `#行情[话题]# #基金[话题]#` 等标签
- 遵循发帖红线（`skills/xhs-op/发帖前必读.md`）

#### 4b. 做图

- 按 `skills/laok-style/SKILL.md` 风格指南准备配图
- **默认用模板 E（大标题文字封面）**：画面主体是醒目中文大标题，右下角放像素扑克牌 icon
- **不要画 K 线图/蜡烛图/数据面板**，除非研究部明确要求概念插画（此时用模板 F）
- 所有图内文字必须中文

#### 4c. 发研究部预览

将完整稿件（标题 + 正文 + 生成的图片）通过飞书发给研究部预览。

---

### Step 5：研究部终审 → 投稿

**研究部说"发"之后**，走投稿发布流程：

```bash
cat > /tmp/post_body_$$.txt << 'BODYEOF'
正文全文（图下正文）
BODYEOF

folder=$(bash skills/xhs-op/submit-to-publisher.sh \
  -a bot7 -t "标题" -b /tmp/post_body_$$.txt \
  -m image -r "direct:ou_xxx" \
  -i "/abs/path/to/cover.png" \
  -T "A股,投资" \
  -V "公开可见")
echo "FOLDER: $folder"
```

**统一规则**：
- 本 skill **只走图文投稿（`-m image`）**，禁止使用 `text_to_image`
- Step 4 生成的封面图必须通过 `-i` 传入投稿脚本
- `-b` 文件内容就是最终图下正文，不再拆分 `text_image` / `content`

**投稿后**：
- 更新 `memory/发帖记录.md`（日期、标题、正文摘要）
- 更新 `memory/YYYY-MM-DD.md`（当日日记）
- 被打回 → 按 `skills/xhs-op/合规速查.md` 修改 → 重新提交
