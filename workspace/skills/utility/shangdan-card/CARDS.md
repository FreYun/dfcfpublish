# 图片规格（CARDS.md）

> Step 2 规划方案时 Read 此文件。

---

## 图 1：封面卡（Hook）

吸引点击，告诉读者「讲什么、为什么跟你有关」。

| 区域 | 布局 | 内容 | 文字量 |
|------|------|------|-------|
| 标题 | `FULL` | 大号加粗 + emoji | ≤ 15 字 |
| 副标题 | `FULL` | 一句话点题 | ≤ 25 字 |
| 概念速览 | `ROW` | 3-4 个关键词 + 小图标 | 每个 2-4 字 |
| 引导文 | `FULL` | 1-2 句勾起好奇心 | ≤ 50 字 |
| 底部 | `FULL` | 风险提示 + 广告标 | 固定 |

**文字总量：~120-150 字**

**标题公式：**
- 疑问句：`债🏦波动还受到哪些影响？`
- 反常识：`债基也会亏钱？`
- 数字+结论：`搞懂这 3 件事，买债基不再慌`

## 图 2：逻辑讲解卡 A

上半文字阐述，下半示意图总结。

| 区域 | 布局 | 内容 | 文字量 |
|------|------|------|-------|
| 小标题 | `FULL` | 本张主题（加粗 + emoji） | ≤ 15 字 |
| 要点阐述 | `FULL` 或 `LR` | 2-3 个要点 = 高亮小标题 + 1-2 句解释 | ~120 字 |
| 示意图 | `FULL` | AI 生成的手绘风/扁平风图解 | 图内关键词 ~30 字 |
| 底部 | `FULL` | 风险提示 | 固定 |

**文字总量：~150-180 字**

**要点排版：**
```
【要点 1】（彩色高亮底色）
1-2 句大白话 + emoji

【要点 2】
1-2 句大白话
```
要点之间用**空行留白**分隔，不要用横线。

**示意图类型：**

| 逻辑类型 | 图解形式 |
|---------|---------|
| A 导致 B | 中心发散型 |
| 步骤/流程 | 线性流程图（→→→） |
| 对立/互补 | 天平/VS |
| 循环 | 环形图 |
| 多因素 | 鱼骨图/三角 |

## 图 4：业绩 + 逻辑总结（必须有）

**左右分栏布局**：左边逻辑阐述，右边业绩走势图。

| 区域 | 布局 | 内容 | 文字量 |
|------|------|------|-------|
| 小标题 | `FULL` | 基金名称或一句话总结 | ≤ 15 字 |
| 主体 | `LR` | **左栏**：2-3 个要点（为什么看好/配置逻辑/关键指标解读）<br>**右栏**：研究部提供的业绩走势图 | 左栏 ~100 字 |
| 风险提示 | `FULL` | 加强版风险提示（见 COMPLIANCE.md） | ~80-120 字 |

**文字总量：~150-180 字**

**⚠️ 业绩走势图必须是研究部提供的真实截图，不是 AI 画的占位符。**

**处理规则：**
- 走势图由研究部提供，**作为真实图片插入卡片右栏**，不要让 AI 画
- 生图时右栏预留图片区域，生成后用工具将真实截图合成进去
- 如果无法合成，**业绩走势图单独作为帖子的一张图发布**（紧跟在图 4 之后），不要用 AI 生成假图
- 左栏写逻辑阐述（配置理由、核心指标解读），不是纯数据罗列
- 风险提示比其他图更详细（涉及业绩数据）

**左栏内容模板：**
```
─
【为什么关注这只基金】
1-2 句配置逻辑
─
【关键指标一句话】
近1年收益 xx% / 最大回撤 xx%
─
【一句话总结】
适合什么样的投资者
```

## 示例 — 债基科普帖

```
图 1（封面）
  标题：「债🏦波动还受到哪些影响？」
  副标题：搞懂这三件事，买债基不再慌
  概念速览：💰资金流向  📊利率变化  🔮经济预期
  引导文：很多人以为债基稳稳的不会亏，其实它也会波动👇

图 2（逻辑）
  小标题：「影响债基波动的三个关键因素」
  要点 1：资金流向 — 大家都追股票时，债就被冷落
  要点 2：利率变化 — 新债更划算，旧债只能降价
  要点 3：经济预期 — 不确定时钱回流债市
  示意图：中心发散型 — 中心「债基波动」

图 3：省略

图 4（业绩+逻辑）[LR 布局]
  小标题：「某债基近 1 年表现」
  左栏：
    【为什么关注】波动小、回撤可控，适合做底仓
    【关键指标】近1年 +3.8% / 最大回撤 -1.2%
    【一句话】适合求稳、不想天天看盘的投资者
  右栏：研究部提供的业绩走势截图
  风险提示：市场有风险 + 过往业绩不预示未来 + 不构成投资建议
```
