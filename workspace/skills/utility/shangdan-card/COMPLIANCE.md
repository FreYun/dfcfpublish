## 图片风险提示

### 每张图底部必带

```
市场有风险，投资需谨慎。
```

小字、灰色、底部居中。**另加广告标。**

### 图 4（业绩图）加强版

图 4 涉及业绩数据，在基础提示之上按内容叠加：

| 内容涉及 | 追加 |
|---------|------|
| 业绩走势/净值 | `过往业绩不预示未来表现。` |
| 市场观点/行业分析 | `以上内容不构成投资建议或行情预测。` |
| 基金经理/代表作 | `基金经理在管的其他产品业绩不代表本产品业绩。` |
| 定投/回测 | `仅根据模拟方法进行回测，不代表历史实际收益水平。` |
| AI 辅助生成 | `本内容部分由AI辅助生成，准确性和时效性可能存在不足。` |
| 止盈类产品 | `目标止盈年化率仅代表投资者的期望，不预示未来表现。` |
| 快取等增值服务 | `快取服务非法定义务，提现有条件，依约可暂停。` |
| 提及证监会核准 | `证监会核准不代表对该基金的风险和收益作出实质性判断。` |
| 管理人股东背景 | `基金管理人与股东之间实行业务隔离制度。` |

## 正文风险提示

正文末尾用 `———` 分隔后，放基础提示 + 场景化叠加（同上表）。

**示例（科普类 + AI 辅助）：**
```
———
市场有风险，投资需谨慎。
以上内容不构成投资建议或行情预测。
本内容部分由AI辅助生成，准确性和时效性可能存在不足。
```

**示例（产品类 + 基金经理 + 回测）：**
```
———
市场有风险，投资需谨慎。
过往业绩不预示未来表现；基金经理在管的其他产品业绩不代表本产品业绩。
仅根据模拟方法进行回测，不代表历史实际收益水平。
```

## 绝对禁止

| ❌ 禁令 | 说明 |
|--------|------|
| 保证/承诺收益 | "稳赚""保底""年化 XX%"都不行 |
| "最终解释权归本公司所有" | 任何营销活动免责声明中禁用 |
| 绝对化用语 | 最/第一/唯一/100% |
| 隐瞒特殊风险 | 特殊品种必须揭示投资范围、策略、风险收益特征 |
| 募集上限/配售作为销售主题 | 可列示但不得加大字号强调 |
| 竞品/个股名称 | 不提雪球/东财等平台，不提个股代码 |
