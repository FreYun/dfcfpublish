## 下游乘数叠加

下游 strategy skill 调用本 skill 后，应在 `playbook.position_limit.total` 基础上再乘以若干系数（可叠加）：

| 条件 | 乘数 |
|---|---|
| `confidence == "medium"` | ×0.8 |
| `confidence == "low"` | ×0.5 |
| `switched == true` (regime 切换当日) | ×0.8 |
| `emergency_switch == true` (逃生门触发当日) | ×0.7 |

**多重叠加**：全部条件满足时 `0.5 × 0.8 × 0.7 = 0.28`，再乘以档位基准仓位。例如中性震荡档 50% × 0.28 = **14% 总仓**。切换首日 + 数据低可信度 + 逃生门 = 极度保守。

## 切换规则

详见 spec §6 与 `regime_rules.py`。核心要点：

- **3 日确认**：不立即响应单日极端得分，要求连续 3 日才切换
- **不对称缓冲区**：向悲观切换需多 1 分缓冲（"宁可踏空不可站岗"）
- **跨档**：仅下行允许一次跨 2 档
- **逃生门**：单日剧变直接覆盖 3 日确认，下行 OR 宽松，上行 AND 严格
