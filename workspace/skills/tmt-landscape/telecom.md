## 供需分析框架

### 需求侧——两大需求引擎 + 一个新兴引擎

**引擎1：AI算力网络建设（当前最强）**
- 跟踪变量：AI算力中心招标/开工数、运营商算力Capex、光模块规格升级节奏
- 数据源：`research_search("算力 光模块 800G 1.6T")` + `news_search("算力中心 招标")`
- 传导链：AI算力需求↑ → 数据中心扩建 → 光模块/光纤/交换机/液冷需求↑
- 推导要点：800G→1.6T升级 = ASP翻倍 → 量×价双击 → 这是通信板块当前最大的增量逻辑
- 验证：光模块公司订单能见度（已签合同/在手订单 vs 产能）

**引擎2：运营商投资周期**
- 跟踪变量：三大运营商年度Capex计划、5G基站建设数、算力资本开支占比
- 数据源：运营商年报/月度运营数据 + `news_search("运营商 Capex")`
- 传导链：Capex方向决定设备商/光纤/IDC景气
- 推导要点：5G建设高峰已过 → 传统通信Capex下滑 → 但算力Capex上升可能对冲
- 关键：总Capex的结构变化比总量更重要

**引擎3：低轨卫星（0→1阶段）**
- 跟踪变量：中国星网/G60星链发射计划、卫星制造订单、手机直连卫星进展
- 数据源：`news_search("低轨卫星 星网 发射")` + `research_search("卫星互联网")`
- 传导链：组网加速 → 卫星制造/地面设备/终端需求从0开始放量
- 推导要点：目前是主题投资阶段（概念→订单→收入的0→1过程），财务验证尚远

### 供给侧

- 光模块：产能扩张中（中际/天孚扩产），但需求增速更快 → 短期供不应求
- 光纤光缆：产能过剩历史已久，需求回升后价格弹性大
- 运营商：自然垄断，供给侧几乎无变化 → 纯看需求
- 卫星：供给瓶颈在火箭运力和卫星产能 → 产能建设进度决定放量节奏

### 价格信号

| 指标 | 含义 |
|------|------|
| 光模块ASP（400G→800G→1.6T） | 规格升级=ASP跳升，量价双击验证 |
| 光纤光缆招标均价 | 价格拐点=供需拐点 |
| 运营商ARPU | 收入质量（量增≠价增） |
| IDC机柜租金/上架率 | 数据中心供需 |
