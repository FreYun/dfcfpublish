## 估值方法

| 子赛道 | 推荐估值 | 理由 |
|--------|---------|------|
| 运营商 | PE + 股息率 | 高分红低估值，看DCF |
| 主设备 | PE + 订单 | 周期性，关注海外订单 |
| 光模块 | PE + PEG | 高景气成长 |
| 光纤光缆 | PE + 量价趋势 | 周期品，看价格拐点 |
| IDC/液冷 | PE 或 EV/EBITDA | 重资产，看上架率/利用率 |
| 卫星 | PS（多数早期） | 看星座组网进度 |

## 当前核心主题

1. **AI 算力基建**：800G→1.6T 光模块升级，液冷渗透率提升
2. **运营商重估**：高分红+云业务增长+AI布局，估值修复逻辑
3. **低轨卫星组网**：中国星网+G60星链，卫星制造/地面设备/终端受益
4. **光纤光缆量价齐升**：海外需求+国内算力网络建设
5. **东数西算**：数据中心向西部迁移，IDC/液冷/网络设备需求
