## 估值方法

| 子赛道 | 推荐估值 | 理由 |
|--------|---------|------|
| AI 服务器 | PE + 订单能见度 | 硬件周期，关注毛利率 |
| AI 应用（盈利） | PS + PEG | 收入增速是核心 |
| AI 应用（亏损） | PS | 看收入规模和增速 |
| 信创 | PE（政策+订单周期） | 关注招标节奏 |
| 网络安全 | PS（多数亏损或微利） | 看 ARR 和续约率 |
| 工业软件 | PS + 研发效率 | 长周期，看客户渗透率 |
| ERP/SaaS | PE 或 EV/EBITDA | 关注云收入占比趋势 |

## 当前核心主题

1. **大模型落地**：从"炒算力"到"炒应用"，关注科大讯飞/金山办公的 AI 功能变现
2. **信创2.0**：国产 OS/DB/中间件向行业信创渗透（金融/电力/教育）
3. **数据要素**：数据交易所建设+数据资产入表，利好数据治理/数据安全公司
4. **自主可控升级**：从软件信创延伸到工业软件/EDA 国产化
5. **AI Infra 投资**：云厂商 Capex 增长 → 服务器/液冷/IDC 受益
