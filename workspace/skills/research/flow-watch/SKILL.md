# 北向资金行业轮动追踪 /flow-watch

**触发词**：`/flow-watch`、"北向最近在买什么方向"、"外资在流入哪个行业"、"北向资金行业偏好"、"外资在撤离哪些赛道"

**核心视角：行业/赛道配置变化，而非单只股票的进出。**

---

## 执行流程

**确定 `TODAY`，追踪周期默认 20 个交易日（可指定）。**

**宏观底色**：北向资金方向与全球 risk-on/risk-off 高度相关。若需要解释总量净流入/流出的宏观驱动，可先运行 `/market-environment-analysis`。

---

### Step 1 — 总量环境（Tushare）

```
tushare_moneyflow_hsgt(start_date=20交易日前, end_date=最近交易日)
```

建立宏观底色：
- 区间总净流入（正/负）、日均、流入/流出天数比
- 近5日 vs 近20日趋势对比（动量是在加强还是衰减？）
- 结论：外资整体风险偏好处于**开启 / 收缩 / 反转**哪个状态

> 如果总量都在净流出，行业层面的"相对多"意义大于"绝对流入"。

---

### Step 2 — 行业归因：北向流向哪个赛道（Tushare）

对最近 **5个交易日**，逐日拉取：
```
tushare_hsgt_top10(trade_date=各交易日)
```

**行业归因步骤**：
1. 将十大成交股逐一对应行业标签（TMT / 消费 / 金融 / 医药 / 制造 / 能源等）
2. 按行业汇总：各行业累计净买入额（正 = 买入，负 = 卖出）
3. 识别：连续多日主导北向成交的是哪个行业/赛道？

**行业流向汇总表**：

| 行业/赛道 | 出现次数(最多25) | 净方向 | 累计净额(亿) | 信号强度 |
|---------|----------------|-------|------------|---------|
| TMT/AI | XX | 买入 | +X.X | 强/中/弱 |
| 消费 | XX | 卖出 | -X.X | 中 |

信号强度判断：出现次数多 + 净额大 → 强；仅偶发出现 → 弱

---

### Step 3 — 深挖重点流入行业

对北向净流入最多的 **1-2个行业**，进一步确认：

- 该行业代表性个股（来自十大名单中）当前估值：`tushare_daily_basic`
- 近60日股价表现：`tushare_daily` → 判断是**逆势吸筹**（下跌中买）还是**趋势追入**（上涨中买）
  - 逆势吸筹 = 外资认为已跌出价值，信号较强
  - 上涨中追入 = 动量交易，需看基本面是否支撑

若用户指定了行业（如 `/flow-watch 半导体`），重点分析该行业内北向的具体行为。

---

### Step 4 — 背景与解读（browser + browser，⚠️ 时间验证）

针对 Step 2 识别出的重点流入/流出行业，用 browser 和搜索了解市场背景。**两者同步进行。**

**browser 浏览**

重点了解：外资买入/卖出该行业背后的逻辑、近期有无触发事件。**雪球是最重要的站点**，可搜索行业名称或直接浏览龙头股主页查看市场解读。根据研究需要自由访问其他相关页面。

**browser 搜索**：
1. `"北向资金 外资 行业轮动 {当前年月}"`
2. `"人民币汇率 近期 走势"`（汇率是先行指标）
3. 针对重点流入行业：`"{行业} 外资 配置逻辑 {当前年月}"`

**⚠️ 时间验证**：
```
1. 每条资讯锁定发布日期
2. 标注：🟢 ≤7天 / 🟡 8-30天 / 🔴 >30天
3. 资讯中的具体流入数字，与 Tushare 实际数据比对核实
4. "近期外资大举买入XX行业"必须确认是哪几天的数据
```

**逻辑核查**：
- 北向流入方向是否与该行业基本面方向一致？（基本面好 + 外资买 = 双重验证）
- 如果北向买入一个基本面较差的行业 → 可能是技术性反弹交易，不一定是价值配置
- 汇率走势与整体北向方向是否吻合？

---

### Step 5 — 逻辑核查（输出前）

在输出报告前，快速检查核心判断：

- **总量与行业方向一致吗**：如果总量净流出，某行业的"净流入"是绝对值还是相对值？结论有没有混淆？
- **信号强度有没有夸大**：连续多日同向才是配置信号；单日大额或偶发出现不能说"持续买入"
- **基本面匹配度有没有来源**：说"外资看好该行业基本面"，有搜索或 browser 内容支撑吗？还是只是推断？
- **时效**：背景资讯中有没有 🔴/⚫ 的内容被当作当前信号引用？

发现问题 → 回到对应步骤补充，不强行输出未支撑的判断。

## 注意事项

- **行业归因是核心**：单只股票的进出信息量有限，汇总到行业层面才有配置含义
- **总量背景先看**：全局净流出时，行业内部的"相对多"才是真信号，绝对流入才是强信号
- **区分配置型 vs 交易型**：连续多日同向 = 配置；单日大额 = 可能是对冲或调仓，信号弱
- **不过度解读**：北向是参考，不是神谕；与国内资金行为和基本面结合才有意义
- **十大成交不等于净买入**：高频出现在十大可能是双向交易，看 net_amount 方向才准
