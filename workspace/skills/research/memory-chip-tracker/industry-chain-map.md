## AI 存储分层协同

```
AI 训练/推理存储架构：

┌─────────────────────────────────────────────────────────┐
│                    GPU / AI 加速器                        │
│  ┌──────────────────────────────────────┐                │
│  │           HBM（计算态）               │                │
│  │  • 高带宽工作内存，直接参与计算       │                │
│  │  • 带宽：HBM4 → 2 TB/s              │                │
│  │  • 容量：48-96 GB/栈                 │                │
│  └──────────────────────────────────────┘                │
│                      ↕                                    │
│  ┌──────────────────────────────────────┐                │
│  │          DRAM（调度态）               │                │
│  │  • CPU侧数据加载、预处理、缓存       │                │
│  │  • 解耦I/O与计算，提升GPU利用率       │                │
│  │  • DDR5 RDIMM 64GB → $620           │                │
│  └──────────────────────────────────────┘                │
│                      ↕                                    │
│  ┌──────────────────────────────────────┐                │
│  │       NAND / SSD（存储态）            │                │
│  │  • 训练数据集、模型 Checkpoint        │                │
│  │  • KV Cache 溢出层（推理场景）        │                │
│  │  • 企业级 NVMe SSD                   │                │
│  └──────────────────────────────────────┘                │
│                      ↕                                    │
│  ┌──────────────────────────────────────┐                │
│  │       HBF（高带宽闪存，新品类）       │                │
│  │  • 介于 HBM 与 SSD 之间              │                │
│  │  • AI推理场景容量扩展 + 能效          │                │
│  │  • SK海力士 + Sandisk 推标准化        │                │
│  └──────────────────────────────────────┘                │
└─────────────────────────────────────────────────────────┘

三者分层协同，缺一不可，功能上难以相互替代。
```

## 核心价格传导链

| 传导链 | 机制 | 分析要点 |
|--------|------|---------|
| HBM 需求 → DRAM 产能挤占 → DDR4/DDR5 缺货 | 原厂将 >20% DRAM 产线转向 HBM | HBM 越火 → 传统 DRAM 越缺 → DDR4 涨幅反超 DDR5 |
| AI 服务器 Capex → 企业级 SSD 需求 → NAND 涨价 | CSP 优先抢占配额 | 企业级挤压消费级产出 → 消费级也缺货 |
| DRAM/NAND 涨价 → 模组厂利润暴增 | 模组厂低买高卖 + 战略囤货 | 佰维/江波龙业绩弹性巨大（佰维1-2月利润 +922%~1086%） |
| 存储扩产 → 半导体设备需求 | CXMT+YMTC 设备采购 ~$1100-1200亿 | 国产设备公司直接受益 |
| NOR Flash 独立周期 | 寡头格局 + 车规/HBM4 新需求 | CR3=90.7%，兆易创新定价权强 |
| 地缘政治 → 国产替代加速 | 美国出口管制 → 自主可控 | 消费级替代窗口打开（原厂聚焦 HBM/高端） |
