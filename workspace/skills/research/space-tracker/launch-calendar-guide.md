## 数据源优先级

按以下顺序获取发射数据，**优先用维基百科**，失败再降级：

### 🥇 首选：维基百科 中国航天发射列表

```
URL: https://zh.wikipedia.org/wiki/{year}年中国航天发射列表
示例: https://zh.wikipedia.org/wiki/2026年中国航天发射列表
```

**使用方式**：用 bot11 浏览器**正常打开网页阅读**，人工方式浏览表格内容，提取发射记录。

```bash
# 打开维基百科发射列表
openclaw browser navigate "https://zh.wikipedia.org/wiki/2026年中国航天发射列表" --browser-profile bot11
# 等待页面加载
openclaw browser wait --text "发射" --browser-profile bot11
# 截图查看页面内容
openclaw browser screenshot --browser-profile bot11
# 或用 snapshot 获取页面文本
openclaw browser snapshot --browser-profile bot11
```

通过阅读页面内容（截图 / snapshot），手动整理发射记录到 `launch-calendar.md`。

**表格通常包含的列**：序号、日期与时间(UTC+8)、火箭(及型号)、载荷(及任务)、发射场(及工位)、结果。

**超时处理**：如果 1 分钟内页面未加载完成，立即切换到备用源。

### 🥈 备用1：联网搜索聚合

维基百科不可用时，用多关键词搜索拼凑：

```
web_search("中国航天 发射计划 2026 最新")
web_search("商业航天 发射日历 本周")
web_search("chinaspaceflight.com 发射 2026")
```

关键信息源网站（可在搜索结果中点击访问）：
- `chinaspaceflight.com` — 中国航天爱好者网站，发射记录详细
- `spaceflightnow.com/launch-schedule` — 全球发射日历（英文）
- `nextspaceflight.com/launches` — 全球发射日历（英文，含中国）

### 🥉 备用2：知识星球/雪球补充

从黑马调研、雪球搜索 `发射` `首飞` 等关键词，提取非公开的发射窗口信息（如券商研报中的预测）。这类信息时效性强但准确度不如官方源，标注为 🟡 待定。

## 更新规则

- **首次建表**：打开维基百科页面阅读，一次性整理全年已发射 + 计划发射列表
- **每日巡逻**：打开维基百科检查是否有新增发射记录或计划变更，页面打不开则用搜索补充
- 发射完成后更新状态：✅ 成功 / ❌ 失败 / 🟡 部分成功
- 移入历史发射记录区
- **发射前 3 天**和**发射当天**发飞书提醒
- 每周日周报时，核对一次完整发射日历的准确性
