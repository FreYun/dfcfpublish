# 产业链地图

## 三大链路全景

```
┌─────────────────── 石油化工链 ───────────────────┐
│                                                    │
│  原油 → 石脑油 → 乙烯 → PE(聚乙烯) / EG(乙二醇)   │
│                       → 苯乙烯(EB) / PVC(乙烯法)    │
│                 → 丙烯 → PP(聚丙烯) / 丙烯酸        │
│                 → 芳烃(BTX) → PX → PTA → 聚酯       │
│           → 燃料油 / 沥青 / LPG                      │
│                                                    │
└────────────────────────────────────────────────────┘

┌─────────────────── 煤化工链 ────────────────────┐
│                                                   │
│  煤 → 甲醇 → MTO → PP / PE（煤制烯烃路线）        │
│            → 甲醛 / MTBE / DME                    │
│     → 合成氨 → 尿素（化肥）                       │
│     → 电石 → PVC（电石法，占中国PVC ~80%）         │
│     → 煤制乙二醇（煤制EG路线）                     │
│                                                   │
└───────────────────────────────────────────────────┘

┌─────────────────── 盐化工链 ────────────────────┐
│                                                   │
│  盐 → 电解 → 烧碱(NaOH) + 氯气(Cl2)              │
│              → 氯气 → PVC / 盐酸 / 有机氯          │
│     → 纯碱(Na2CO3) → 玻璃 / 光伏                  │
│                                                   │
│  ⚡ 氯碱平衡：PVC消耗氯，烧碱联产，                 │
│     PVC需求↑ → 氯消耗↑ → 烧碱被动增产 → 烧碱价格↓  │
│     PVC需求↓ → 烧碱减产 → 烧碱供紧 → 烧碱价格↑     │
│                                                   │
└───────────────────────────────────────────────────┘
```

## 跨链联动关系（核心分析逻辑）

| 联动关系 | 传导机制 | 分析要点 |
|----------|---------|---------|
| 原油 ↔ PTA/EG/EB | 石脑油成本传导 | 油价涨 → 石脑油涨 → 石化品成本抬升 |
| 煤 ↔ 甲醇 ↔ PP/PE (MTO) | 煤价驱动甲醇成本，甲醇驱动MTO经济性 | MTO利润 = PP价格 - 3×甲醇价格 |
| 煤 ↔ 电石 ↔ PVC | 电石法PVC占中国80%产能 | 煤价+电价 → 电石成本 → PVC成本底 |
| PVC ↔ 烧碱 | 氯碱平衡：联产关系 | 一方强另一方弱，跷跷板效应 |
| PTA + EG → 聚酯 | 两者汇聚于聚酯 | PTA价格变动影响EG需求，反之亦然 |
| 尿素 ↔ 煤 + 天然气 | 原料成本 + 农业季节性需求 | 春耕/秋季追肥是需求高峰 |
| 纯碱 ↔ 玻璃 ↔ 地产/光伏 | 需求传导 | 地产压平板玻璃，光伏推光伏玻璃 |
| 油价 vs 煤价 | 决定油头 vs 煤头竞争格局 | 油贵煤便宜 → 煤化工利润好 |

## 成本比较框架（同一产品不同路线）

| 产品 | 油基路线 | 煤基路线 | 气基路线 | 分析意义 |
|------|---------|---------|---------|---------|
| PP | 石脑油裂解/PDH | 煤→甲醇→MTO | — | 边际成本决定价格底 |
| PE | 石脑油裂解 | 煤→甲醇→MTO | — | 同上 |
| PVC | 乙烯法(~20%) | 电石法(~80%) | — | 电石法主导定价 |
| EG | 石脑油→环氧乙烷 | 煤制EG | 天然气→EG | 三路线竞争激烈 |
| 甲醇 | — | 煤制甲醇(国内主) | 气制甲醇(进口主) | 进口依赖度~15% |
