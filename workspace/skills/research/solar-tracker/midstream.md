## 电池片供需框架

### 需求侧
- **核心驱动**：组件排产量（组件厂自有电池产能不足的部分=外采需求）
- 跟踪：组件厂排产计划、一体化率（自有电池/总组件产量）
- 数据源：`research_search("电池片 排产 开工率")` + `news_search("电池 TOPCon 价格")`
- 推导：一体化率↑ → 外采电池需求↓ → 独立电池厂承压（通威/钧达）

### 供给侧
- **技术路线是核心变量**：
  - PERC：成熟技术，效率天花板（~23.5%），产能大量过剩，正在被替代
  - TOPCon：当前主流新技术（~25.5%），产能快速扩张中
  - HJT：效率略高但设备贵，产能扩张较慢
  - BC（背接触）：隆基/爱旭押注，差异化但量产规模小
- 供给推导：TOPCon产能大幅扩张 + PERC产能未退出 → 电池总产能严重过剩 → 价格承压
- **出清逻辑**：PERC产能退出速度决定电池环节何时触底。当PERC毛利率为负 → 被迫关停 → 供给收缩

## 组件供需框架

### 需求侧
- **核心驱动**：全球光伏新增装机量(GW)
- 跟踪：中国/欧洲/美国/印度/中东月度装机数据、组件出口数据（海关）
- 数据源：`research_search("光伏 装机 组件出口")` + `news_search("光伏装机 GW")`
- 推导：装机增速>组件产能增速 → 需求偏紧；反之 → 过剩
- 地域结构：中国占比↑ → 价格压力（低价市场）；海外占比↑ → 利润改善（高溢价）

### 供给侧
- 组件产能极度过剩（全球产能~1200GW vs 需求~500GW）→ 长期看出清
- **竞争格局**：前五（隆基/晶科/天合/晶澳/阿特斯）份额合计~60%，集中度在提升
- 品牌+渠道壁垒 → 龙头可以维持溢价，二三线只能拼价格

### 价格信号
| 产品 | 数据源 |
|------|--------|
| 双面双玻 TOPCon 组件(元/W) | InfoLink/PVInfoLink |
| 组件出口均价(美元/W) | 海关/研报 |
| 组件招标均价(元/W) | 北极星光伏/新闻 |

- 国内招标价持续新低 → 内卷加剧
- 出口均价企稳/回升 → 海外需求韧性
- 龙头与二线价差 → 品牌溢价是否存在
