## 终端装机需求框架

### 全球装机需求
- **核心驱动**：光伏度电成本(LCOE) vs 传统能源电价 → LCOE越低，经济性越强，装机意愿越高
- 跟踪变量：月度/季度新增装机(GW)、光伏发电量同比、并网项目核准数
- 数据源：`research_search("光伏 装机 新增 GW")` + `news_search("光伏 并网 装机")`
- 地域拆分：
  - **中国**：政策驱动（整县推进、大基地项目）+ 经济性驱动（工商业分布式）
  - **欧洲**：能源安全+碳中和，户用为主
  - **美国**：IRA补贴+关税政策（对中国组件），利好本土制造
  - **新兴市场**（中东/印度/东南亚/拉美）：增量最快，但价格敏感

### 推导逻辑
- 组件价格↓ → LCOE↓ → 项目IRR↑ → 装机意愿↑（量的弹性）
  - 但组件价格↓也可能反映需求不足 → 必须区分"降价刺激需求"和"需求不足导致降价"
  - 区分方法：装机量同比增速 + 组件库存变化 → 量增+库存降=真需求；量平+库存升=假刺激
- 电网消纳能力 = 装机天花板。弃光率上升 → 即使项目经济性好也装不了

## 设备商供需框架

### 逆变器（光伏设备最优赛道）
- **为什么逆变器赛道好**：
  1. 技术壁垒高（电力电子+软件算法+认证周期长）
  2. 产品差异化强（不像硅片/电池是标准品）
  3. 海外高毛利（欧美市场逆变器溢价30-50%）
  4. 储能延伸（逆变器技术通用于储能PCS）
- 跟踪变量：逆变器出口金额/数量、海外市占率、储能PCS出货量
- 数据源：`research_search("逆变器 出口 出货量")` + `news_search("阳光电源 锦浪 逆变器")`
- 推导：光伏装机量 × 逆变器单W价值 = 逆变器市场规模。关注单W价值变化（功率密度提升→单W降价 vs 储能功能增加→单W升价）

### 光伏玻璃（周期品特征最明显）
- 跟踪变量：光伏玻璃价格(元/㎡)、产能利用率、库存天数
- 数据源：`news_search("光伏玻璃 价格")` + `research_search("光伏玻璃 产能")`
- 推导：双玻渗透率↑ → 单位组件用玻璃量↑ → 需求增速>装机增速
- 供给：产能投放节奏（点火/冷修周期约8年，进出门槛高）→ 寡头格局（福莱特+信义占比>50%）

### 胶膜/焊带（辅材）
- 跟踪：POE胶膜渗透率（N型电池需POE，P型用EVA）、EVA/POE粒子价格
- 推导：N型渗透率↑ → POE需求结构性增长 → 福斯特受益
- 焊带：SMBB（多主栅）技术 → 单位用量变化
