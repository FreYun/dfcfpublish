# 光伏全链跟踪 /solar-tracker

**触发**：`/solar-tracker`、"光伏现在怎么样"、"硅料/组件价格"、"光伏产业链景气"、"N型电池进展"

## 产业链总览

```
硅料(多晶硅) → 硅片 → 电池片 → 组件 → 电站(地面/分布式)
                                          ↑
                              设备商(逆变器/支架/玻璃/胶膜/焊带)
```

这是一条**价格信号极其丰富**的产业链——每个环节都有周度报价，供需推导可实时验证。

## 铁律

1. **光伏是周期制造业**：供需决定价格，价格决定利润，利润决定股价。别被"碳中和长期故事"干扰短周期判断
2. **价格是最诚实的信号**：硅料/硅片/电池/组件每周都有报价（InfoLink/PVInfoLink/硅业分会），价格方向直接验证供需推导
3. **产能过剩是常态**：光伏行业每2-3年一轮过剩→出清→再扩张。当前在周期什么位置比长期空间更重要
4. **技术迭代改变成本曲线**：PERC→TOPCon→HJT→BC，新技术 = 成本曲线下移 = 老产能加速出清

## 执行流程

### Step 1 — 周期定位：光伏当前在哪个阶段？

光伏产能周期：**扩产→过剩→亏损出清→供需平衡→涨价→再扩产**

**数据拉取（并行）：**

| 数据 | 工具 |
|------|------|
| 光伏产业指数近60日 | `tushare_index_daily(ts_code='931151.CSI', start_date='<60日前>', end_date='<今日>')` |
| 沪深300近60日 | `tushare_index_daily(ts_code='000300.SH', start_date='<60日前>', end_date='<今日>')` |
| 光伏研报 | `research_search("光伏 产业链 价格 景气", top_k=5, search_day_ago=30)` ← 供需逻辑/产能判断 |
| 光伏新闻 | `news_search("光伏 硅料 组件 价格", top_k=5, search_day_ago=7)` ← 近期价格动态 |
| browser（实时价格）| `browser_search("硅业分会 多晶硅价格 本周")` 或 `browser_search("InfoLink 光伏产业链价格周报")` |

**周期定位框架：**

| 阶段 | 特征 | 投资含义 |
|------|------|---------|
| 扩产高峰 | 产能投产加速，价格开始松动 | 卖出信号 |
| 过剩期 | 价格持续下跌，龙头毛利压缩 | 回避，等出清 |
| 亏损出清 | 二三线停产/破产，产能退出 | 关注拐点 |
| 供需再平衡 | 价格企稳，库存去化 | 左侧布局窗口 |
| 涨价期 | 供不应求，价格上行 | 确认右侧 |

→ 输出：当前处于 [阶段]，依据 [价格数据+产能数据]

### Step 2 — 全链价格扫描（核心步骤）

**光伏最大的分析优势 = 每个环节都有周度报价。**

**数据源优先级：browser（实时）> news_search（近7日）> research_search（近14日研报）**

**具体工具调用方式：**
- **browser**（优先）：`browser_search("<产品> 价格 本周")`；实时价格首选来源：硅业分会（sic.org.cn）、InfoLink（infolink-group.com）、北极星光伏
- **news_search**（browser 失败/补充）：`news_search("<产品> 价格", top_k=5, search_day_ago=7)`
- **research_search**（交叉验证/缺口填补）：`research_search("光伏 产业链 价格 周报", top_k=3, search_day_ago=14)`

**browser 搜索获取最新价格（必须执行）：**

| 搜索词 | 目标数据 | 参考来源 |
|--------|---------|---------|
| `browser_search("多晶硅 致密料 价格 本周")` | 硅料价格+趋势 | 硅业分会 / InfoLink |
| `browser_search("硅片 182 210 价格 本周")` | 硅片价格+趋势 | InfoLink / PVInfoLink |
| `browser_search("电池片 TOPCon 182 价格 本周")` | 电池片价格+趋势 | InfoLink |
| `browser_search("光伏组件 TOPCon 双玻 报价 本周")` | 组件价格+趋势 | InfoLink / 北极星光伏 |

**research-mcp 补充（browser 失败时，或作为交叉验证）：**
```
research_search("光伏 产业链 价格 周报", top_k=3, search_day_ago=14)
news_search("硅料 硅片 电池 组件 价格", top_k=5, search_day_ago=7)
```

**输出价格链表：**

| 环节 | 产品 | 当前价格 | 周变动 | 月变动 | 趋势 |
|------|------|---------|--------|--------|------|
| 硅料 | 致密料(元/kg) | | | | ↑↓→ |
| 硅片 | 182mm P型/N型(元/片) | | | | |
| 电池 | TOPCon 182(元/W) | | | | |
| 组件 | 双面双玻 182(元/W) | | | | |

**推导逻辑：**
- 全链价格同向下跌 → 终端需求不足 or 产能过剩传导
- 上游跌+下游稳 → 上游过剩，下游受益（利润向下游转移）
- 上游稳+下游跌 → 组件端恶性竞争
- 全链企稳/上涨 → 供需再平衡确认

### Step 3 — 供需推导（按环节）

**子文件调用规则：**

| 触发方式 | 读取哪些子文件 | 深度 |
|---------|--------------|------|
| **全景模式**（默认，无特定指定）| 并行 `read_file` 全部三个子文件（`upstream.md`、`midstream.md`、`downstream.md`），每个仅输出"推导模板"部分快速汇总 | 快扫（每环节3-5行结论）|
| **聚焦模式**（用户指定具体环节，如"硅料"/"TOPCon电池"/"逆变器"）| 仅 `read_file` 对应子文件深入推导，其余两个环节一句话带过，无需读取 | 深入（逐指标展开）|
| **对比模式**（如"上游 vs 中游利润"）| `read_file` 指定的两个子文件，第三个一句话带过 | 中等（对比结构输出）|

子文件与环节对应：
- `upstream.md` — 硅料 + 硅片（含光伏设备商）→ 调用：`read_file("upstream.md")`
- `midstream.md` — 电池片 + 组件 → 调用：`read_file("midstream.md")`
- `downstream.md` — 电站装机 + 逆变器/玻璃/胶膜等辅材设备商 → 调用：`read_file("downstream.md")`

每环节必须回答：
1. 产能：在产+在建+规划，年底总产能多少？
2. 需求：对应的下游拉动多少？（硅料看硅片开工率，组件看装机量）
3. 库存：当前库存天数，去化还是累积？
4. 价格验证：与 Step 2 的价格方向是否一致？

### Step 4 — 盈利验证

**Tushare 批量并行（全链龙头 8-12 家）：**

| 工具 | 用途 |
|------|------|
| `tushare_daily_basic(ts_code='<代码>', trade_date='<最近交易日>')` | PE/PB/市值 |
| `tushare_stock_daily(ts_code='<代码>', start_date='<60日前>', end_date='<今日>')` | 涨跌幅 |
| `tushare_income(ts_code='<代码>', period='<最近报告期>')` | 营收/净利润增速（取最近2期对比）|

**research-mcp：** `get_stock_valuation(ts_code='<代码>')` 获取估值历史分位

**推导：**
- 毛利率方向 → 验证价格链判断（价格跌+毛利率跌=一致；价格跌+毛利率稳=成本下降更快or库存延迟）
- 营收增速 → 验证需求端（量增价跌→以价换量；量价齐跌→真衰退）
- **PB 是光伏核心估值指标**（重资产周期股），PE 在亏损期失效
- PB < 1 + 龙头仍盈利 → 可能是底部区域

输出横向表：

| 环节 | 公司 | 代码 | 市值 | PB | PE | 毛利率趋势 | 营收增速 | 近3M |
|------|------|------|------|----|----|-----------|---------|------|

**政策（browser + research-mcp）：**

- 贸易政策（欧美关税/东南亚产能转移）→ 影响出口需求
- 每条政策回答：改变了哪条曲线？方向？定价程度？

## 异常降级

| 场景 | 处理 |
|------|------|
| 价格数据获取失败 | 标注数据缺失，用研报价格替代 |
| Tushare某标的无数据 | 标N/A |
| 非交易日 | 用最近交易日 |
| 子文件读取失败（upstream/midstream/downstream.md）| 在 SKILL.md 供需框架中按"产能→需求→库存→价格验证"做简要推导，不中断流程；报告中注明"[子文件未加载，基于概要推导]" |
| browser 无法访问 InfoLink/硅业分会 | 降级调用 `news_search("硅料 硅片 电池 组件 价格", top_k=5, search_day_ago=7)`，标注"[价格来源：资讯，非实时周报]" |
| research_search/news_search 返回空 | 扩大 `search_day_ago`（7→14→30），或更换关键词（如"光伏 价格"替代"InfoLink 周报"）|
| tushare_income 无最新期数据 | 取上一期财报，报告中注明数据截止期 |

## 注意

- **PB > PE**：光伏是重资产周期股，PB是核心估值锚，亏损期PE失效
- **价格趋势 > 绝对值**：硅料从300跌到40不重要，重要的是40是否企稳
- **出清信号比涨价信号更早**：关注二三线企业停产/破产新闻 → 领先于价格拐点
- **技术路线是长期α**：TOPCon/HJT/BC路线选择影响公司3-5年命运，但短期看供需周期
