## 核心铁律：推导 ≠ 总结

**每个结论 = 数据 + 经济学逻辑 → 结论**

| ❌ 禁止 | ✅ 要求 |
|--------|--------|
| "半导体行业机遇与挑战并存" | 产能利用率72%（↓13pct QoQ）+ DRAM报价连跌3月 → 供过于求 → PE 45x隐含30%增速难兑现 → 估值下修压力 |
| "研报普遍看好AI" | 云厂商Capex同比+42% + 光模块交期拉长至16周 → 算力需求仍在扩张 → 供给偏紧 → 光模块ASP支撑 |
| "消费电子有望回暖" | Q3全球智能手机出货量同比+6%（IDC）+ MLCC价格环比持平 → 温和复苏，非强周期反转 |

推导链缺失 = 结论无效，必须补数据重推。

## 七步流程概览

### Step 1 — 定价环境
**核心问题**：当前TMT定价由哪个变量主导？

TMT定价 = f(利率, 风险偏好, 盈利周期)

推导链：
1. 10Y国债方向 → DCF折现率 → 成长股估值弹性方向
2. TMT超额收益（vs沪深300）+ 成交额占比 → 资金偏好方向
3. PE分位 vs 盈利增速方向 → 估值领先还是滞后

**输出**：当前TMT定价由 [利率/风偏/盈利] 主导

---

### Step 2 — 四方向景气推导
**核心问题**：每方向供给增速 vs 需求增速，谁快？

每方向必须输出：
1. 需求变量 + 数据 → 方向和斜率
2. 供给变量 + 数据 → 方向和斜率
3. 价格信号 → 验证供需判断
4. 景气结论 + 推导链

**四方向需求引擎速查**：

| 方向 | 主要需求引擎 | 关键价格信号 |
|------|------------|------------|
| 电子 | AI算力Capex / 换机周期 / 国产替代 | DRAM合约价 / 光模块ASP / 晶圆代工报价 |
| 计算机 | AI Infra投资 / 信创替代 / AI应用变现 | AI服务器ASP / SaaS续约率 / 信创客单价 |
| 通信 | 运营商Capex / AI算力基建 / 卫星商用 | 光模块交期 / 光纤光缆价格 / IDC上架率 |
| 传媒 | 游戏流水/爆款 / 广告预算 / AIGC商业化 | 游戏ARPU / 广告CPM / AIGC订阅价 |

---

### Step 3 — 盈利验证
**核心问题**：景气推导（应然）与财务数据（实然）是否对齐？

| 财务指标 | 验证逻辑 |
|---------|---------|
| 营收增速 | 验证需求端；景气上行但营收放缓 → 推导有误或份额转移 |
| 毛利率方向 | 验证供需；扩张=供给偏紧，收缩=产能过剩 |
| PE vs 盈利增速 | PE高+增速放缓 = 危险；PE高+增速加速 = 合理溢价 |

**每方向选2-3个龙头**，横向对比：估值、毛利率趋势、营收增速、估值匹配度。

---

### Step 4 — 资金行为解读
**核心问题**：资金为什么这样流动？

资金方向是现象，不是结论。必须回答"为什么"：

- 北向增持 → 估值洼地？盈利确定性？全球产业链受益？
- 主力净流入 → 与基本面推导方向是否一致？
- 北向 vs 内资分歧 → 谁的逻辑更硬？（一般北向更看重全球比较优势，内资更看政策节奏）

---

### Step 5 — 政策：供需曲线的位移器
**核心问题**：政策改变了谁的供给/需求曲线？改变了多少？

每条政策必须回答：
1. 改变了供给曲线还是需求曲线？方向和幅度？
2. 市场定价了多少？（发布日 vs 股价反应时点的对比）

政策 ≠ 催化剂清单，是改变供需平衡的外力。未定价的政策才是α。

---

### Step 6 — 逻辑自洽检验

| 检查项 | 一致条件 | 矛盾处理 |
|-------|---------|---------|
| 景气 vs 毛利率 | 景气上行 → 毛利率稳定/扩张 | 不一致 → 重查供需数据 |
| 估值 vs 盈利 | 说便宜 → 当前增速 > PE隐含增速 | 不一致 → 修正估值判断 |
| 资金 vs 基本面 | 方向应一致 | 不一致 → 分析分歧来源，不模糊掩盖 |

矛盾不可掩盖，必须回补数据或修正推导链。

---

## 方法论原则

### 关于估值
- TMT高估值常态 → **用PE vs 盈利增速匹配度**，不用PE绝对值评判
- PE隐含增速 = 市场共识预期，实际增速超/低于此 = 估值重估触发器

### 关于子行业轮动
- 轮动本质 = 景气差（各方向景气周期不同步）
- 全景模式核心价值 = 识别景气差 → 判断轮动方向
- 全景 → 发现机会 → 聚焦 → 深度验证

### 关于研报使用
- 研报是**数据源**，不是**结论源**
- 提取：出货量、产能利用率、Capex金额、毛利率数据
- 不直接引用：研报观点、评级、目标价（需自己推导）

### 关于传媒特殊性
- 传媒（尤其游戏）是**产品驱动**行业，传统供需框架解释力有限
- 爆款上线 → 短期流水脉冲（公司级景气，不代表行业景气）
- 需结合产品管线密度 + 版号节奏 + 出海流水综合判断

## 异常降级规则

| 场景 | 处理方式 |
|------|---------|
| 数据缺失 | 标注 N/A + 说明推导链不完整 |
| 非交易日 | 使用最近交易日数据，标注日期 |
| 研报不足 | browser补雪球/东方财富/wind资讯 |
| 北向数据不可用 | 跳过北向分析，仅用主力资金流向；标注"北向数据不可用" |
| 价格信号矛盾 | 不强行圆场，回补数据或标注"供需判断待验证" |
