# 内容输出模板

> 发布前必须阅读 `quality-checklist.md`

---

## 每日快讯（飞书/小红书）

**触发**：碳酸锂价格波动 > 2%，或锂电板块出现显著异动

**标题模板**：
- `🔋 碳酸锂突破16万！锂电产业链全线上涨`
- `⚡ 六氟暴涨30%！电解液涨价潮要来了？`
- `📊 宁德时代出货创新高，储能成第一增长极`

**内容结构**（200-300字）：
```
[开头] 今天锂电圈最值得关注的1件事
[数据] 关键价格变动（不超过3个品种）
[逻辑] 为什么涨/跌？背后的供需逻辑
[关联] 对上下游的传导影响
[展望] 短期怎么看
[互动] 引导评论
```

**Tags**：`#锂电池` `#碳酸锂` `#新能源` `#{具体品种/公司}` `#投资` `#产业链`

## 周报（公众号文章）

**结构**：
```markdown
## 本周概览

### 🔋 碳酸锂
- 现货：XX万元/吨（±X.X%），期货：XX万元/吨
- [核心驱动：供给扰动/需求变化/库存方向]

### 🧪 四大材料表现

| 材料 | 周涨跌 | 核心驱动 |
|------|--------|---------|
| 六氟磷酸锂 | ±X.X% | [原因] |
| 磷酸铁锂 | ±X.X% | [原因] |
| ... | | |

### 📦 库存与排产
- 碳酸锂库存：XX万吨（↑/↓）
- 下游排产：3月环比 +XX%

### 🏭 产能与事件
- [本周投产/停产/检修]
- [政策变动/技术突破]

### 📈 板块行情
- 锂电池板块周涨跌：±X.X%
- 核心标的表现：[TOP3涨/跌]

### 🔮 下周关注
- [待发布数据]
- [催化事件]
```

## 品种深度（公众号文章）

**触发**：品种出现重大变化，或积累了足够的跟踪数据

**结构**：
```markdown
# [品种]产业链全解：当前在哪，未来怎么走

## 是什么（一句话说清楚）
## 产业链位置
## 供给分析（产能/开工率/新增/检修）
## 需求分析（下游结构/季节性/新增场景）
## 成本分析（各路线成本对比）
## 库存周期
## 竞争格局（CR3/CR5/龙头）
## 核心矛盾
## 观点与展望
## 风险提示
```
