# 个股定位 /stock-locator

**触发词**：`/stock-locator`、"帮我看看XX现在在什么位置"、"XX处于什么阶段"、"分析一下我持仓的XX"、"这只票现在贵不贵"

**定位**：对单只股票进行客观的多周期位置定位，回答"这只股票现在走到哪了"。可独立使用，也可作为持仓审视、加减仓决策、行业对比的前置分析。

---

## 铁律：成本价与分析无关

> **你的买入成本、持仓盈亏、浮盈浮亏，与这只股票会涨还是会跌没有任何因果关系。**

在执行本 skill 时：

1. **禁止**用"持仓亏了X%所以应该补仓"、"已经赚了X%该止盈了"这类逻辑
2. **禁止**将成本价标注在支撑/阻力位中（成本价不是市场认可的技术位）
3. **必须**先完成客观定位，再在最后单独一节讨论持仓策略（如需要的话）
4. **区分两个问题**：
   - "这只股票值不值得在当前价格持有？" → 由本 skill 的客观分析回答
   - "我这笔交易赚了还是亏了？" → 这是账户管理问题，不是分析问题

## 数据获取

### 必须获取的数据

按需选用可用的数据工具（tushare MCP、browser、research-mcp 等）：

```
# 行情数据（核心）
tushare_daily(ts_code, start_date=3年前)       → 日K线，用于计算位置和趋势
tushare_weekly(ts_code, start_date=5年前)       → 周K线，用于判断中长期结构
tushare_daily_basic(ts_code, 最近交易日)        → PE/PB/市值/换手率

# 均线（从日K线计算或图表读取）
MA20 / MA60 / MA120 / MA250                    → 短中长期均线

# 成交量
近20日日均成交额 vs 近60日日均 vs 近250日日均    → 量能水平变化
```

### 日内实时行情（盘中分析时必须使用）

盘中执行定位分析时，**不能只看昨日收盘数据**，必须获取当前实时行情：

```python
import tushare as ts
pro = ts.pro_api()

# 1. 实时快照 — 当前价、开高低、买五卖五、成交量额
#    注意：用旧版 ts 接口，代码不带后缀（6位纯数字）
df = ts.get_realtime_quotes('000001')         # 单只
df = ts.get_realtime_quotes(['000001','600519'])  # 多只
# 返回字段：name, open, pre_close, price, high, low, bid, ask,
#           volume, amount, b1_v~b5_v, b1_p~b5_p, a1_v~a5_v, a1_p~a5_p,
#           date, time, code

# 2. 实时分钟K线 — 日内走势结构（用 Pro API，代码带后缀）
#    freq 支持：1min / 5min / 15min / 30min / 60min
#    限频：每分钟最多 2 次调用
df = pro.stk_mins(ts_code='000001.SZ', freq='5min')
# 返回字段：ts_code, trade_time, open, high, low, close, vol, amount
# 返回约7个交易日的分钟数据
```

**日内实时行情的用途**：
- 获取当前最新价格，而非用昨日收盘价做定位
- 通过分钟K线判断日内走势结构（是高开低走还是低开高走、有无放量异动）
- 买五卖五数据反映当前多空即时力量对比
- 日内成交额与历史日均对比，判断今日量能水平

### 建议获取的数据

```
# 筹码与资金
tushare_hsgt_top10(近10个交易日)               → 北向资金动向
tushare_top10_holders(ts_code)                → 股东结构变化

# 估值分位
tushare_daily_basic(ts_code, 近3年)            → 计算PE/PB历史分位数

# 板块上下文
browser 搜索同板块个股近期表现                   → 判断是个股行情还是板块行情
```

## 分析框架：三层定位法

### 第一层：位置感 — 当前价在历史中的什么位置

这是最基础也是最重要的判断。**不是看1个月涨跌幅，是看3-5年视角下当前价处于什么区间。**

| 指标 | 计算方法 | 含义 |
|------|---------|------|
| **3年价格分位** | (当前价 - 3年最低) / (3年最高 - 3年最低) × 100% | 0-20%=底部区域, 80-100%=顶部区域 |
| **5年价格分位**（如有足够数据） | 同上，5年窗口 | 更长周期的位置感 |
| **距离历史高点** | (当前价 - 历史高点) / 历史高点 × 100% | 腰斩=-50%，脚踝=-80% |
| **距离250日线** | (当前价 - MA250) / MA250 × 100% | 偏离度反映短期极端程度 |
| **PE/PB历史分位** | 当前PE在近3年PE序列中的百分位 | 估值是贵还是便宜 |

**位置区间判定**：

```
90-100%  极度高位 — 历史性顶部区域，风险极高
70-90%   高位区 — 上方空间有限，需要强催化才能继续
40-70%   中位区 — 不上不下，方向取决于趋势和催化
20-40%   低位区 — 历史偏低，如有催化可能是机会
0-20%    极度低位 — 但低可以更低，需判断是底部还是下跌中继
```

### 第二层：趋势与阶段 — 当前在往哪个方向走、走到第几步了

用多周期均线叠加判断：

| 维度 | 看什么 | 怎么判断 |
|------|--------|---------|
| **短期趋势** | MA20 方向 + 价格与MA20关系 | MA20向上且价在其上=短期多头 |
| **中期趋势** | MA60 方向 + 价格与MA60关系 | MA60是中线投资者的生命线 |
| **长期趋势** | MA120 / MA250 方向 | MA250向上=长期牛股特征 |
| **均线排列** | MA20/60/120/250的顺序 | 多头排列=趋势向上；空头排列=趋势向下 |
| **量价配合** | 上涨时量能 vs 下跌时量能 | 放量上涨+缩量回调=健康；放量下跌=危险 |

**阶段判定**（关键输出）：

| 阶段 | 典型特征 | 操作含义 |
|------|---------|---------|
| **底部构筑** | 价格在MA250下方走平或缓慢抬升，成交量萎缩到地量，MA逐步粘合 | 左侧机会，需要耐心 |
| **突破启动** | 放量突破MA60/MA120，均线开始由空转多 | 右侧确认信号 |
| **主升浪** | 多头排列，沿MA20/MA60上行，回调不破MA60 | 趋势最强阶段 |
| **加速赶顶** | 远离所有均线，换手率暴增，周线级别大阳线连续 | 情绪极端，风险累积 |
| **高位震荡** | MA20走平/下穿MA60，放量滞涨，阴阳交替 | 趋势动能衰竭 |
| **破位下跌** | 跌破MA120/MA250，均线空头排列，反弹受MA60压制 | 趋势转空，杀伤力大 |
| **阴跌磨底** | 价格在MA250下方，所有均线向下但斜率趋缓，成交低迷 | 还没到底部构筑，时间换空间 |

### 第三层：催化与驱动 — 有没有让它动的理由

| 维度 | 看什么 |
|------|--------|
| **板块热度** | 所属概念/行业近期是市场主线还是冷门？板块指数趋势如何？ |
| **个股 vs 板块** | 个股走势与板块走势一致（跟随）还是独立（有独立逻辑）？ |
| **事件催化** | 近期有无：财报发布、重大合同、政策利好/利空、股东增减持、回购？ |
| **资金信号** | 北向连续流入/流出？大宗交易？融资融券变化？ |
| **分析师动态** | 近期有无评级调整、目标价变化？ |
| **舆情温度** | 雪球/股吧讨论热度和情绪倾向 |

## 注意事项

1. **周期要够长**：位置分析至少看3年，有数据的话看5年。1年太短，容易把阶段性高点误判为历史高位
2. **不是预测**：本 skill 回答"现在在哪"，不是"明天涨不涨"。定位是事实描述，预测是概率推断
3. **低分位 ≠ 该买**：处于历史低位可能是价值陷阱（基本面恶化导致的合理下跌），必须结合催化层判断
4. **高分位 ≠ 该卖**：强趋势股可以持续在高分位运行很久，看趋势阶段是否还在主升浪
5. **个股 vs 大盘**：大盘处于退潮期时，个股的所有技术信号都要打折，系统性风险优先于个股分析
