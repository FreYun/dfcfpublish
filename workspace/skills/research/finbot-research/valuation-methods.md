# 估值方法论详解

## 方法 A：EV/EBITDA 倍数法（置信度 70%）

1. 收集历史 3-5 年 EV/EBITDA 倍数 → 均值（μ）+ 标准差（σ）
2. 目标 EV = EBITDA × μ
3. 股权价值 = EV - 净负债 → 目标价 = 股权价值 / 总股数
4. 区间 = ±1σ 对应价格

适用：盈利稳定、有足够历史数据的成熟公司。不适用亏损公司。

## 方法 B：对标估值法（置信度 60%）

1. 选 2-3 家对标公司（同行业、相似规模、相似增速）
2. 取对标 EV/EBITDA 均值/最小/最大
3. 目标 EV = 公司 EBITDA × 均值 → 区间 = min~max 对应价格

对标选取：同赛道优先，规模差异 <5 倍，增速差异 <10pp。

## 方法 C：DCF 折现法（置信度 50%）

1. 基础 FCF（若无数据，FCF = EBITDA × 60%）
2. 第1-5年增长率（默认10%），第6-10年（默认5%）
3. 终值 = FCF₁₀ × (1+永续增长率) / (WACC-永续增长率)
4. 企业价值 = Σ PV(FCF) + PV(终值)
5. 敏感性区间 = WACC ±1%

| 参数 | 默认 | 调整 |
|------|------|------|
| WACC | 10% | 蓝筹8-9%，成长10-12%，新兴市场12-15% |
| 永续增长率 | 2.5% | 不超过GDP长期增速，通常2-3% |

## 综合加权

目标价 = Σ(方法目标价 × 置信度) / Σ(置信度)

## 估值判断

| 当前价 vs 目标价 | 判定 |
|-----------------|------|
| < 80% | 显著低估，关注安全边际 |
| 80%-100% | 轻度低估，关注执行风险 |
| 100%-120% | 合理估值，持有为主 |
| > 120% | 显著高估，警惕回调 |
