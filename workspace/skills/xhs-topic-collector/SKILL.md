# 小红书素材积累 Skill

独立的素材巡逻与积累 skill。**平时多攒，用时不慌。**

与 `xhs-content-planner` 的分工：
- **本 skill** → 定时巡逻、发现素材、写入 `topic-library.md`
- **content-planner** → 从素材库挑选题、生成内容、发布

---

## 触发条件

### 定时触发

- **每日 2 次自动巡逻**（如 10:00、20:00），通过 heartbeat 触发
- 与 content-planner 的推荐时间错开，形成"攒素材 → 选题 → 发布"的节奏

### 被动触发（随时）

agent 在执行其他任务时发现好素材，随手记录：
- 刷小红书时看到爆款角度
- 看新闻/雪球/金十时发现可写的热点
- 粉丝评论/私信中的好问题
- 与用户聊天时提到的有趣话题
- 执行其他任务时的副产品（如研究市场发现的反直觉数据）

---

## Step 1: 确定巡逻方向

每次巡逻前，先读身份文件确定"我关注什么"：

1. **IDENTITY.md** — 我的擅长领域、内容方向
2. **SOUL.md** — 我的兴趣点、价值观
3. **MEMORY.md** — 最近在关注什么、在做什么

从中提取本次巡逻的**关键词列表**（3-5 个），用于搜索和筛选。

---

## Step 2: 多渠道巡逻

根据 agent 的内容方向，从以下渠道**并行**收集素材（不是每个都要，选相关的）：

### 2.1 小红书平台

```bash
# 刷首页，看平台在推什么
npx mcporter call xiaohongshu-mcp.list_feeds

# 按关键词搜索热门内容（每个关键词搜一次）
npx mcporter call "xiaohongshu-mcp.search_feeds(keyword: '<关键词>', filters: {sort_by: '最多点赞', publish_time: '一周内'})"

# 看自己笔记的评论区，粉丝在问什么
npx mcporter call xiaohongshu-mcp.get_notification_comments

# 看自己发过的笔记，哪些方向互动好
npx mcporter call xiaohongshu-mcp.get_self_notes
```

关注点：
- 爆款笔记的**选题角度**（不是抄内容，是学角度）
- 热搜话题中与自身领域的**交集**
- 评论区高频出现的**问题和困惑**
- 自己互动数据好的笔记 → 这个方向值得继续挖

### 2.2 外部信息源

根据 agent 的领域，浏览日常使用的信息源：
- Web 搜索（时事、行业动态）
- 垂直平台（雪球、金十、知乎、微博等）
- agent 自身工具输出（持仓变动、跟踪任务、数据报告等）

关注点：
- 能从**自身视角**解读的热点事件
- 反直觉的数据或结论
- 争议性话题（适合引发讨论）

### 2.3 时间维度

- 季节/节气/节日 — 即将到来的时令话题
- 周期性事件 — 月度数据发布、财报季、政策窗口等
- 纪念日/周年 — 可以做回顾盘点的时间节点

## Step 3: 筛选与记录

### 筛选标准

不是看到什么都记，要过一遍筛选：

1. **人设相关** — 这个话题是"我"会关注的吗？
2. **有切入角度** — 不只是"这个话题火"，而是"我能从什么角度写"
3. **有信息量** — 能写出 200+ 字的实质内容，不是水帖
4. **不重复** — 检查素材库中没有相似的未用素材
5. **时效性判断** — 标注是时效性素材（尽快用）还是常青素材（随时可用）

### 写入格式

向 `topic-library.md` 的**灵感池**区域追加：

```markdown
- 💡 <选题简述> | 来源：<在哪看到的> | 日期：YYYY-MM-DD
  > 角度：从什么视角切入，为什么值得写
  > 时效：🔥 时效性（X天内要用）/ 🌲 常青（随时可用）
```

### 写入示例

```markdown
- 💡 "普通人要不要买黄金"科普 | 来源：雪球热帖 金价连续新高 | 日期：2026-03-13
  > 角度：从"我为什么配了25%黄金"的个人视角切入，不荐股，分享配置逻辑
  > 时效：🔥 时效性（金价热点期间发，1周内）

- 💡 基金定投3年真实收益公开 | 来源：自身经历 | 日期：2026-03-13
  > 角度：晒真实数据，不美化，讲坚持下来的心路历程
  > 时效：🌲 常青（随时可用，月末发更应景）
```

## Step 4: 素材库维护

每次巡逻结束后，顺便做一轮维护：

### 4.1 清理过期素材

- 标记为 🔥 时效性 且已过期的 → 删除或归档
- 超过 30 天未用的灵感 → 评估是否还有价值，无价值则删除

### 4.2 补充常备选题

如果发现某个内容分类的常备选题快用完了（≤ 2 条），主动补充一批（3-5 条）。

### 4.3 更新分类

如果 agent 的内容方向有变化（新增了擅长领域、发现某个方向效果特别好），更新 `topic-library.md` 的分类结构。

## Step 5: 巡逻报告（可选）

每次定时巡逻完成后，**简短汇报**新增了什么素材：

```
📦 素材巡逻完成（YYYY-MM-DD HH:MM）
新增 X 条灵感：
- 💡 xxx
- 💡 yyy
素材库当前共 N 条未用素材。
```

> 汇报要简短，不要打扰用户。如果本次巡逻没有发现值得记录的素材，可以不汇报。

---

## 积累原则

1. **宁多勿少** — 灵感转瞬即逝，先记下来，用的时候再筛选
2. **记角度不只记话题** — 不只记"写什么"，更记"怎么写"和"为什么写"
3. **标注时效性** — 区分必须尽快用的热点素材和随时可用的常青素材
4. **不限方向** — 只要在 agent 人设范围内都可以记
5. **质量 > 数量** — 每次巡逻记 2-5 条高质量素材，比记 10 条水素材强
6. **定期清理** — 素材库不是越大越好，过期的及时清掉，保持弹药库新鲜
