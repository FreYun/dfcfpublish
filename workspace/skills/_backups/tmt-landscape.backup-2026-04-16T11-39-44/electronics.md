## 供需分析框架（聚焦模式执行）

### 需求侧——三个需求引擎

**引擎1：AI算力需求**
- 跟踪变量：云厂商Capex增速、AI服务器出货量、GPU/AI芯片采购量
- 数据源：`research_search("AI服务器 出货量 Capex")` + browser搜索
- 传导链：云厂商Capex↑ → AI服务器需求↑ → GPU/HBM/光模块/PCB/电源 → 拉动设计+封测+材料
- 推导要点：Capex增速 vs 上季对比 → 加速=需求扩张中，减速=可能见顶

**引擎2：消费电子换机周期**
- 跟踪变量：全球智能手机出货量(IDC/Canalys)、PC出货量、折叠屏渗透率
- 数据源：browser搜索 + `news_search("手机出货量")`
- 传导链：手机出货↑ → CIS/SoC/存储/连接器/声学需求↑ → 韦尔/立讯/歌尔
- 推导要点：出货量同比增速方向 + ASP变化 → 量升价升=强复苏，量升价跌=结构降级

**引擎3：国产替代（政策驱动需求位移）**
- 跟踪变量：半导体设备中标金额、信创芯片招标、出口管制升级事件
- 数据源：`news_search("半导体设备 中标 国产")` + `research_search("国产替代 半导体")`
- 传导链：管制升级 → 进口替代需求右移 → 国产设备/材料/EDA订单↑
- 推导要点：这是需求曲线右移（政策外力），不是自然增长 → 关注可持续性和产能承接能力

### 供给侧——产能与库存

- 跟踪变量：晶圆厂产能利用率、存储芯片库存天数、设备交付周期
- 数据源：`research_search("晶圆 产能利用率 库存")` + browser搜索
- 推导逻辑：
  - 利用率>85% + 库存下降 → 供给偏紧 → 支撑涨价 → 景气上行
  - 利用率<75% + 库存累积 → 供过于求 → 价格承压 → 景气下行
  - 新产能投产时间表 → 判断供给拐点

### 价格信号——验证供需判断

| 价格指标 | 数据源 | 含义 |
|---------|--------|------|
| DRAM/NAND合约价 | browser(DRAMeXchange/TrendForce) | 存储周期拐点 |
| 晶圆代工报价 | 研报搜索 | 制造端供需 |
| 光模块ASP（400G→800G） | 研报搜索 | 算力需求验证 |
| MLCC/被动元件价格 | 新闻搜索 | 消费电子需求 |

价格上涨+量增 = 供需判断正确（偏紧）；价格下跌+量增 = 以价换量（过剩）。
