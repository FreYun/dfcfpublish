## 怎么用

- **审稿时**：只读下方速查表，按问题类型找到对应模式 ID
- **需要案例/公式**：按 ID 前缀打开对应文件（T→titles.md, O→openings.md, S→structures.md, E→emotions.md, I→interactions.md, V→visuals.md, F→failures.md）
- **排期时**：扫"适用场景"列，挑匹配的模式推荐给 bot
- **写稿时**：bot 参考模式公式和案例，但不要照搬——模式是骨架，人设是灵魂

## Batch4 核心发现（2026-04-13）

**治愈/焦虑是本批天花板**：治愈关键词 14 篇 avg 69503 赞，焦虑 17 篇 avg 47174 赞。两者共同撑起了 T-11 / T-12 / V-04 三个新模式。

**评赞比天花板刷新**：《比有文化都"内卷"这儿了》评赞比 7.69（100k 评论 / 13k 点赞），是历史最高的 3 倍。建立 I-04 梗图段子接龙模式。

**金句图文崛起**：T-11 "这段话X我"型跨 7+ 篇高赞聚类，动词序列"治愈→点醒→平复→杀死"形成明确梯度。这是焦虑/治愈垂类可复制性最强的新模式。

**幽默反讽作为情绪出口**：E-07 不是纯搞笑，是用幽默制造情绪距离。读者既承认痛又得到笑的出口。代表作《当星露谷农夫开始内卷穿搭》《论极简我只服女儿》。
