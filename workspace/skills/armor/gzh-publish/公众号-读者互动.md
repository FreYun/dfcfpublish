# 公众号读者互动

gzh-op 子技能。管理留言、回复读者、维护粉丝关系。

---

## 公众号互动机制

与小红书不同，公众号的互动主要通过**留言精选**机制：

- 读者留言**默认不公开**，需要作者手动「精选」才能让其他读者看到
- 作者可以回复留言，回复会跟在精选留言下方公开展示
- 留言精选是重要的粉丝运营手段 — 精选+回复让读者感到被重视

---

## 执行流程

### 1. 进入留言管理

通过浏览器进入公众号后台 → 互动管理 → 留言：

```
https://mp.weixin.qq.com/cgi-bin/commentmanage?action=list&token={TOKEN}&lang=zh_CN
```

或从后台左侧菜单点击「互动管理」→「留言」。

### 2. 精选留言

浏览未精选的留言，选择值得公开展示的：

**精选原则**：
- 有实质内容的留言（提问、补充观点、分享经历）→ 精选
- 正面反馈（"写得好""学到了"）→ 适量精选，不全选
- 善意的不同观点 → 精选，展示包容度
- 纯灌水/无意义 → 不精选
- 负面攻击/恶意 → 不精选，不回应
- 引流/广告 → 不精选

**精选数量**：每篇文章精选 3-10 条为宜，太多稀释质量。

### 3. 回复留言（精选后）

精选的留言可以回复，回复会公开展示：

**回复原则**：
- **用自己的人设说话**，不模板化
- **有信息量**：补充数据、回答问题、分享想法，30-100 字为宜
- 提问类留言 → 认真回答
- 分享类留言 → 肯定+补充
- 不同观点 → 理性讨论，"你说的有道理，我的看法是..."
- 不引流、不荐股、不预测涨跌
- 不暴露研究部
- 涉及投资观点加"个人看法仅供参考"

### 4. 私信管理（可选）

公众号后台可以查看读者私信。处理原则：
- 正常问题 → 简短回复
- 引流/广告 → 忽略
- 敏感问题（要求推荐个股等）→ 婉拒，"不方便给具体建议"

---

## 互动节奏

| 时间 | 任务 |
|------|------|
| 文章发布后 2-4 小时 | 第一轮精选+回复（趁热度） |
| 每日 1 次 | 检查新留言，精选+回复 |
| 周末 | 可以放松，不强求 |

---

## 互动铁律

1. **人设一致** — 回复风格与文章风格一致
2. **不对线** — 遇到杠精不争论，精选时跳过即可
3. **不引流** — 不引导加微信/关注其他平台
4. **不荐股** — 评论中也不推荐具体标的
5. **不暴露** — 不提研究部、不暴露多人运营
6. **及时性** — 留言超过 48 小时不回复，读者已经不期待了
