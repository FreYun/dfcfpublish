# 公众号合规速查

> **每次写稿前必读。** 本文件整合了《基金宣传推介材料合规指引》的专业合规要求。
> 文章被打回时，按违规类型找到对应条目修改。
>
> 禁用词完整清单和绝对红线见「公众号-发文前必读.md」。

---

## 金融类违规修改方向

| 违规表现 | 修改方向 |
|---------|---------|
| 绝对化收益（"必涨""稳赚""XX%收益"） | 改为"可能""我的判断是""历史上曾经" |
| 明确投资建议（"现在买入""建议配置""我推荐"） | 改为"我在关注""如果我来做选择""仅供参考" |
| 缺少风险提示 | 文末加风险提示（见下方模板） |
| 炫富（晒具体收益金额） | 删除具体数字，改为感性描述 |
| 提及个股代码/名称 | 替换为行业描述，如"某新能源龙头" |

---

## 业绩展示规则

| 规则 | 要求 |
|------|------|
| 合同生效不足6个月 | **禁止**展示业绩 |
| 展示业绩区间 | 必须超过6个月（不含），且必须同时展示业绩比较基准 |
| 合同生效6个月~1年 | 从合同生效日起算 |
| 合同生效1年~5年 | 包含自生效当年起所有完整会计年度业绩 |
| 合同生效5年以上 | 包含最近5个完整会计年度业绩 |
| 展示超6个月/近1年区间业绩 | **必须同时展示成立以来业绩** |
| 基金经理业绩 | 须覆盖该经理管理的**全部同类产品**业绩，不得片面选取 |
| 基金经理变更 | 展示区间内若有变更，必须特别说明 |
| 模拟业绩 | 禁止模拟未来业绩；定投可用代表性指数模拟≥3年历史，须注明来源、方法、公式 |
| 业绩比较 | 须使用可比的数据来源、统计方法和比较期间 |
| 必加声明 | "基金的过往业绩并不预示其未来表现""基金管理人管理的其他基金的业绩并不构成基金业绩表现的保证" |

---

## 数据引用规则

| 规则 | 要求 |
|------|------|
| 基金业绩数据来源 | 须经托管人复核或摘取自基金定期报告，不能单独用 choice/wind |
| 评价机构引用 | 只能引用：银河证券、海通证券、上海证券、招商证券、晨星中国、济安金信、天相投顾 |
| 评价结果时效 | 不得仅引用距今超过2年的评价结果 |
| 奖项引用 | 只能用：金牛奖、金基金奖、明星基金奖、晨星基金奖、济安金信群星汇奖；须注明获奖时间，不得仅引用2年前奖项 |
| 统计图表 | 须清晰准确，注明数据出处 |
| 研报/观点引用 | 禁止用"建议""推荐"，只能说"可关注"；须注明出处 |

## 风险提示模板

**所有文章必加（基础）：**
> 市场有风险，投资需谨慎。

**涉及市场观点/行业分析时加：**
> 以上内容仅为个人观点，不构成投资建议或行情预测。

**涉及过往业绩时加：**
> 基金的过往业绩并不预示其未来表现。基金管理人管理的其他基金的业绩并不构成基金业绩表现的保证。

**涉及基金经理代表作时加：**
> 过往业绩不预示未来表现；基金经理在管的其他产品业绩不代表本产品业绩。

**涉及定投回测时加：**
> 仅根据页面所示模拟方法进行回测，不代表历史实际收益水平，不构成未来业绩预测。

**涉及 AI 生成内容时加：**
> 本内容部分或全部由人工智能（AI）生成，其准确性、完整性和时效性可能存在不足，不代表任何投资建议或行情预测。

**涉及个股时加：**
> 以上内容仅供参考，文中涉及个股的，不构成股票推荐和投资建议，股票市场波动大，购买前请审慎操作。

## 视频类特别规定

| 违规表现 | 修改方向 |
|---------|---------|
| 视频中说"我推荐""我建议""权威" | 删除，改为中性表述 |
| 宣传单日涨跌幅或净值估算 | 删除 |
| 提供具体操作建议（买卖时机、数量） | 删除，改为信息分享 |
| 自行对基金排行/创造榜单 | 删除，如引用专业机构观点须注明出处 |
| 虚构场景暗示获益 | 删除或加提示"故事场景、人物关系为虚拟拍摄" |
| 风险提示不足5秒 | 影像形式须≥5秒显示风险提示 |

## 版权类（公众号特有）

| 违规表现 | 修改方向 |
|---------|---------|
| 使用他人图片未授权 | 替换为自有图片或可商用素材 |
| 大段引用他人文章 | 缩减引用量，标注出处，或使用转载功能 |
| 声明原创但含非原创内容 | 取消原创声明，或删除非原创部分 |
| 使用来源不明的网络图片 | 任何图片即使无水印也有版权，须用已授权素材 |
| 转载文章未确认图片版权 | 文字转载授权不含图片，须单独获得图片授权 |
| 使用他人肖像未获同意 | 删除或获得肖像权人同意 |
