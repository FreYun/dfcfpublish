# 🎨 历史风格参考

> **触发场景**：polish 环节——主播说"钩子太弱"、"金句改一下"、"整体更亲和一点"、"这段语气不对"。
>
> **来源**：从 12 篇主播本人历史脚本抽取的片段库（`workspace-mag1/inbox/口播脚本整合-2026-04-17/口播脚本整合/`）。直接当素材调用；不要照搬原句，**改写或重组成当前选题里的说法**。

## 用法

1. 先判断主播要改什么（钩子 / 金句 / CTA / 语气）
2. 从对应库里挑 2-3 个**同构**片段
3. 把当前脚本的数据 / 名词填进句式，产出 2-3 个替换选项
4. 返回选项，让主播挑

**绝对禁止**：原样复制片段当脚本用。片段是骨架不是词稿。

## 1. 钩子库

### 1.1 数据反差型（行情解读最常用）

```
句式：{大数字A}，这是{时段/背景}。这一轮{事件}，{相同时段}，**"只有"{小数字B}**。

样本：
- "10年24倍，这是1969年到1980年黄金跟美元脱钩之后的涨幅。这一轮黄金牛市，
  从2015年底算起，同样10年，**'只有'4.5倍。**"
```

### 1.2 反常识判断型

```
句式：==从{视角/数据}来看，{主流想法}的反面其实才是真的。==

样本：
- "==从历史数据来看 a 股的确是越来越抗揍了。=="
- "2018 到 2019 年，海外一有风吹草动，A 股平均跌 **7.8%**；
   2020 年之后，同样级别的冲击，==平均跌幅只有 3.74%==。"
```

### 1.3 提问悬念型

```
句式：{事件 X} 发生了。这是危还是机？今天就跟宝子们聊聊。

样本（禁用 →）：
- ❌ "大家好，今天聊聊 X。"
- ❌ "今天我们来讲一讲 Y。"

合格改写 ✅：
- "金价从 4800 一路跌到 4200，创下 4 月新低。这波到底是见顶了，还是被错杀？"
```

### 1.4 经历代入型（必修课常用）

```
句式：{最近踩坑/操作} 让我意识到 —— 原来 {概念}，比我以为的要 {形容词}。

样本：
- "这是第二期基金入门知识的查漏补缺，基础不牢固的宝子们一定要看完哦。"
- "我自己就栽过这个跟头——浮亏的时候死拿，回本就跑，结果越拿越亏。"
```

## 2. 金句 / 立场句库

### 2.1 历史参照型

```
- "==这波涨幅，主要来自美元信用崩塌，而不是通胀。=="
- "==2020 年之后，A 股的韧性跟以前不一样了。=="
```

### 2.2 风险提示型（温和但明确）

```
- "我不是说不能上车，只是==上车前要想清楚自己的仓位==。"
- "宝子们，**高位追涨的代价，历史上从来没便宜过**。"
- "当大家都在讨论'会不会错过'的时候，往往==已经不是最好的位置==了。"
```

### 2.3 价值观落脚型

```
- "==宁可少赚，不要大亏==。"
- "拿得住比择时准重要。"
- "投资是长期的事情，**不是赌一把就能翻身**的游戏。"
- "多元配置，穿越周期。"
```

### 2.4 真人自揭型（拉近距离）

```
- "我自己的持仓现在也是浮亏的，没什么不能说的。"
- "这波我保守了，没加仓——但也没追。"
- "我没踩对节奏，但至少==没让自己出局==。"
```

## 3. CTA 库（温和，禁推销）

> **系列差异极其重要**：
> - **行情解读（日更 12 篇样本）里 0 次出现**"记得关注哦，我们明天见"这种预告型 CTA。行情解读的收束都是自然落在最后一条建议 / 金句 / 感慨上
> - **必修课 / 专题**（基金入门脚本、十五五规划专题、康波周期）才会用"记得关注哦，下期见 / 明天见"——因为它们有续集感
>
> 选 CTA 之前先看系列归属，别跨系列乱抄。

### 3.1 建议型（两系列通用）

```
- "宝子们注意控制仓位。"
- "最近这种位置，**我个人会更偏向分批**。"
- "强烈建议宝子们==先搞清楚自己的风险承受能力再动手==。"
- "风险偏好较低的宝子要注意控制仓位。"（0325）
- "不建议现在轻易割肉，可以再等一等。"（0306）
```

### 3.2 行情解读的自然收束（无 CTA 或 弱 CTA）

```
样本原句（12 篇里随手摘）：

- 0316 收尾：
  "在市场共振下跌的时候，任谁也很难做到独善其身、逆势上涨，
   但是我们可以趁此机会，检查自己的资产配置是否合理，是否能做到穿越周期。"

- 0325 收尾：
  "中长期在央行购金跟降息预期下，上行趋势不变。"（金句收束）

- 0303 收尾：
  "对于权益市场，长期还是偏乐观，但短期...所以不建议盲目抄底。"

- 0213 收尾（节前）：
  "没想到节前还要给宝子们按摩一下，星星也提前拜个早年，祝宝子们新年快乐，财源滚滚！"

写法特征：
- 收在最后一条建议 / 立场判断 / 金句上，不再另起一行说"记得关注"
- 节日 / 重大时点才用感慨式 / 拜年式收尾
- ==绝不==写"记得关注哦，我们明天见"之类的固定 CTA
```

### 3.3 必修课 / 专题的预告型 CTA（限这两个系列）

```
样本原句：

- 脚本.md（基金入门 2 期）：
  "下一篇我们详细来讲讲基金的分类。记得关注哦。"

- 脚本.md（基金入门后期）：
  "下期视频，会跟大家分享一些挑选基金的实用方法，记得关注哦~"

- 十五五规划专题：
  "那后面规划正式发布后，我也会为大家带来更深入的解读。
   记得关注哦，我们明天见。"

写法特征：
- 必须带**具体预告**（"下一篇讲 X / 下期分享 Y / 后面规划发布后再解读"）
- 不是空洞的"记得关注哦"
- 预告的内容必须真的是下一期会做的
```

### 3.4 禁区（两系列都不许出现）

```
- ❌ "点赞加关注，不迷路！"
- ❌ "现在买入稳赚不赔！"
- ❌ "3200 点抄底就完事！"
- ❌ "关注我，带你翻倍！"
- ❌ 任何具体买卖点 / 具体股票代码推荐
- ❌ 行情解读结尾写"记得关注哦，我们明天见"（历史 0 次，纯 AI 味）
```

## 4. 过渡 / 节奏句库

### 4.1 条分缕析（三板斧）

```
- "这个事情要从两个维度看——外部是 X，内部是 Y。"
- "1 是 {…}；2 是 {…}；3 是 {…}。"
- "先说结论：{…}。为什么这么说，我们拆三层。"
```

### 4.2 术语白话化（每引术语必配）

```
- "专业点说叫'{术语}'，==说人话就是 {白话}==。"
- "这个词听着玄乎，其实就是 {一句话翻译}。"
- "机构爱用 {术语}，**对普通人来说，意思是 {操作含义}**。"
```

### 4.3 情绪节拍（【音效】位）

```
- "====【空旷音效】===="   ← 重大转折处
- "【停顿】"                ← 钩子到开篇之间
- "【强调】"                ← 金句前
```

## 6. polish 使用示例

**场景**：主播说"这个钩子太弱"

脚本原钩子：
> "今天跟大家聊聊黄金最近的表现。"

你的响应：
```
读了历史风格参考.md 的「钩子库 1.1 数据反差型」和「1.2 反常识判断型」。
给主播三个替换选项：

选项 A（数据反差）：
"10 年 4.5 倍，这是 2015 年到现在黄金的涨幅。
 听着很猛？但 1969 到 1980 年同样 10 年，黄金涨了 24 倍。"

选项 B（反常识）：
"==金价 4 月刚创新低，但真正值得关注的不是跌了多少，
  而是这波下跌的原因，跟 2020 年那次完全不是一回事。=="

选项 C（提问悬念）：
"金价从 4800 一路跌到 4200，创下 4 月新低——
 这波到底是见顶了，还是被错杀？今天就跟宝子们聊聊。"

主播挑一个，我按这个钩子往下改。
```

> **注意**：三个选项都**直接进话题**，没有"宝子们，我是星星。今天跟你们拆解一下"这种开场白。这种自我介绍历史 12 篇里 0 次出现，polish 必须剔除。

## 7. 铁律

- **片段是骨架，不是词稿**：每次 polish 必须把数据 / 名词替换成当前选题的内容
- **一次给 2-3 个选项**：主播挑，不要替主播定
- **不跨系列串用**：必修课的钩子不要用在行情解读里，反之亦然；"记得关注哦我们明天见"这种 CTA 只在必修课 / 专题里用，**行情解读绝不用**
- **失去手感就复读样本**：如果 polish 出来的句子自己都觉得"不像那个人"，去读 2 篇原样本再来
- **AI 味自检清单**（polish 时内心过一遍）：
  - [ ] 有没有写 `【钩子】/【开篇】/【主体】` 这种台词标签？历史 0 次，有就删掉
  - [ ] 开头有没有"宝子们，我是星星，今天跟你们拆解一下"？历史 0 次，改成直接进话题
  - [ ] 主体有没有 `**先说数据。** **为什么涨这么猛？**` 这种 PPT 式加粗小标题？历史基本没有，改成连续叙事
  - [ ] 结尾是不是套路化"记得关注哦，我们明天见"？行情解读里不能有
  - [ ] 金句有没有连续堆 2-3 句？历史是散落着写，不是尾部堆砌
