# 公众号浏览器发布

通过 Playwright CDP 连接浏览器，操作微信公众号后台（mp.weixin.qq.com）完成文章发布。

**重要：默认「保存为草稿」，不直接发表。除非研究部明确要求。**

---

## 前置条件

1. CDP 端口上的浏览器已启动（端口号见 bot 自己的 TOOLS.md）
2. 已登录微信公众号后台，或准备好扫码登录
3. Python 3 + Playwright 已安装（Python 路径见 TOOLS.md）

## 连接方式

```python
from playwright.sync_api import sync_playwright

# CDP_URL 和 PYTHON 路径从 TOOLS.md 获取，不要硬编码
CDP_URL = "http://localhost:{CDP_PORT}"

with sync_playwright() as p:
    browser = p.chromium.connect_over_cdp(CDP_URL)
    context = browser.contexts[0]
    page = context.pages[0]  # 或 context.pages[-1] 取最新页
    # ... 操作 ...
    browser.close()  # 断开连接，不关闭浏览器
```

---

## Step 0: 登录

### 0.1 打开公众号后台

导航到 `https://mp.weixin.qq.com/`。

### 0.2 检查登录状态

截图检查页面：
- 看到后台首页（左侧导航栏、账号名称、数据概览）→ 已登录，跳到 Step 1
- 看到登录页面（二维码）→ 需要登录

### 0.3 登录流程

1. 页面上显示微信扫码二维码
2. 提示用户："请用微信扫码登录公众号后台，登录完成后告诉我。"
3. 用户确认后，截图验证登录状态
4. 登录后 URL 变为 `https://mp.weixin.qq.com/cgi-bin/home?t=home/index&lang=zh_CN&token=XXXXX`
5. **记住 token 值**，后续操作 URL 都需要这个 token

---

## Step 1: 进入文章编辑器

**方式一（推荐）：** 直接通过 URL 打开：

```
https://mp.weixin.qq.com/cgi-bin/appmsg?t=media/appmsg_edit&action=edit&type=77&lang=zh_CN&token={TOKEN}
```

**方式二：** 草稿箱点击「新的创作」→「写新文章」。注意会打开新标签页，用 `context.pages[-1]` 获取。

### 编辑器布局

| 区域 | 内容 |
|------|------|
| **顶部工具栏** | 图片、视频、音频、超链接、格式（字号/加粗/斜体/对齐/列表等） |
| **左侧预览区** | 文章卡片预览（封面图 + 标题）、「+ 新建内容」、历史版本 |
| **右侧编辑区** | 标题输入（0/64）、作者输入、正文编辑区（ProseMirror） |
| **底部操作栏** | 字数统计、「保存为草稿」（绿色）、「预览」、「发表」 |

---

## Step 2: 填写内容

### 标题
```python
page.fill("textarea#title", "文章标题")  # 最多 64 字
```

### 作者
```python
page.fill("input.js_author", "作者名")  # 可选，最多 8 字
```

### 正文
```python
page.click(".ProseMirror")
page.keyboard.type("正文内容...", delay=20)
```

**富文本格式操作：**
- 换行：`page.keyboard.press("Enter")`
- 加粗：选中文字后 `page.keyboard.press("Control+b")`
- 插入图片：点击 `#js_editor_insertimage`，通过 `input[type="file"]` 上传

### 插入图片到正文

1. 点击顶部工具栏图片按钮 `#js_editor_insertimage`
2. 弹出上传菜单，通过 `input[type="file"]` 上传本地图片
3. 等待上传完成

---

## Step 3: 一键排版（推荐）

编辑器右下角浮动按钮，正文填完后、保存前使用。

### 操作流程

```python
# 1. 点击「一键排版」— 会打开新标签页
with context.expect_page() as new_page_info:
    page.locator('.edui-for-ailayout .ailayout-btn_wrp').click()
layout_page = new_page_info.value
layout_page.wait_for_load_state("domcontentloaded")

# 2. 选择排版模板（可选）
layout_page.locator('.layoutstyle_item_thin').click()   # 精简
# 或 layout_page.locator('.layoutstyle_item_standard').click()  # 标准（默认）
import time; time.sleep(1)

# 3. 选择主题色（可选，索引从 0 开始）
layout_page.locator('.themecolor-item_wrp').nth(3).click()  # 深绿
time.sleep(1)

# 4. 确认使用
layout_page.locator('.weui-desktop-btn_primary').click()

# 5. 回到编辑器
time.sleep(2)
page = context.pages[0]
```

### 排版模板

| 模板 | 选择器 | 说明 |
|------|--------|------|
| 标准（默认） | `.layoutstyle_item_standard` | 含装饰元素 |
| 精简 | `.layoutstyle_item_thin` | 干净简洁 |

### 主题色（12 种）

深蓝、浅蓝、紫色、深绿、浅绿、深黄、橙色、红色、粉色、深灰、浅灰、渐变。也可选「全文无彩色字体」(`.themecolor-desc`)。

---

## Step 4: 正文下方设置

滚动到正文下方：

| 设置项 | 说明 | 操作 |
|--------|------|------|
| **原创** | 声明原创获平台保护和推荐 | 点击「未声明 >」→ 弹窗中选择「文字原创」→ 勾选协议 → 确定 |
| **赞赏** | 需先声明原创 | 声明原创后可配置 |
| **留言** | 由作者精选公开 | 默认开启 |
| **合集** | 添加到合集 | 点击「未添加 >」选择 |
| **原文链接** | 阅读原文链接 | 点击「未添加 >」输入 URL |
| **创作来源** | AI 辅助声明等 | 点击「未添加 >」选择来源类型 |
| **平台推荐** | 是否允许推荐 | 默认开启 |

---

## Step 5: 封面图

左侧预览区的灰色区域（`.card_appmsg_thumb`）：
- 点击可上传/更换封面图
- 建议 900×383 像素（2.35:1），JPG/PNG
- 不设封面，发表时可能提示需要添加

---

## Step 6: 保存 / 发表

### 保存为草稿（默认操作）

```python
page.click("#js_continue_save")
```

截图确认保存成功。

### 发表（需研究部明确同意）

1. 点击底部「发表」(`button.mass_send`)
2. 弹出创作来源声明弹窗
3. AI 辅助内容 → 选「去声明」标注来源
4. 非 AI 内容 → 选「无需声明并发表」

---

## Step 7: 多图文（可选）

1. 左侧预览区点击「+ 新建内容」
2. 添加第二篇文章卡片
3. 每篇独立编辑标题、作者、正文
4. 第一篇为头条（大封面），后续为次条（小封面）

## 故障排除

| 问题 | 处理 |
|------|------|
| CDP 连接失败 | 确认浏览器已启动，端口号正确 |
| 跳转到登录页 | 回到 Step 0 重新扫码 |
| token 失效 | 从当前 URL 提取新 token，或重新登录 |
| 中文选择器失败 | 改用 CSS 选择器（id、class） |
| 图片上传失败 | 检查格式和大小，用 `set_input_files()` |
| 编辑器没输入 | 先 `click()` 聚焦再 `keyboard.type()` |
| 弹窗无法关闭 | 按 `Escape` 或点击弹窗外区域 |
| 找不到元素 | 截图观察，可能需要等待加载或滚动 |
