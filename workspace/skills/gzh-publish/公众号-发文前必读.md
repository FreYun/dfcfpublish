# 公众号写稿必读

> **每次写稿前必读。** 微信对公众号内容审核严格，违规可能导致删文、封号、限流。

---

## 绝对红线

- **不引流** — 不留个人微信号/手机号/二维码/外链（阅读原文链接除外）
- **不提竞品平台** — 不提抖音、小红书、B站、知乎等竞品名称；引用观点用"看到有人分析"
- **不提具体博主** — 不出现任何博主名字/昵称/ID
- **不提具体个股** — 不出现个股名称或代码；只聊板块/行业/大方向
- **不荐股荐基** — 无资质不得推荐具体标的
- **不承诺收益** — 不说"稳赚""保证""100%有效"
- **不虚假人设** — 不虚构业绩/学历/收入
- **不搬运** — 内容必须原创，转载需走转载功能
- **不暴露研究部** — 永远以第一人称写，不提幕后团队
- **不诋毁同行** — 不贬低其他机构或基金，不唱衰、调侃金融市场
- **不使用负面新闻/政治敏感事件** — 不引用未经权威部门核实的海外媒体内容

## 禁用词完整清单

### 通用禁用词

**极限词**：最、最佳、最优、第一、唯一、顶级、极致、完美、无敌、超级、超强

**承诺词**：保证、100%有效、速效、无效退款、永久

**营销词**：返利、私信领取、点击领奖、卖疯了、再不抢就没了、限时折扣

### 金融禁用词（来自《基金宣传推介材料合规指引》）

**暗示保本/承诺收益：**
不惧震荡、高收益、高回报、赚钱、年年连续正收益、压舱石、行稳致远、求稳求进、稳中求进、抗通胀、坐享财富增长、收益升级、躺赢、躺赚、月月正收益、季季正收益、年年正收益

**暗示无风险/忽视风险：**
安全、省心、安心、放心、追求稳健、稳定、安稳、保证、承诺、保险、有保障、避险、无风险、规避风险、规避波动、抵御波动、能涨抗跌、平滑波动、尽享牛市

**无客观证据不得使用（加"历史"前缀可能可用）：**
低回撤、低波动、抗震、减震、抗跌、业绩稳健、业绩优良、名列前茅、位居前列、业绩亮眼、低估、历史低位、价值洼地、市场见底、性价比

**夸大形容：**
牛基、牛股、大牛、黑马基金、长期景气、长牛赛道、爆款基金

**择时/催促购买：**
即将结募、欲购从速、好时机、申购良机、黄金布局时机、配置正当时

**基金经理明星化：**
大V、网红、顶流、明星、男神、女神、当红炸子鸡、粉丝

**暗示政策背书：**
政策力挺、政策加持（"政策利好"可用于宣传行业，但不能用于宣传基金）

**其他：**
- 不得以公司名义发布证券研究报告
- 不得使用净值归一等误导投资人的表述
- 不得使用单位、个人的推荐性文字
- 收益率、投资回报、保底收益、稳赚不赔、返本、众筹

## 替代写法

| 想说的 | 改成 |
|--------|------|
| 推荐买入 | 个人觉得可以关注 |
| 收益率 XX% | 历史表现大概在这个区间（过往不代表未来） |
| 涨疯了/暴涨 | 涨得不少 / 走得挺强 |
| 抄底/梭哈 | 可少量用于个人记录，不放标题 |
| 建议/推荐（引用研报） | 可关注；须注明出处 |

## 公众号特有规则

1. **AI 创作声明** — AI 辅助生成的内容，发表时必须选择「去声明」标注来源，否则可能被平台处罚
2. **时事/公共政策** — 涉及国内外时事、公共政策、社会事件的内容，发表时需声明创作来源
3. **原创声明** — 确实原创的内容建议声明原创，获平台著作权保护和推荐加权
4. **封面图必须有** — 不设封面图发表时会提示，建议每篇都配封面
5. **标题不超 64 字** — 但建议控制在 30 字内，过长在手机端会被截断
6. **留言需精选** — 公众号留言默认不公开，需要作者手动精选才能让其他读者看到

---

## 金融内容附加规则

1. 定位"分享者"不是"导师" — 只分享看法和学习过程
2. 只谈逻辑，不承诺结果 — "我个人比较看好" 而非 "买这个稳赚"
3. 涉及产品/操作/收益 → 文末加风险提示
4. 穿插生活内容，降低被判金融营销号风险
