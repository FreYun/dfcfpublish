## 硅料供需框架

### 需求侧
- **核心驱动**：下游硅片产量 = f(电池/组件排产) = f(终端装机需求)
- 跟踪变量：硅片企业开工率、月度硅片产量（中国光伏行业协会）
- 数据源：`research_search("硅片 开工率 产量")` + `news_search("硅片 排产")`
- 推导：硅片开工率↑ → 硅料需求↑；开工率↓ → 硅料需求↓
- 季节性：Q1淡季（春节+装机低谷）、Q3-Q4旺季（装机冲量）

### 供给侧
- **核心变量**：在产产能(万吨) + 在建/规划产能 + 产能利用率
- 数据源：`research_search("多晶硅 产能 产量 投产")` + `news_search("硅料 停产 检修")`
- 推导：新产能投产+需求平稳→过剩→价格下行；二三线停产+需求恢复→收缩→企稳
- **出清信号**：成本高于售价的产能退出（现金成本曲线：头部<40元/kg，尾部>60元/kg）

### 价格信号
| 产品 | 数据源 | 频率 |
|------|--------|------|
| 致密料/菜花料(元/kg) | 硅业分会周报 / InfoLink | 周 |
| 颗粒硅(元/kg) | 同上 | 周 |

### 推导模板
```
需求：硅片开工率 [X%]，趋势 [↑↓→] → 硅料需求 [增/减/平]
供给：在产产能 [X万吨]，利用率 [X%]，近期投/停产 [事件]

```

## 硅片供需框架

### 需求侧
- **核心驱动**：电池片排产量
- 尺寸结构：182mm vs 210mm 占比变化 → 影响产线兼容性和开工率

### 供给侧
- 产能严重过剩 → 竞争格局比总量更重要
- 双寡头（隆基+中环）份额变化 → 份额稳=格局好，被蚕食=价格战
- N型占比提升是结构性趋势

## 光伏设备商供需框架

### 需求侧（设备需求 = 下游扩产Capex）
- **核心驱动**：硅料/硅片/电池/组件厂的资本开支
- 跟踪：光伏企业Capex同比、新产线招标、设备中标金额
- 数据源：`research_search("光伏设备 订单 扩产")` + `news_search("晶盛 捷佳 迈为 奥特维 订单")`
- 推导：下游扩产→设备订单高峰（领先产能投产12-18月）；下游亏损→扩产暂停→订单断崖
- **特殊驱动**：技术迭代换线需求（PERC→TOPCon/HJT/BC）→ 即使总产能不增，换线也产生设备需求
- **新增驱动**：太空光伏/特斯拉订单 → 海外增量，独立于国内周期

### 供给侧
- 设备技术壁垒远高于制造环节，格局好
- 技术路线押注决定命运：
  - **晶盛**（拉晶）→ 路线中性，TOPCon/HJT/BC都需要单晶炉
  - **捷佳/拉普拉斯**（TOPCon整线）→ 绑定TOPCon路线
  - **迈为**（HJT整线）→ 高弹性，HJT放量则暴涨，单GW价值量5亿远超TOPCon的1.5亿
  - **帝尔**（激光）→ BC扩产受益，路线差异化
  - **奥特维**（串焊机）→ 路线中性，组件环节通用

### 价格信号
| 指标 | 含义 |
|------|------|
| 在手订单(亿元) | 未来12-18月收入能见度 |
| 订单同比增速 | 扩产周期方向 |
| 设备毛利率 | 竞争格局+议价能力 |
