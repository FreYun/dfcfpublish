# 叙事风格库 /writing-styles

通用写作风格组件，供其他 skill 引用。

**使用方式**：在你的 skill 中写 `Read skills/writing-styles/xxx.md` 即可加载对应风格。

| 风格 | 文件 | 核心手法 | 适用场景 |
|------|------|---------|---------|
| 传导链 | `style-chain.md` | A→B→C→D 多米诺推演 | 产业链价格传导、宏观→行业 |
| 先破后立 | `style-debunk.md` | 数据推翻共识→给出真相 | 市场存在普遍误解 |

| 灵魂三问 | `style-three-questions.md` | 围绕3个核心问题展开 | 投资决策型文章 |

**决策树：**
```
行业正在经历什么？
├── 价格传导明显 → 传导链
├── 市场有误解 → 先破后立
├── 新旧交替 → 双轨叙事
├── 龙头有故事 → 庖丁解牛
├── 读者问"能不能买" → 灵魂三问
└── 不确定 → 默认传导链
```

## 通用写作技巧（所有风格共用）

不管选哪种风格，以下技巧必须贯穿全文。

### 表格→叙事转化

| 场景 | 研报写法 | 文章写法 |
|------|---------|---------|
| 价格变动 | `硅料 40元/kg -50% 底部震荡` | "硅料从300块跌到40块，跌了87%——比2015年钢铁去产能还狠" |
| 多指标对比 | 5列×8行大表格 | 只挑最有故事性的2-3个数据，用对比句串联 |
| 公司排序 | `第一梯队/第二梯队` 表格 | "如果只能选一家，选XX——因为..." |

全文表格最多1个（横向对比 ≥ 4家时可用）。

### 引用自然化（保留出处）

| 研报式 | 文章式 |
|-------|--------|
| 据浙商机械（2026-03-30）研报指出 | 浙商机械最新研报提到 |
| 东吴计算机（2026-03-22）研报认为 | 根据东吴计算机的跟踪 |
| 据天风电新（2026-03-30）研报测算 | 天风电新给了一笔账 |

- 保留券商名称，不写具体日期
- 引用后紧跟自己的解读
- 完整出处（券商、标题、日期）交付时以独立清单附给研究部

### 开头 Hook 公式

| 类型 | 模板 | 示例 |
|------|------|------|
| 反直觉数据 | "XX从A到B，[类比]" | "硅料从300跌到40——比房价崩盘还狠" |
| 人物驱动 | "[名人]又搞事了——这次是[行业]" | "马斯克又搞事了——这次是光伏板" |
| 灵魂拷问 | "如果只能选一个方向，你选什么？" | "光伏里只能下一个注，你选哪个环节？" |
| 反差场景 | "A在[惨状]，B在[盛况]" | "硅料厂在裁员，设备商在加班" |
| 时间锚定 | "[时间]前还是[旧]。现在，[新]" | "两年前光伏是最爱赛道，现在基金经理摇头" |
