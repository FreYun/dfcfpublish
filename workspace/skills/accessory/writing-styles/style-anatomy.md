# 庖丁解牛

**对标**：思想钢印
**核心手法**：选一家公司/一个产品作为切入点，解剖整个行业
**适用**：某龙头公司特别有代表性，某个产品/技术路线值得深入分析

## 什么时候用

- 一家公司的经历能代表整个行业
- 一个产品/技术路线的兴衰能映射行业趋势
- 读者对抽象行业分析没感觉，需要具体案例拉近距离

## 结构模板

```
Hook：这家公司的一个反常/有趣的数据点（100-200字）

公司/产品画像（500-800字）
  — 它是谁，做什么，在行业里什么位置
  — 用一个类比帮读者理解

解剖刀：从这家公司看到的行业真相（1200-1800字）
  — 维度一：它的处境反映了行业什么问题？
  — 维度二：它的选择反映了行业什么趋势？
  — 维度三：它的财务数据验证了什么逻辑？
  — 每个维度从公司出发，延伸到行业

一叶知秋：从一家公司推导整个行业（500-800字）
  — 如果龙头都这样了，二三线呢？
  — 行业拐点/趋势/机会在哪？

收束 + 风险提示
```

## 关键技巧

- 选对公司很关键——要有代表性，读者最好听过名字
- 不要写成公司分析报告——公司是工具，目的是理解行业
- 从公司到行业的延伸要自然，不能硬拗
