# 先破后立

**对标**：微积分量化价投
**核心手法**：先用数据推翻一个主流认知，再给出替代方案
**适用**：市场对某行业有普遍误解，某个"常识"其实不成立

## 什么时候用

- 主流观点和数据有矛盾（如"都说光伏要完，但有公司偷偷在赚钱"）
- 一个看似合理的投资逻辑存在致命漏洞
- 想给读者一个"原来如此"的认知刷新

## 结构模板

```
Hook：抛出一个"大家都这么认为"的共识（100字）

破：用数据推翻共识（800-1200字）
  — 共识是什么 + 为什么大家这么想
  — 但数据告诉我们...（甩出反驳数据）
  — 一步步展示共识哪里错了
  — 关键转折句："但如果你打开数据看..."

立：给出正确的认知框架（800-1200字）
  — 实际情况是什么
  — 为什么真相和直觉不同
  — 正确的分析框架应该是...

So what：基于正确认知的投资含义（300-500字）
  — 如果你还在用旧认知做决策，会...
  — 正确的关注点是...

收束 + 风险提示
```

## 关键技巧

- "破"要足够有力——"数据证明这是错的"，不是"有些人可能不太对"
- "立"要有建设性——不能只破不立，读者要带走新认知
- 共识来源要真实——不能虚构稻草人来打

## Before/After 示范

**Before（研报式）：**
> 太空光伏是2026年最强产业催化，正处于从0到1的产业爆发期。设备产业链成为攻守兼备的最优选择。

**After（先破后立式）：**
> "光伏完了"——这大概是过去一年基金经理说得最多的一句话。
>
> 硅料跌了87%，硅片腰斩，基金持仓里光伏从15%降到不足3%。好像全行业在等死。
>
> 但如果你打开设备商的财报，会看到一个完全不同的世界：某HJT设备龙头在手订单够做两年，某串焊机企业新签订单同比翻倍。
>
> 光伏不是"完了"，而是"分化了"。传统制造卖的是"产品"，价格随行就市；设备商卖的是"能力"，有壁垒+认证+订单锁定。一个在红海打价格战，一个在蓝海接订单。
