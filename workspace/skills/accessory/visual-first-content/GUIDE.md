# 图文分离型博主 - 正文写作指南

## 核心原则

**图片 = 内容主体**（信息密度高）
**正文 = 简短引导**（几句话即可）

---

## 正文结构公式

### 3句话结构（推荐）

```
第1句：【免责声明】（可选，视账号类型）
第2句：【一句话引导】引导用户看图/收藏
第3句：【互动/标签】引导互动或标签
```

### 4句话结构（稍详细）

```
第1句：【免责声明】个人学习复盘|非投资推荐
第2句：【价值点】一句话说清这篇的价值
第3句：【引导语】建议收藏/一张图看懂/详细看上图
第4句：【互动引导】有问题评论区/你更看好哪个
```

## 不同类型账号的正文模板

### 1. 产业链分析类（青柠檬丸子风格）

**字数**：30-80字
**结构**：

```
【免责声明】个人学习复盘|非投资推荐

一张图看懂XXX产业链，建议收藏👆

#产业链 #行业研究 #投资学习
```

**示例**：
> 【免责声明】个人学习复盘|非投资推荐
> 
> 一张图看懂人形机器人产业链，建议收藏👆
> 
> #产业链 #人形机器人 #行业研究

---

### 2. 财经知识类（钞级马力风格）

**字数**：100-300字
**结构**：

```
【痛点引入】XXX看不懂？别慌
【一句话价值】看完这篇你就超过90%的人
【引导看封面】详细知识点在上图
【收藏引导】建议收藏，用的时候翻出来
【标签】
```

**示例**：
> 人形机器人产业链太复杂看不懂？别慌💪
> 
> 看完这篇你就比90%的人懂！核心零件、龙头公司、投资逻辑都整理好了👆
> 
> 📌建议收藏，有进展翻出来对照看
> 
> #人形机器人 #投资 #小白入门

### 3. 极简型

**字数**：10-30字
**结构**：

```
【图片说明】XXX产业链全景图
【收藏引导】建议收藏
```

**示例**：
> 人形机器人产业链全景梳理，建议收藏👆

## 正文写作原则

### ✅ 要做的
- 控制在100字以内（极简型30字内）
- 引导用户看图片
- 包含"建议收藏"CTA
- 加相关标签（3-5个）

### ❌ 不要做
- 不要在正文重复图片内容
- 不要写长段落
- 不要展开解释（那是图片的事）
- 不要超过300字

---

## 常见引导语

### 引导看图片
- 一张图看懂XXX，建议收藏👆
- 详细内容见上图👆
- 知识点都在图里，建议保存
- 整理不易，建议收藏👆

### 收藏引导
- 建议收藏，用的时候翻出来
- 建议保存，投资前对照看
- 收藏起来慢慢看
- 干货满满，建议转发收藏

### 互动引导
- 你更看好哪个环节？评论区聊聊
- 有问题评论区留言
- 还想了解哪个行业？告诉我

## 配图逻辑

### 图片必须是内容主体

| 账号类型 | 图片内容 | 信息量 |
|---------|---------|--------|
| 产业链分析 | 产业链图谱（上中下游+公司） | 90% |
| 财经知识 | 知识卡片/清单/对比表 | 85% |
| 企业分析 | 公司结构图/商业模式图 | 80% |

### 正文与图片的分工

- **正文**：吸引点击、免责声明、引导看图、引导收藏
- **图片**：核心信息、数据、图表、清单、对比

## 对比：错误 vs 正确

### ❌ 错误（正文太长，重复图片内容）

> 人形机器人产业链很复杂，上游有减速器、电机、传感器。减速器占成本35%，主要有谐波减速器和RV减速器，绿的谐波是龙头。电机占成本25%，汇川技术是龙头。中游是本体制造，有优必选、宇树科技。下游应用有工业制造、商业服务、家庭消费...
> 
> （正文写了500字，图片反而没信息量）

### ✅ 正确（正文简短，引导看图）

> 【免责声明】个人学习复盘|非投资推荐
> 
> 一张图看懂人形机器人产业链，建议收藏👆
> 
> #产业链 #人形机器人 #行业研究

---

## 总结

**核心口诀**：

> 正文引流，图片留人
> 正文简短，图片干货
> 正文引导，图片教学

**字数配比**：
- 正文：5-10%
- 图片：90-95%

**不要试图在正文里讲完所有内容**，那是图片的事！
