# /dae-fly-style

大E滚雪球 × insummerwefly 融合风格复刻技能

## 快速使用

```
/dae-fly-style [主题/今日市场信息]
```

## 风格定位

融合两位大V的核心特征：
- **大E滚雪球**（4.4万粉）：日频逻辑记录、深度挖掘、文学性表达
- **insummerwefly**（38万粉）：长文深度分析、第一人称叙事、反审查创意表达

## 核心特征

| 维度 | 要求 |
|------|------|
| 标题 | 悬念/判断/反问，不用"一文秒懂"等教学标题 |
| 正文 | 长文分析，❶❷❸标记段落 |
| 人称 | ��一人称，有态度有判断 |
| 数据 | 精确到数字、百分比、点位 |
| 敏感词 | 🈶色、🐮市、📈、醉哒等创意替代 |
| 声明 | 每篇必加免责声明 |
| 标签 | #XXX[话题]# 格式，3-6个 |

## 文件说明

- `SKILL.md` — 完整风格分析与模板
- `SYSTEM_PROMPT.md` — 创作系统prompt
- `EXAMPLES.md` — 3个完整示例（日评/深度/哲理）
- `README.md` — 本文件

## 与其他风格的区别

- vs 钞级马力：不是科普教学，是投资者分享思考
- vs 青柠檬丸子：不是产业链研究，是带态度的市场判断
