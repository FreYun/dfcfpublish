# 搞钱小财迷 封面 & 内页生成 Skill

> **每次生图前必读此文件。不要凭记忆写 prompt。**

## 铁律

1. **每次生图前读此文件** — prompt 模板会更新，不要用记忆中的老版本
2. **不擅自改风格参数** — 保持视觉一致性
3. **先建议，研究部确认后再生图** — 用「封面建议」格式请示
4. **记录好用的 prompt** — 保存到 `memory/好用的图片prompt.md`
5. **尺寸规范**：竖图 1024×1536（3:4，小红书帖子），横图 1536×1024（3:2）
6. **模型使用 banana2** — 中文文字渲染更好
7. **中文可能乱码** — 人物/场景图有价值就保留，文字后期叠加

## Template A — 电影感复古剧照封面（主力封面）

> 深度科普长图的封面，营造「文艺 + 专业」调性

### 视觉定义

- **风格**：电影感复古剧照，胶片质感，自然光
- **人物参考**：欧美女性，复古气质（参考朱莉·德尔佩 Before Sunrise 系列氛围）
- **色调**：温暖胶片感，微粒感，偏暖黄/琥珀色调
- **构图**：人物占画面 40-60%，留出文字叠加空间
- **不出现**：真实亚洲面孔、品牌 logo、现代数码感

### 情绪与场景对照表

| 内容情绪 | 场景描述 | 色调倾向 |
|---------|---------|---------|
| 科普/解读 | 咖啡馆读报、书店翻书、窗边沉思 | 暖黄、琥珀 |
| 看好/积极 | 阳光街道漫步、露台微笑、花园 | 暖阳、金色 |
| 警示/避坑 | 雨天窗边、深色书房、暗调灯光 | 冷暖对比、偏暗 |
| 教程/实操 | 桌面笔记本、手写清单、图表特写 | 柔和自然光 |
| 思考/犹豫 | 站在路口、望向远方、倚墙沉思 | 柔焦、淡雅 |
| 热点/紧急 | 匆忙街景、报纸特写、时钟 | 对比度高、偏冷 |

### Prompt 模板

```
style: 电影感复古胶片风格，自然光，微颗粒质感，暖色调
content: {场景描述}，欧美女性，复古着装，{情绪动作}。画面{构图说明}，留出{位置}空间用于文字叠加。胶片质感，浅景深，色调偏{色调}。
```

### 封面文字叠加（后期处理）

- **标题**：衬线体或手写体，放在画面留白处
- **字号**：大而醒目，一眼能读
- **颜色**：白色/米色/深棕色（根据背景亮度选择）
- **位置**：上方 1/3 或下方 1/3

## Template B — 浅米色质感内页（长图内页）

> 3-4 页滑动长图的内容页

### 视觉定义

- **背景**：浅米色 / 奶茶色质感纸感底（#FFF5E6 ~ #F5E6D3）
- **排版**：分点用「数字 + emoji」标注，板块化
- **字体风格**：正文无衬线（黑体/苹方），标题加粗
- **装饰**：细线分隔、圆角卡片、淡色色块点缀
- **配色**：深棕文字 (#3D2B1F) + 金色强调 (#C5975B) + 浅色辅助

### 排版模板

```
┌──────────────────────────────┐
│  📌 小标题（加粗，金色下划线）  │
│                              │
│  1️⃣ 第一个要点              │
│     → 简短解释                │
│                              │
│  2️⃣ 第二个要点              │
│     → 简短解释                │
│                              │
│  3️⃣ 第三个要点              │
│     → 简短解释                │
│                              │
│  ─────────────────────────── │
│  ⚠️ 提醒/风险提示（灰色底色块）│
└──────────────────────────────┘
```

### Prompt 模板

```
style: 浅米色质感纸张背景，排版清晰，适合阅读的信息图
content: 浅米色(#FFF5E6)纸质感背景，{内容区域描述}。使用深棕色文字，金色作为强调色。分点排列，每点带 emoji 图标。底部有淡灰色卡片区域作为提醒框。整体风格干净、温暖、专业。
```

## Template C — 纯色短帖封面（日更概念用）

> 日更概念短帖的简洁封面

### 视觉定义

- **背景**：纯色或渐变色，柔和不刺眼
- **文字**：大号加粗标题居中，一眼可读
- **装饰**：极简，最多一个小 emoji 或图标
- **尺寸**：1024×1536（3:4 竖图）

### 配色方案

| 话题类型 | 背景色 | 文字色 |
|---------|--------|--------|
| 基金/ETF 科普 | 浅蓝 #E8F4FD | 深蓝 #1A365D |
| 黄金/贵金属 | 浅金 #FFF8E7 | 深棕 #5C3D2E |
| 股票概念 | 浅绿 #E8F5E9 | 深绿 #1B5E20 |
| 宏观经济 | 浅紫 #F3E5F5 | 深紫 #4A148C |
| 避坑/警示 | 浅橙 #FFF3E0 | 深橙 #E65100 |
| 通用/中性 | 浅米 #FFF5E6 | 深棕 #3D2B1F |

### Prompt 模板

```
style: 纯色简洁封面，适合小红书信息帖
content: {背景色}纯色背景，画面中央是大号加粗文字「{标题文字}」，{文字色}，字体简洁有力。底部小字「搞钱小财迷 💰」。整体极简、清爽、一目了然。
```

## 封面建议格式

> 生图前先用此格式向研究部请示：

```
📋 封面建议：
- 模板：{A/B/C}
- 内容情绪：{科普/看好/警示/教程/思考/热点}
- 场景/配色：{具体描述}
- 封面文字：「{标题文字}」
- 页数：{封面 + N页内页}
```

---

## 生图流程

1. 确定内容形式（深度长图 / 日更短帖）
2. 选择模板（A 封面 / B 内页 / C 短帖封面）
3. 根据内容情绪选择场景/配色
4. 填写「封面建议」格式 → 请示研究部
5. 研究部确认后，复制对应 prompt 模板 → 替换变量
6. 调用 `image-gen-mcp.generate_image(style, content)` — 模型 banana2
7. 检查生成结果，如有中文乱码，保留图片、后期叠加文字
8. 好用的 prompt 记录到 `memory/好用的图片prompt.md`
