# TMT 全景研究
📅 2026-04-04（更新版）

## 核心判断

**TMT 标配偏低配** — 短期受估值消化+关税冲击+风偏回落三重压制，AI算力基建景气仍在但市场需要更多盈利兑现。

**排序：通信（算力网络）> 传媒（安全边际）> 电子（景气确认但贵）> 计算机（估值消化中）**

推导链：AI Capex扩张→光模块/液冷需求确认→通信估值与盈利匹配度最佳；传媒PE底部+高股息→下行风险小；电子盈利高增验证但PE 62x仍贵→等估值消化；计算机AI应用商业化待兑现→回避。

**结论：TMT定价仍处于估值消化期。** 3月以来持续回调已释放部分风险，但关税不确定性叠加盈利兑现期（一季报窗口），短期仍需谨慎。半导体的4月初反弹值得关注。

**盈利验证（中际旭创2025年报，3/31披露）：**
- 营收382.4亿（YoY +60.3%）— EPS 9.80元
- 归母净利107.97亿（YoY +108.8%）
- 毛利率42.0%（2024: 33.8%，大幅扩张8.2pct）→ 供需偏紧确认
- 投资收益3.79亿 → 非主营占比低，盈利质量高
- **但PE 62.4x隐含约40%+增速** → 2026若增速放缓则有估值回调压力

**中兴通讯2025年报：**
- 营收1339亿（稳定），归母净利56.2亿
- PE 27.3x / PB 2.0 / 股息率1.9% → 估值合理

**需求侧：**
- AI算力：GPU服务器需求持续→CPU供应偏紧→涨价10-15%（验证需求强度）
- 国产替代：半导体设备持续受益（北方华创/中微/拓荆），"十五五"政策定调
- 存储：DRAM/NAND合约价持续上行，第三轮涨价周期延续

**供给侧：**
- 先进制程产能（5nm/3nm）被AI芯片挤占 → DSP等芯片交付周期拉长
- 光模块：800G→1.6T升级 ASP翻倍 → 量价双击逻辑最硬

**结论：电子景气处于扩张中期，盈利已验证（中际旭创净利翻倍）。** 但当前PE 49-62x（龙头）仍较高，需要一季报持续验证才能支撑估值。半导体指数4月初反弹是积极信号。

- 新易盛锁定2026年15亿元光芯片订单（vs 2025年1.4亿 → 10x）

**供给侧：需求增速 > 产能扩张 → 供不应求**
- DSP芯片：博通/Marvell双寡头，先进制程产能挤占

- 隔离器：亚微米精密制造壁垒 → 扩产周期长

**盈利验证：**

**结论：通信行业景气处于扩张加速阶段。** 光模块量价双击逻辑最硬（CAGR 40%），上游供应趋紧。PE 27-62x（中兴最合理，光模块龙头偏高但有增速支撑）。**估值与盈利匹配度在TMT四方向中最优。**

## 投资优先级

逻辑最硬：OFC2026验证光通信CAGR 40%，1.6T量价双击，上游供应趋紧。估值最匹配：中兴PE 27x+股息1.9%。
代表：中兴通讯、中际旭创、新易盛、天孚通信

安全边际逻辑：PE 15-17x历史底部 + 高股息4.6%-6% + 拥挤度极低。新品催化待兑现。
代表：三七互娱、分众传媒、完美世界

### 🥉 等待：电子（超跌后可关注）
景气确认但估值偏高。4月初半导体反弹是积极信号。等一季报+估值进一步消化。
代表：北方华创、中际旭创、立讯精密

### ⏸️ 回避：计算机（高估值AI概念）
PE 34-126x + 多数AI应用盈利未兑现。
