# TMT 全景研究
📅 2026-04-04

## 核心判断

**TMT 标配偏低配** — 短期受估值消化+利率微升+风偏回落三重压制，但AI算力基建景气未破。

**排序：通信（算力相关）> 电子（半导体设备+存储）> 传媒（游戏底部）> 计算机（估值消化中）**

推导链：AI Capex仍在扩张→光模块/PCB/存储需求验证→通信最受益；电子盈利兑现但估值高位→次优；传媒估值底部+产品周期→安全边际；计算机PE 50x+盈利待兑现→等待。

## 定价环境

**主导变量：估值消化（盈利未跟上前期涨幅）+ 风险偏好回落**

| 指标 | 数值 | 含义 |
|------|------|------|

| TMT vs 沪深300超额（2月至今） | -4.6% | 跑输大盘 |
| 半导体指数 | 1718，2月至今-11.3% | 调整最深 |
| 科创50 | 1256，2月至今-13.4% | 硬科技杀跌 |
| 创业板指 | 3150，2月至今-3.5% | 成长风偏弱 |
| 10Y国债 | 1.82%（3月初1.78%，+3.9bp） | 利率微升，但绝对水平极低 |
| SW电子PE(TTM) | 76.7x（近年均值54.3x） | 显著高于中枢 |

| SW通信PE(TTM) | 28.6x（均值22.0x） | 高于均值但合理 |
| SW传媒PE(TTM) | 25.6x（均值26.2x） | 略低于均值 |

**推导：**
1. 10Y国债1.82%绝对水平极低→DCF折现率低→理论上利好成长股估值→但利率3月以来微升3.9bp→边际上不再宽松→估值扩张受限
2. TMT超额大幅跑输（-4.6% vs沪深300）→前期涨幅过大后的均值回归→资金从高估值TMT向防守切换
3. 电子PE 76.7x远高于均值54.3x→即使盈利高增长，估值已隐含较多乐观预期→需要持续高增速才能消化

**结论：当前TMT定价主要由"估值消化"主导。** 利率端温和友好但不再边际宽松，盈利端AI相关仍强但市场需要看到更多兑现，风偏端受地缘+关税等外部因素压制。

## 四方向景气

### 🔬 电子：景气扩张中，但估值领先于盈利

**需求：AI算力主导（驱动力强） + 国产替代加速 + 消费电子温和复苏**

- AI算力引擎：云厂商Capex持续增长，GPU服务器需求激增导致CPU供应紧张（英特尔/AMD 3-4月涨价10-15%，交期从1-2周延至8-12周），充分验证算力产业链高景气
- 国产替代：BIS管制持续加码→国产设备/材料需求曲线右移→北方华创/中微/拓荆受益；"十五五"强调"抢占科技制高点"
- 消费电子：NAND Flash涨价→手机平均存储容量YoY+4.8%（被动升级），但关税不确定性压制出货

**供给：产能利用率回升，存储进入涨价周期**

- 存储芯片第三轮涨价周期延续（DRAM+NAND合约价持续上行）→多因素共振（AI服务器+手机升级+PC换机）
- 晶圆代工：先进制程（5nm/3nm）被AI芯片挤占→DSP等芯片交付周期大幅拉长→供给偏紧

**价格信号验证：**
- CPU涨价10-15% ✅ 供需偏紧
- 存储合约价持续上行 ✅ 涨价周期延续
- 光模块ASP: 800G→1.6T翻倍 ✅ 量价双击

**盈利验证（中际旭创2025年报）：**
- 营收382.4亿（YoY +60.3%）
- 归母净利108.0亿（YoY +108.8%）
- 毛利率42.0%（2024: 33.8%，大幅扩张）→供需偏紧确认
- 但PE 62.4x隐含约40%+增速→2026年若增速放缓则有估值压力

**结论：电子行业景气处于扩张中期。** AI算力+国产替代+存储涨价三引擎共振，龙头盈利高增长验证需求。但SW电子PE 76.7x远高于均值→估值已较充分反映乐观预期。
**驱动排序：AI算力 > 国产替代 > 消费电子**

### 💻 计算机：AI Infra景气高，但传统IT拖累+估值偏高

**需求：AI Infra投资强劲 + 信创节奏稳定 + AI应用商业化待兑现**

- AI Infra：深圳发布AI服务器产业链发展行动计划（2026-2028），光模块800G→1.6T/3.2T升级写入政策；算电协同首次写入政府工作报告→政策推力
- 信创：招标节奏稳定但未见显著加速，2.0向行业端渗透尚早期
- AI应用：国产大模型持续迭代（GLM-5/M2.7/Kimi K2.5），日均Token调用量140万亿（YoY +1000x），但收入兑现仍偏早期——多数AI应用在"用户增长但收入未兑现"阶段

**供给：竞争格局分化**
- AI服务器：浪潮/曙光受益于国内算力需求，但服务器代工利润率低（PS 0.7x/0.5x说明收入大但利润薄）
- SaaS/安全：门槛低→竞争激烈→即使需求增长也难传导为利润
- 数据库/EDA：寡头格局→定价权强

**盈利验证：**
- 浪潮信息PE 33.7x/PB 4.0x/PS 0.55x→典型"大收入薄利润"→收入增速高但利润弹性有限
- 科大讯飞PE 126.3x→盈利兑现远未到来

**结论：计算机行业景气分化明显。** AI Infra硬件端景气确定但利润薄，AI应用商业化仍在早期（PS估值为主），传统IT偏弱。PE 50.4x整体偏高。
**关键分化：AI Infra硬件（量增利薄）vs AI应用（概念先行，盈利待兑现）vs 传统IT（弱复苏）**

### 📶 通信：AI算力网络驱动，景气最确定

**需求：算力网络建设最强 + 运营商Capex结构性转移 + 卫星0→1**

- AI算力网络（当前最强引擎）：
  - OFC2026释放极强信号——Lumentum指引光通信行业规模2025→2030年从180亿→900亿美金（CAGR 40%）
  - InP芯片需求2026-2030年每年+85%
  - CPO 2027年商业化元年，OCS规模上修至40亿美金
  - 800G→1.6T升级=ASP翻倍→量价双击→通信板块当前最大增量逻辑
  - 国内外硅光晶圆厂加速扩产（Tower日本300mm扩4倍、苏州星钥光子50亿投资）→验证需求强劲

- 运营商：5G基站490.9万个，但传统通信Capex见顶→算力Capex对冲→结构比总量更重要
- 卫星：工信部推动太空算力产业发展，频率收费标准优化→0→1阶段，主题投资为主

**供给：需求增速>产能扩张速度→供不应求**
- DSP：博通/Marvell双寡头，台积电先进制程产能被AI芯片挤占→交付周期拉长
- EML：日本住友扩产保守→国产加速替代（索尔思/长光华芯）
- 隔离器：亚微米级精密制造壁垒→扩产周期长→隐形瓶颈
- 新易盛锁定2026年15亿元光芯片订单（vs 2025年1.4亿→10x增长）→量产加速

**价格信号：** 800G→1.6T光模块ASP跳升 ✅ 量价双击验证

**盈利验证：**
- 中兴通讯PE 27.3x/PB 2.0x/股息率1.9%→估值合理
- 中际旭创营收+60%、净利+109%、毛利率42%→景气高位

**结论：通信行业景气处于扩张加速阶段。** 光模块量价双击逻辑最硬（行业规模CAGR 40%），上游供应趋紧进一步支撑景气判断。PE 28.6x高于均值22x但考虑到盈利增速50%+→估值与盈利匹配度最好。
**结构分化：算力相关（光模块/液冷/IDC）>>> 传统通信 >>> 卫星（主题投资）**

### 🎬 传媒：估值底部，等待催化

**需求：游戏产品周期驱动 + 广告温和恢复 + AIGC降本增效**

- 游戏（产品驱动）：
  - 核心龙头PE回落至15x左右→历史底部分位
  - 成交拥挤度MA5处于过去1年0.8%分位→极低
  - 存量产品流水稳定：世纪华通《无尽冬日》、三七互娱海外SLG月流水700万美元
  - 新品管线：完美世界《异环》4/23上线（预约3000万+）、《洛克王国：世界》首日登顶双榜
  - 版号稳定发放→供给侧压制消除
  - 苹果中国区佣金30%→25%→增厚利润率

- 广告（GDP函数）：强顺周期，分众PE 16.8x/股息率6%→防守属性强
- AIGC：OpenAI关停Sora→行业从"技术炫技"转向"商业闭环"；国内OPC模式兴起

**供给：版号稳定+苹果降佣→利好头部**

**盈利验证：**
- 三七互娱PE 15.4x/PB 3.7x/股息率4.6%→极具安全边际
- 分众传媒PE 16.8x/股息率6%→类债属性

**结论：传媒行业景气处于底部修复阶段。** 游戏估值+拥挤度双低→向下空间有限，向上弹性看产品催化。
**特殊说明：传媒是产品/爆款驱动行业，供需框架解释力有限→当前更多是估值底部的安全边际逻辑**

## 投资优先级

### 🥇 首选：通信（算力相关）
逻辑最硬：OFC2026验证光通信CAGR 40%，1.6T量价双击，上游供应趋紧。估值最匹配：PE 28.6x vs 盈利增速50%+。
代表：中际旭创、新易盛、天孚通信、中兴通讯

### 🥈 次选：传媒（游戏+分众）
安全边际逻辑：PE 15-17x历史底部 + 高股息4.6%-6% + 拥挤度极低。催化待兑现。
代表：三七互娱、分众传媒、巨人网络

### 🥉 等待：电子
景气确认但估值过高：SW电子PE 76.7x vs 均值54.3x。等一季报+估值消化。
代表：北方华创、中际旭创

### ⏸️ 回避：计算机（高估值AI概念）
PE 50.4x + 多数AI应用盈利未兑现。例外：恒生电子（PE合理）。
