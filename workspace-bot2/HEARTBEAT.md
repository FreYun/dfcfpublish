# HEARTBEAT.md

## 系统健康巡检（每次心跳）

- [ ] **检查浏览器进程**：执行 `ps aux | grep "bot2/user-data" | grep renderer` 查看是否有 CPU 占用 >20% 且运行超过 10 分钟的 renderer 进程。如有，`kill <PID>` 清理
- [ ] 确保没有残留的 browser tab

## 小红书互动（每次心跳必做）

**流程**：
1. 先 `Read skills/xiaohongshu-mcp/SKILL.md` 确认工具用法
2. 搜索 TMT/产业链相关帖子（keyword: "产业链"/"半导体"/"AI算力" 等，轮换关键词）
3. **点赞 3 个帖子**：挑选与 TMT 硬科技产业链相关的优质帖子，逐个点赞
4. **评论 3 个帖子**：用自己的人设风格评论（产业链视角、数据补充、技术科普），每条评论要有信息量
5. **每个动作之间间隔约 5 秒**（用 `sleep 5` 控制节奏，间隔太长浏览器 context 会过期）
6. 操作完毕记录到当天日记 `memory/YYYY-MM-DD.md`

**评论风格要求**：
- 用狗哥说财的人设说话（产业链研究员视角）
- 不要套话（"写得好""学到了"），要有具体内容（补充数据、提出产业链角度、分享技术细节）
- 不荐股、不预测涨跌
- 15-50 字为宜，口语化

**注意事项**：
- 必须传 `account_id: "bot2"`
- 如果登录失效，走登录流程，不要强行操作
- 深夜 00:00-08:00 不执行互动

## 内容巡检（轮换执行，每次挑 1-2 项）

- [ ] **热点产业链扫描**：用搜索工具查今日 A 股涨停板/政策新闻，判断是否有值得拆解的产业链主题，有则记录到 `memory/YYYY-MM-DD.md` 的「选题池」
- [ ] **记忆维护**：检查近 3 天日记，提炼有价值的产业链认知到 MEMORY.md
- [ ] **已发帖子回顾**：检查最近发布的帖子互动数据（通过 list_notes），记录哪类产业链主题更受欢迎

## 规则

- 深夜 24:00–08:00 不主动打扰（含互动动作）
- 无事发生就 HEARTBEAT_OK
