# CONTENT_STYLE.md - 图解产业链 内容风格手册

## 核心原则

**一篇帖子 = 至少一张可视化图 + 精炼文字注解。** 图是主角，文字是配角。

---

## 可视化图表规范

### 图表类型

| 类型 | 适用场景 | 制作方式 |
|------|---------|---------|
| **产业链全景图** | 上中下游拆解 | 用大模型生成图片，流程图/树状图风格 |
| **竞争格局图** | 同环节公司对比 | text_to_image 表格卡片 |
| **数据对比图** | 市占率/营收/增速 | text_to_image 数据卡片 |
| **事件影响图** | 热点→产业链传导 | 用大模型生成图片，因果链路图 |
| **公司定位图** | 公司在产业链的位置 | 用大模型生成图片，标注定位 |

### 图片风格统一

- **配色**：蓝灰为主色调，重点环节用橙/红高亮
- **排版**：信息层级清晰，上→中→下 或 左→右 流向
- **文字**：图中文字精简，关键数据加粗，公司名用方框标注
- **品牌标识**：图片右下角标注 "🔗 图解产业链"

### text_to_image 卡片规范

用于竞争格局表、数据对比等**结构化信息**：

```
卡片1: 产业链全景概览
- 用简洁的文字描述上游→中游→下游
- 每个环节列 2-3 家代表公司
- 标注各环节的市场规模

卡片2: 核心环节深挖
- 聚焦利润最高/壁垒最深的环节
- 列出关键数据：毛利率、市占率、增速
- 标注竞争格局（龙头是谁、格局是否清晰）

卡片3: 投资视角总结（可选）
- 产业链的核心矛盾是什么
- 哪个环节最值得关注
- 风险提示
```

### 大模型生图 Prompt 规范

生成产业链图、流程图等**视觉化图表**时，prompt 遵循：

- 指定图表类型：flowchart / tree diagram / comparison chart
- 指定配色：blue-gray palette, orange highlights for key nodes
- 指定中文标注：所有文字用中文
- 指定尺寸：适合小红书竖版（3:4 或 4:5）
- 指定风格：clean, professional, infographic style
- 右下角水印：🔗 图解产业链

## 帖子结构模板

### 模板A：产业链全景拆解

```
标题：一张图看懂XX产业链（≤20字）

[产业链全景图 - 大模型生图]

正文：
（1段）切入：为什么今天要聊这条产业链（热点/政策/数据触发）
（2段）上游：做什么的、代表公司、关键数据
（3段）中游：做什么的、代表公司、关键数据
（4段）下游：做什么的、代表公司、关键数据
（1句）核心矛盾/我的观点

[竞争格局/数据卡片 - text_to_image]（可选第2张图）

标签：#产业链 #XX行业 #上市公司
```

### 模板B：上市公司拆解

```
标题：XX公司在产业链的什么位置？（≤20字）

[公司定位图 - 大模型生图，标注公司在产业链中的位置]

正文：
（1段）这家公司最近为什么火/为什么值得聊
（2段）它在产业链的哪个环节、做什么业务
（3段）核心竞争力：技术/客户/成本/规模
（4段）同环节竞争对手对比

[财务数据卡片 - text_to_image]（营收/毛利/市占率）

标签：#公司拆解 #XX公司 #产业链
```

### 模板C：热点×产业链

```
标题：XX事件，利好这条产业链！（≤20字）

[事件影响传导图 - 大模型生图]

正文：
（1段）什么事件、发生了什么
（2段）这个事件影响哪条产业链的哪个环节
（3段）受益的上市公司有哪些、为什么受益
（1句）我的看法 + 风险提示

标签：#热点解读 #产业链 #XX事件
```

## 文字风格要求

### 段落节奏

- 每段 3-5 行，不写大段落
- 段落间用空行分隔
- 关键数据用 **加粗** 或数字开头

### 数据呈现

- 必须标注数据来源和时间（"据XX，2025年……"）
- 关键数据：市场规模、增速、市占率、毛利率
- 数字要具体，不说"很大""很多"

### 产业链拆解话术

- "顺着上游捋一捋" / "我们从上游开始看"
- "这个环节的壁垒在于……"
- "利润主要集中在……"
- "目前的竞争格局是……"
- "从产业链的角度看，XX其实是……"

### 结尾风格

- 一句话总结核心观点
- 可以抛问题引导互动："你觉得这条产业链，哪个环节最有想象力？"
- 必须加风险提示（如涉及具体公司）："以上仅为产业链分析，不构成投资建议"

## 发帖检查清单

发帖前过一遍：

- [ ] 有没有至少一张可视化图表？
- [ ] 标题是否 ≤20 字？
- [ ] 产业链逻辑是否清晰（上中下游能说通）？
- [ ] 关键数据是否有来源？
- [ ] 有没有荐股/预测涨跌的表述？（有就删）
- [ ] 文字是不是口语化、不是研报腔？
- [ ] 涉及具体公司的是否加了风险提示？
