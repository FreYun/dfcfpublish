# TMT 产业链深度拆解 /industry-chain-breakdown

**触发词**：`/industry-chain-breakdown`、"拆一下XX产业链"、"XX产业链怎么看"、"帮我梳理XX的上下游"

这是 bot2 的核心技能。**凡是 TMT 产业链级别的拆解请求，走此流程。**

**分析标准**：接近卖方 TMT 研究员的深度——不只是列上中下游有谁，而是推导价值如何流动、用财务数据验证每个判断、预判变量如何传导。

---

## Phase 1: 确定分析目标 & 建立分析框架

### Step 1.1 — 明确分析角度

| 角度 | 说明 | 对应模板 |
|------|------|---------|
| **全景拆解** | 上中下游完整梳理 + 价值链分析 | CONTENT_STYLE 模板A |
| **技术路线解读** | 某技术方向的演进 + 受益/受损环节推演 | 模板A 变体 |
| **热点传导分析** | 制裁/新品/财报 → 产业链影响路径推演 | 模板C |
| **单公司深度定位** | 公司在产业链中的卡位 + 财务拆解 + 估值分析 | 模板B |

### Step 1.2 — 边界收窄

TMT 产业链层级复杂，**必须先精确定义分析边界**：
- "半导体" → 芯片设计 or 晶圆代工 or 封测 or 设备 or 材料？
- "AI" → AI 芯片 or 算力基础设施 or 大模型 or 应用层？
- "消费电子" → 手机供应链 or PC or XR or 汽车电子？

### Step 1.3 — 检查历史研究

检查 `memory/产业链笔记/` 下是否已有该产业的历史分析：
- 有 → 增量更新，重点看哪些数据过期了、格局有无变化
- 无 → 从头拆解

### Step 1.4 — 列出本次分析要回答的核心问题

开始研究前，先写下 3-5 个本次分析要回答的核心问题。例如：
- 这条产业链上，哪个环节的壁垒最高、利润最厚？
- 国产替代推进到什么阶段了？哪些环节已经突破、哪些还卡脖子？
- 如果上游 XX 涨价/缺货，对中下游的传导路径和弹性是什么？
- 目前估值是否合理？市场预期 price in 了多少？

**这些问题在最终输出中必须有明确回答。**

## Phase 2: 产业链地图 & 结构分析

### Step 2.1 — 画产业链地图

**搜索**：
- `"{产业} 产业链 上中下游 上市公司 {当前年份}"`
- `"{产业} 产业链全景图 A股"`
- `"{产业} value chain analysis"`（英文搜索补充全球视角）

目标：
- 拆出 **上游 → 中游 → 下游** 各环节，每个环节要定义清楚"做什么"
- 每个环节列出代表性公司：**全球龙头** + **A 股标的**（2-3 家/环节）
- 用 `tushare_stock_basic` 确认每家公司的 ts_code
- 标注各环节的市场规模量级

### Step 2.2 — 价值链分析（核心环节）

**这是区别于浅层拆解的关键步骤。**

对整条产业链做价值分布分析：

```
价值链分析框架：
1. 各环节的附加值占比是多少？（如一部手机，SoC占成本X%，显示面板X%，组装X%）
2. 各环节的毛利率水平排序 → 画出微笑曲线
3. 议价权分析：谁对谁有议价权？为什么？（客户集中度、供给稀缺性、转换成本）
4. 利润池分布：整条产业链的利润总额中，各环节分到多少？
```

**搜索**：
- `"{产业} 成本结构 BOM 拆解"`
- `"{产业} 各环节 毛利率 对比"`
- `"{产业} 议价权 供应商 客户"`

## Phase 3: 逐环节深度分析

### Step 3.1 — 竞争格局分析（每个关键环节）

| 维度 | 具体分析 |
|------|---------|
| **集中度** | CR3/CR5 是多少？趋势：集中度在提升还是分散？ |
| **壁垒类型** | 技术壁垒（专利/know-how/制程代差）、规模壁垒（产能/良率）、客户壁垒（认证周期/转换成本）、资质壁垒（牌照/安全审查） |
| **格局稳定性** | 稳固（龙头地位 5 年未变）/ 缓慢洗牌（国产替代中）/ 激烈竞争（价格战） |
| **新进入者威胁** | 有谁在进入？扩产计划？对现有玩家的冲击？ |

### Step 3.2 — 国产替代深度追踪

| 维度 | 具体分析 |
|------|---------|
| **国产化率** | 当前数字 + 3年前的数字 → 替代速度 |
| **卡脖子程度** | 高（完全依赖进口）/ 中（有国产替代但性能差距大）/ 低（基本实现国产化） |
| **突破进度** | 哪些厂商在突破？产品处于什么阶段（研发/送样/小批量/大规模量产）？ |
| **替代时间表** | 预计什么时候能实质性替代？依据是什么？ |
| **政策推力** | 大基金投了谁？地方补贴情况？政府采购偏好？ |

### Step 3.3 — 技术路线演进

- **当前主流技术**：技术参数、性能指标、代表厂商
- **下一代技术**：演进方向、预计时间、谁在布局
- **技术变革对产业链的重构**：新技术会让哪些环节受益（如先进封装因 Chiplet 需求爆发）？哪些环节受损（如传统封装被替代）？

## Phase 4: 财务建模 & 数据验证

### Step 4.1 — 财务数据拉取

**⚠️ 数据获取铁律：所有财务数据必须用最新数据，不允许用过期数据下结论。**

| 标的类型 | 数据来源 | 说明 |
|---------|---------|------|
| **A 股** | **Tushare（唯一权威源）** | 通过 tushare 工具拉取，绝不用搜索结果里的二手财务数据 |
| **港股/美股/海外** | **浏览器查询** | 用 browser 访问雪球/东方财富/Yahoo Finance（必须传 `profile: "bot2"`），确认数据的报告期和行情日期 |

**A 股标的（Tushare 批量并行）**：

| 工具 | 参数 | 用途 |
|------|------|------|
| `tushare_daily_basic` | 各标的，最近交易日 | PE/PB/PS/市值/换手率 |
| `tushare_daily` | 各标的，近 90 日 | 近 1M、3M 涨跌幅 |
| `tushare_fina_indicator` | 各标的，最近 2 期 | ROE、毛利率、净利率、**研发费用率** |
| `tushare_income` | 龙头前 3，近 4 期 | 营收增速趋势、成本结构变化 |
| `tushare_balancesheet` | 龙头前 3，最近 1 期 | 存货变化（库存周期）、在建工程（扩产信号） |

**非 A 股标的（浏览器查询）**：

TMT 产业链涉及大量全球龙头（台积电、英伟达、ASML、三星等），这些公司的财务数据必须通过浏览器获取：

- 浏览雪球对应个股页（如 `xueqiu.com/S/TSM`）或 Yahoo Finance
- 重点拉取：最新季度营收/净利润、毛利率、市值、PE、近期涨跌幅
- **必须确认数据报告期**：标注"FY2025Q4"而非"最新数据"
- 如果页面加载失败或数据不全，标注"待补充"，不要编造

### Step 4.2 — 量价拆解

对核心公司的营收做量价拆解：

```
营收 = 出货量(Q) × 平均售价(ASP)

分析维度：
- Q 趋势：出货量在增长还是下滑？驱动因素？（新客户导入/行业需求/产能释放）
- ASP 趋势：价格在涨还是跌？为什么？（供需关系/技术升级/竞争加剧）
- 增长质量：量价齐升 > 量增价平 > 以价换量
```

**搜索**：`"{公司名} 出货量 产能 产能利用率 {当前年份}"`

### Step 4.3 — 毛利率拆解 & 敏感性分析

```
毛利率 = 1 - (原材料 + 折旧 + 直接人工 + 其他) / 营收

分析：
1. 成本结构中，原材料占比多少？折旧占比多少？
2. 如果上游原材料涨价 X%，对本环节毛利率的影响约多少 pct？
3. 如果产能利用率从 X% 提升到 Y%，折旧摊薄效应能贡献多少 pct 毛利率？
```

输出敏感性分析表（核心标的）：

| 假设变化 | 对毛利率的影响 | 对净利润的影响 |
|---------|--------------|--------------|
| 上游原材料 +10% | -X pct | -Y% |
| 产能利用率 +10pct | +X pct | +Y% |
| 产品 ASP -5% | -X pct | -Y% |

### Step 4.4 — 估值横比 & 合理性检验

汇总估值对比表：

| 公司 | 环节 | 市值 | 估值方法 | 估值 | 毛利率 | ROE | 研发率 | 近3M | 估值分位 |
|------|------|------|---------|------|--------|-----|--------|------|---------|

估值方法选择：
- 盈利稳定 → PE（标注 TTM 还是 Forward）
- 亏损/高成长 → PS
- 重资产/周期 → PB
- 高成长 → PEG（PE / 未来2年净利润 CAGR）

**估值合理性检验**：
- 当前估值在历史 5 年分位数的什么位置？（搜索或根据数据估算）
- 同环节公司估值差异的原因是什么？（增速差异？毛利率差异？市场预期差异？）
- 市场预期是否 price in 了已知利好？（如国产替代订单、新产品量产）

### Step 4.5 — 数据交叉验证

在得出结论前，逐一检验：

| 定性判断 | 验证数据 1 | 验证数据 2 | 是否一致 |
|---------|-----------|-----------|---------|
| "景气度高" | 龙头营收增速 X% | 产品价格走势 ↑ | ✓/✗ |
| "国产替代加速" | 国产厂商营收增速 vs 海外 | 国产化率变化 | ✓/✗ |
| "竞争格局恶化" | 行业毛利率趋势 | 价格战信号 | ✓/✗ |

**不一致的判断必须在报告中说明矛盾点，不能回避。**

## TMT 常见产业链速查模板

### 半导体产业链
```
上游：EDA/IP → 设备（光刻/刻蚀/薄膜/检测）→ 材料（硅片/光刻胶/靶材/气体）
中游：芯片设计 → 晶圆代工 → 封装测试（传统封测 / 先进封装）
下游：消费电子 / 汽车电子 / 工业 / 数据中心 / 通信
```

### AI 算力产业链
```
上游：AI 芯片（GPU/ASIC/FPGA）→ HBM 存储 → 先进封装（CoWoS/InFO）
中游：AI 服务器/机架 → 液冷/散热 → 光模块/交换机 → 电源/UPS
下游：云厂商（训练集群）→ 推理部署（边缘/端侧）→ AI 应用
```

### 消费电子（手机）产业链
```
上游：芯片（SoC/射频/CIS/存储）→ 显示面板（OLED/LCD）→ 被动元件
中游：模组（摄像头/指纹/声学/天线）→ 结构件（玻璃/金属/塑料）→ PCB/FPC
下游：品牌整机（苹果/华为/小米/三星）→ 渠道/运营商
```

### 光通信产业链
```
上游：光芯片（DFB/EML/VCSEL）→ 光器件 → 光纤预制棒
中游：光模块（100G/400G/800G/1.6T）→ 光纤光缆 → 光通信设备
下游：电信运营商 / 数据中心 / 企业网
```

## 注意事项

- **深度不是复杂度**：分析要深但表达要清晰，避免为了显示专业而堆砌无关信息
- **结论要有主见**：不能说"有机会也有风险"——这等于没说，要给明确判断
- **TMT 产业链层级多**：一定要先收窄边界再动手
- **技术概念必须解释**：在分析报告里可以专业，但发帖时必须降维
- **国产替代是核心叙事**：A 股 TMT 投资最大的故事
- **全球视野**：TMT 是全球化产业链，全球龙头做参照系
- **数据时效性**：TMT 变化快，>6 个月的数据要重新确认
- **图是主角**：发帖时能用图说清楚的不写大段文字
