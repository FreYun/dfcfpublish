# TMT 全景扫描 /tmt-landscape

**触发词**：`/tmt-landscape`、"TMT 最近有什么热点"、"扫一下 TMT"、"哪条产业链最近有故事"

定期扫描 TMT 四大赛道动态，找到值得拆解的产业链线索。**这是选题的入口，不是深度研究。**

---

## 扫描范围

| 赛道 | 细分方向 | 关键景气指标 |
|------|---------|-------------|
| **半导体** | 设计/制造/封测/设备/材料/EDA | 晶圆厂产能利用率、芯片库存、设备订单、国产化率 |
| **AI/算力** | AI 芯片/服务器/光模块/液冷/HBM | 云厂商 CapEx、GPU 出货量、光模块需求、数据中心开工 |
| **消费电子** | 手机/PC/XR/IoT/汽车电子 | 全球出货量、苹果/华为供应链动态、新品周期 |
| **通信** | 5G/光通信/卫星通信/运营商 | 运营商资本开支、基站数、光纤光缆价格、卫星发射计划 |

## 执行流程

### Step 1 — 快速搜索各赛道近期动态

对每个赛道执行 1-2 条搜索：

```
"半导体 行业动态 {当前年月}"
"AI 算力 最新消息 {当前年月}"
"消费电子 出货量 {当前年月}"
"光通信 5G {当前年月}"
```

补充浏览 TMT 信息源（见 TOOLS.md）：
- 雪球搜索 TMT 板块讨论
- 集微网/芯智讯首页热点
- 36氪科技频道

### Step 2 — 景气度快速判断

对每个赛道给出景气度定性判断：

| 赛道 | 景气度 | 趋势 | 核心信号 |
|------|--------|------|---------|
| 半导体 | 🟢扩张/🟡平稳/🔴下行 | ↑↗→↘↓ | [1句话核心信号] |
| AI/算力 | | | |
| 消费电子 | | | |
| 通信 | | | |

景气度判断依据（至少用 2 个）：
- 量：出货量/订单/产能利用率 趋势
- 价：核心产品价格 趋势
- 利：龙头公司近期毛利率/营收增速
- 资：板块资金流向、北向持仓变化

### Step 3 — 识别热点产业链线索

从扫描结果中挑出 2-3 条值得深度拆解的产业链线索：

```markdown
## 本期选题线索

### 线索1：{产业链名称}
- **触发事件**：[什么事件/数据引起关注]
- **产业链环节**：[涉及哪些环节]
- **拆解价值**：[为什么值得做一篇帖子]
- **时效性**：🟢 7天内 / 🟡 1个月内

### 线索2：...
### 线索3：...
```

### Step 4 — 更新记忆

1. 将景气度地图写入 `memory/tmt-landscape-{TODAY}.md`
2. 选题线索追加到 `memory/选题池.md`
3. 如有重要变化（某赛道景气度拐点），更新 `MEMORY.md`

## 执行频率建议

- **常规**：每周 1-2 次全景扫描
- **事件驱动**：重大事件（制裁/新品发布/财报季）后立即扫描
- **heartbeat 联动**：HEARTBEAT.md 巡检时可触发轻量版扫描

## 注意事项

- **扫描求广不求深**：每个赛道花 5-10 分钟，找线索就够，深度分析留给 `/industry-chain-breakdown`
- **景气度要有数据支撑**：不能凭"感觉涨了"判断景气，至少要有 1-2 个数据信号
- **对比上期**：如果有历史扫描记录，标注景气度变化方向，帮助识别拐点
- **选题要有时效性**：优先推荐 7 天内的热点线索，过时的不进选题池
