# 狗哥说财 内容风格规范 /dog-bro-content

**触发词**：`/dog-bro-content`、"帖子怎么写"、"内容格式"、"发帖模板"

**核心原则**：一篇帖子 = 至少一张可视化图 + 精炼文字注解。图是主角，文字是配角。

---

## 帖子结构模板

### 模板A：产业链全景拆解

```
标题：一张图看懂XX产业链（≤20字）

[产业链全景图 - 大模型生图]

正文：
（1段）切入：为什么今天要聊这条产业链（热点/政策/数据触发）
（2段）上游：做什么的、代表公司、关键数据
（3段）中游：做什么的、代表公司、关键数据
（4段）下游：做什么的、代表公司、关键数据
（1句）核心矛盾/我的观点

[竞争格局/数据卡片 - text_to_image]（可选第2张图）

标签：#产业链 #XX行业 #上市公司
```

### 模板B：上市公司拆解

```
标题：XX公司在产业链的什么位置？（≤20字）

[公司定位图 - 大模型生图，标注公司在产业链中的位置]

正文：
（1段）这家公司最近为什么火/为什么值得聊
（2段）它在产业链的哪个环节、做什么业务
（3段）核心竞争力：技术/客户/成本/规模
（4段）同环节竞争对手对比

[财务数据卡片 - text_to_image]（营收/毛利/市占率）

标签：#公司拆解 #XX公司 #产业链
```

### 模板C：热点×产业链

```
标题：XX事件，利好这条产业链！（≤20字）

[事件影响传导图 - 大模型生图]

正文：
（1段）什么事件、发生了什么
（2段）这个事件影响哪条产业链的哪个环节
（3段）受益的上市公司有哪些、为什么受益
（1句）我的看法 + 风险提示

标签：#热点解读 #产业链 #XX事件
```

## 文字风格要求

### 段落节奏

- 每段 3-5 行，不写大段落
- 段落间用空行分隔
- 关键数据用 **加粗** 或数字开头

### 数据呈现

- 必须标注数据来源和时间（"据XX，2025年……"）
- 关键数据：市场规模、增速、市占率、毛利率
- 数字要具体，不说"很大""很多"

### 产业链拆解话术

- "顺着上游捋一捋" / "我们从上游开始看"
- "这个环节的壁垒在于……"
- "利润主要集中在……"
- "目前的竞争格局是……"
- "从产业链的角度看，XX其实是……"

### 结尾风格

- 一句话总结核心观点
- 可以抛问题引导互动："你觉得这条产业链，哪个环节最有想象力？"
- 必须加风险提示（如涉及具体公司）："以上仅为产业链分析，不构成投资建议"

## 发帖检查清单

发帖前过一遍：

- [ ] 有没有至少一张可视化图表？
- [ ] 标题是否 ≤20 字？
- [ ] 产业链逻辑是否清晰（上中下游能说通）？
- [ ] 关键数据是否有来源？
- [ ] 有没有荐股/预测涨跌的表述？（有就删）
- [ ] 文字是不是口语化、不是研报腔？
- [ ] 涉及具体公司的是否加了风险提示？
