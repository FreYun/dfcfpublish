# 青柠檬丸子风格复刻 Skill

基于小红书产业链研究博主「青柠檬丸子」的内容分析，一比一复刻她的发文风格。

## 博主画像

- **昵称**: 青柠檬丸子 (@416554487)
- **粉丝**: 10万+
- **定位**: 产业链/行业/企业研究博主
- **特点**: 一天吃透一个主题，产业链图谱可视化
- **签名**: 每日分享一个行业/一家公司 | 个人学习复盘|非投资推荐

## Skill 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 完整风格分析，包含标题公式、内容结构、话题关注点等 |
| `SYSTEM_PROMPT.md` | 可直接使用的系统prompt |
| `EXAMPLES.md` | 3个完整示例（人形机器人、蜜雪冰城、储能） |

## 快速使用

### 方式1：使用System Prompt

复制 `SYSTEM_PROMPT.md` 的内容作为系统prompt，然后输入主题：

```
主题：宠物经济产业链
```

### 方式2：直接引用Skill

```
请使用青柠檬丸子风格skill，生成一篇关于【跨境电商产业链】的小红书笔记。
```

## 核心风格要点

### ✅ 要做的
- 标题使用"一天吃透"格式
- 必须添加免责声明
- 内容涵盖上中下游
- 列出核心公司
- 包含市场规模数据
- 封面5102x6803px超长图

### ❌ 不要做
- 不要用主观判断
- 不要推荐买卖
- 不要用太多emoji
