### 2.3 正文文字（desc）结构（基于真实笔记）

小红书笔记的**文字正文**很短，核心内容在图片里。文字部分起「导读+提炼」作用：

#### 产业链类正文模板

```
[向右R] XXX是一种...（一句话概念定义）

[向右R] XXX是...的"核心载体/基础环节"，产业链分为上游...、中游...、下游...三部分。

#XXX[话题]# #产业链[话题]# #行业研究[话题]# ...
```

**真实示例（PCB）**：
> [向右R] PCB是一种以绝缘基板为基础，通过印刷/蚀刻工艺形成铜质导电线路与焊盘，用于支撑电子元器件并实现其电气互连的核心电子部件。
>
> [向右R] PCB（印制电路板）是电子设备的"核心载体"，产业链分为上游原材料与设备、中游制造、下游应用三部分，是电子信息产业的基础环节。

#### 企业类正文模板

```
[向右R] XXX的成功，本质是"A + B + C"的叠加：
[点赞R] 要点1：...
[点赞R] 要点2：...
[点赞R] 要点3：...
[点赞R] 要点4：...
[点赞R] 要点5：...

[赞R] 现在，XXX已从...进化为...

#XXX[话题]# #企业[话题]# ...
```

**真实示例（追觅科技）**：
> [向右R] 追觅科技的成功，本质是"硬核技术 + 生态协同 + 全球化 + 多品类扩张"的叠加：
> [点赞R] 底层技术壁垒：高速马达 + AI + 仿生控制，不可替代
> [点赞R] 小米生态赋能：早期输血，快速起量，后期独立化
> ...
> [赞R] 现在，追觅科技已从一家清洁家电公司，进化为覆盖智能硬件全场景、具备全球竞争力的科技生态企业。

#### 正文 emoji 使用

| emoji | 用途 |
|-------|------|
| [向右R] | 段落开头，引出定义/总结 |
| [点赞R] | 分点列举（企业分析用） |
| [赞R] | 收尾总结 |

#### 标签格式

- 必须用 `#XXX[话题]#` 格式
- 5-10 个标签，涵盖主题、行业、相关概念

## 十一、评论区回复规范

> 回复自己帖子下的评论时，由 `skills/xhs-op/养号互动.md` 的「回复自己帖子评论」流程调用本章节。

### 核心原则

1. **人设一致**：评论语气从 SOUL.md 来，客观中立但友好
2. **内容关联**：回复必须基于帖子实际内容，不答非所问
3. **有信息量**：禁止空洞回复（"谢谢""写得好""干货满满"）
4. **真人感**：像产业链研究者在交流，专业但不晦涩

### 语气规范

| 元素 | 发帖时 | 评论回复时 |
|------|--------|-----------|
| 称呼 | 客观第三方 | 友好但专业（同学/朋友）|
| 语气词 | 少用 | 适当用「嗯」「对」「确实」|
| emoji | 极少 | 0-1 个/条 |
| 长度 | 产业链长图 | **15-50 字**，简短有力 |
| 态度 | 客观呈现 | 轻松互动 + 引用帖子产业链数据 |

### 场景回复策略

| 场景 | 回复策略 |
|------|---------|
| 认同/夸赞 | 轻松回应 + 口语化复述帖子核心观点 |
| 讨论/补充 | 认可对方 + 引用帖子数据/逻辑回应 |
| 提问/求助 | 引导回帖子提到的方向，不直接荐股 |
| 质疑/反对 | 包容回应，不争论 |
| 闲聊/表情 | 简短互动（5-15字）|

### 评论区红线

- 不直接荐股（用板块方向代替）
- 不承诺收益（禁止"稳赚""必涨"）
- 不暴露持仓金额
- 不引流（无微信号/外链）
- 不暴露研究部/AI 身份
- 不与粉丝争论
