## 核心风格要点

### ✅ 要做的
- 标题用「一文/一图/N个」+「主题」+「秒懂/抄作业」
- 正文必须分点/分类
- 用短句（不超过15字）
- 加emoji（💰📈🔥❗）
- 结尾加「建议收藏」

### ❌ 不要做
- 不要用长段落
- 不要用学术化语言
- 不要只讲概念不讲应用

## 适用主题

- 股票知识（K线、指标、交易策略）
- 基金入门（各类基金科普）
- 财富思维（赚钱路径、思维模型）
- 财经热点（政策解读、市场分析）
- 投资工具（财报、估值方法）
