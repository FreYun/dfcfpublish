## 一、标题公式

### 1.1 融合标题风格

两位博主标题风格截然不同，需要融合使用：

| 类型 | 公式 | 示例来源 |
|------|------|----------|
| **日记记录型** | 「日期」+「逻辑记录/思路」+「！」 | 大E: "3月9日 逻辑记录！" |
| **主题深挖型** | 「主题」+「走到哪里了？/怎么看？」 | 大E: "当前电网设备走到哪里了？" |
| **悬念反问型** | 「现象/事件」+「要XXX了？/还是XXX？」 | fly: "击穿心理防线，牛市要结束？" |
| **判断宣言型** | 「主语」+「判断动词」+「结论」 | fly: "国家队的操盘思路在悄然变化" |
| **感叹惊讶型** | 「时间/程度词」+「事实」+「太XXX了」 | fly: "才一年，中国机器人进步太大了" |
| **引用金句型** | 「诗句/名言」变体 + 新解读 | fly: "穷游、富游，都不如少年游" |
| **引号概念型** | 「"新造词"」+「分析/发力/来了？」 | fly: ""酱香科技"发力，消费春天要来？" |
| **事件叙事型** | 「N万粉」+「博主/账号」+「事件」 | 大E: "9万粉基金博主跑路后续…" |

### 1.2 标题核心原则

1. **不用数量词堆砌**（区别于钞级马力）
2. **不用"秒懂""抄作业"**（区别于科普风格）
3. **像朋友在聊天时说的第一句话** — 自然、有悬念、有态度
4. **可以用问号，但必须带有自己的判断倾向**
5. **标题字数控制在8-18字**，短促有力
6. **emoji极少用在标题**，最多用🧧等场景emoji

### 1.3 标题禁忌

- ❌ "一文读懂XXX"
- ❌ "N个XXX你必须知道"
- ❌ 标题塞满emoji
- ❌ "小白入门"、"建议收藏"
- ❌ 教育/科普口吻

## 二、正文结构

### 2.1 核心结构模板

```
【开头钩子】（1-3句）
  — 从当天事件/现象/数据切入
  — 直接亮出自己的判断或态度
  — 例: "坐不住了，连续两天熔断后…"
    ↓
【分析主体】（3-8段）
  — 用 [一R][二R][三R] 或 [打卡R] 标记大段落
  — 每段一个论点，数据+逻辑+推演
  — 穿插个人观点和市场解读
  — 必要时使用反审查表达
    ↓
【核心判断/结论】（1-2句）
  — 明确给出方向性判断
  — 例: "快则周二，慢则周三"
    ↓
【附加信息】（可选，0-2段）
  — 当天其他重要消息的简评
  — 用 [加一R] 标记
    ↓
【声明/免责】（固定格式）
  — "声明：个人观点和思考不构成投资决策建议。
     没有群，不做投资咨询，所有决策均应视自身
     风险承受能力以及预期目标来做出。"
    ↓
【标签区】
  — 3-5个 #XXX[话题]# 格式标签
```

### 2.2 内容类型分布

| 类型 | 占比 | 特征 |
|------|------|------|
| **市场日评/逻辑记录** | 40% | 当日行情分析+核心板块逻辑+方向判断 |
| **主题深度分析** | 25% | 单一主题（板块/个股/宏观）深入推演 |
| **热点事件解读** | 20% | 突发事件的市场影响分析+应对策略 |
| **生活哲理/情感** | 10% | 从生活场景切入，映射投资/人生道理 |
| **布局策略** | 5% | 中长期投资方向分析+入场时机判断 |

### 2.3 段落格式要求

1. **段间空行**：每段之间用空行（或\t缩进）分隔，呼吸感强
2. **段落长度**：每段3-6句，不写超长段
3. **缩进风格**：正文段落开头用Tab缩进（insummerwefly特色）
4. **标记符号**：大段落用 `[一R]` `[二R]` `[三R]` `[打卡R]` `[加一R]` 标记
5. **总字数**：正文500-1200字，适合深度阅读但不冗长

## 三、行文风格

### 3.1 语言风格DNA

```
insummerwefly的语言 = 金融专业 × 口语化 × 第一人称 × 自信判断
大E的语言 = 深度逻辑 × 文学性 × 克制含蓄 × 暗线铺垫
融合 = 专业分析 × 自然口语 × 有态度的判断 × 偶尔文学性表达
```

### 3.2 必须遵循的语感规则

1. **第一人称叙事**："我的判断"、"在我看来"、"跟大家聊聊"
2. **口语化但不低幼**：
   - ✅ "这保密工作做的也太好了"
   - ✅ "别嫌少"
   - ✅ "平心而论，这已经好于普遍心理预期了"
   - ❌ "秒懂"、"抄作业"、"小白必看"
3. **自信判断+谦逊声明**：
   - 正文里大胆判断方向
   - 结尾固定加免责声明
4. **不教育读者**：
   - ❌ "大家一定要记住"
   - ✅ "值得注意一点"
   - ✅ "有个细节很有意思"
5. **用数据说话**：
   - "100万亿韩元≈5300亿RMB"
   - "韩国股市总市值19万亿RMB，占比约3%"
   - 具体到数字、日期、百分比
6. **敢下结论**：
   - "先说我的判断：这是一轮错杀"
   - "快则周二，慢则周三"
   - "5400点判断逻辑：1、…… 2、……"

### 3.3 反审查创意表达（insummerwefly特色）

这是insummerwefly的标志性写法，在涉及敏感词时使用替代表达：

| 原词 | 替代写法 | 示例 |
|------|----------|------|
| 有色金属 | 🈶色 / 有*色* / 有S | "🈶色继周五绝地反击后…" |
| 牛市 | 🐮市 / 🐮 | "🐮市就这样结束了？" |
| 涨/上涨 | 📈 | "今天继续📈稳住了情绪" |
| 钱 | 💰 | "挣认知范围内的💰" |
| 监管 | 监G | "没有监G权" |
| 量化 | QW / QH | "A里QW负责人离职" |
| 基金 | 基J | "新发基J建仓接力" |
| 减持 | - / 负 | "短期不会再－" |
| 投资 | 投Z | "金融投Z风险管理" |
| 信号 | 📶 | ""救市"信号📶发出后" |
| 抄��� | 抄D | "关于周五抄D详见…" |
| 云 | ☁ | "背靠A里☁" |
| 最大 | 醉哒 | "才是公司醉哒的问题" |
| 最强 | 醉强 | "醉强推手就是🈶色" |

**规则**：
- 只在涉及可能被平台审核的金融术语时使用
- 不要每个词都替换，只替换高风险词
- 要自然融入句子，读者能秒懂含义
- 同一篇文章中同一个词的替换方式要统一

### 3.4 emoji使用原则

| 使用场景 | emoji | 频率 |
|----------|-------|------|
| 段落标记 | [一R] [二R] [三R] [打卡R] [加一R] | 每篇2-5个 |
| 方向/涨跌 | 📈📉 | 文中自然出现 |
| 反审查替代 | 🈶🐮💰☁📶 | 按需使用 |
| 表情反应 | [笑哭R] [汗颜R] [doge] | 点缀，每篇1-3个 |
| 重要强调 | ❗️ | 偶尔 |
| 场景emoji | 🧧等 | 极少 |

**禁忌**：
- ❌ 不用💪🔥📌📚这类教学风emoji
- ❌ 不用✅❌1️⃣2️⃣3️⃣这类清单emoji
- ❌ emoji不是装饰，是功能性替代或语气补充

### 5.1 市场日评/逻辑记录

```
【标题】X月X日 逻辑记录！/ X月X日 [市场关键词]

【正文】
[开头] 直接切入今日最重要的市场变化（1-2句）

[一R] 第一个核心逻辑
  — 数据+事实描述
  — 逻辑推演：因为A → 所以B → 导致C

[二R] 第二个关注点
  — 板块/个股表现
  — 资金面分析
  — 方向判断

[打卡R] 核心结论/操作思路
  — 明确方向："XX仍看XXXX点位"
  — 或者给时间节点："快则X，慢则X"

[加一R] 今日花絮/附加信息（可选）
  — 某条消息的简短点评

声明：个人观点和思考不构成投资决策建议。没有群，不做投资咨询，所有决策均应视自身风险承受能力以及预期目标来做出。

#行情[话题]# #基金[话题]# #理财[话题]# #我的理财日记[话题]#
```

### 5.2 主题深度分析

```
【标题】[主题]，[判断/趋势/反问]？

【正文】
[开头钩子] 从一个数据/现象/新闻切入（2-3句）
  — 建立共鸣或制造悬念
  — 例："坐不住了，连续两天熔断后…"

[核心判断] 先说结论
  — "先说我的判断：……"
  — 不藏着掖着，结论先行

[一R] 分析维度一
  — 数据支撑
  — 逻辑链条
  — 与其他板块/市场的联动

[二R] 分析维度二
  — 同上结构
  — 可以加入历史对比

[三R] 分析维度三（可选）
  — 风险提示或反面论证

[打卡R] 总结展望
  — 重申核心判断
  — 给出具体点位/时间/条件

声明：文章仅记录个人思考，个人观点分享不构成投资决策建议。没有群，不做投资咨询，所有决策均应视自身风险承受能力以及预期目标来做出。

#[相关话题标签]
```

