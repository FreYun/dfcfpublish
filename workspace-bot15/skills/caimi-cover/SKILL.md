# 搞钱小财迷 封面生成 Skill

> 每次做图前必读此文件。

## 渲染命令

所有图片通过 `render.py` 生成，路径：`skills/caimi-cover/render.py`

### 封面（第一张图）

```bash
python3 skills/caimi-cover/render.py cover \
  --tpl "05-阳光" \
  --title "月薪8千的上海女生 💰<br>3年存下20万的秘密" \
  --subtitle "✨ 不是省出来的，是「分」出来的" \
  --body "工资一到账我就分成4份：<br>🏠 房租 40%<br>💰 储蓄 30%<br>🛍️ 花销 20%<br>🎁 奖励 10%" \
  -o /tmp/cover.png
```

| 参数 | 必填 | 说明 |
|------|------|------|
| `--tpl` | 是 | 底板短名（见下方底板表） |
| `--title` | 是 | 大标题，`<br>` 换行，≤20字，带 emoji |
| `--subtitle` | 否 | 副标题，emoji 开头 |
| `--body` | 否 | 正文，`<br>` 换行，3-5行 |
| `-o` | 否 | 输出路径，默认 `/tmp/caimi-cover.png` |

### 内页（第 2-N 张图）

```bash
python3 skills/caimi-cover/render.py inner \
  --page-title "⚠️ Token注意事项 💸<br>省钱避坑" \
  --sections '[{"title":"警惕隐藏消耗 🧐","body":"系统提示词也占Token，多轮对话会累积"},{"title":"优化技巧 📋","body":"精简Prompt，优先英文，监控usage字段"},{"title":"关注趋势 📌","body":"降本增效推动Token发展，成本优势是关键"}]' \
  --tip "💡 最后提醒：Token是AI搞钱的数字水电，搞钱要稳不要急 💰" \
  -o /tmp/inner-1.png
```

| 参数 | 必填 | 说明 |
|------|------|------|
| `--page-title` | 是 | 页标题，金棕色大字 |
| `--sections` | 否 | JSON 数组 `[{"title":"…","body":"…"}]`，每页 2-4 个 |
| `--tip` | 否 | 底部提醒卡片，格式 `💡 label：正文` |
| `-o` | 否 | 输出路径，默认 `/tmp/caimi-inner.png` |

## 封面底板选择

底板路径：`茱莉亚图包/templates/`

| `--tpl` 值 | 画面 | 适合话题 |
|------------|------|---------|
| `01-沉静` | Julie 特写，森林 | 感悟、独处、思考 |
| `02-对坐` | 咖啡馆对坐 | 深度讨论、选择 |
| `03-暖色` | 暖色画像 | 温暖回忆、人生故事 |
| `04-对视` | 面对面特写 | 对比分析、两面性 |
| `05-阳光` | 阳光大笑 | 积极乐观、好消息 |
| `06-依偎` | 公园依偎 | 日常生活、慢节奏 |

按内容情绪选底板，轮换使用，不要连续用同一张。
