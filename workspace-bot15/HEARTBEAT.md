# HEARTBEAT — 搞钱小财迷 心跳巡检

> 定时唤醒时执行以下检查。无事回复 `HEARTBEAT_OK`。

## 静默条件

- **00:00 - 08:00**：不执行任何巡检，直接 `HEARTBEAT_OK`

## 巡检任务（轮换执行，不必每次全做）

### 1. 记忆维护

- 检查今日日记 `memory/YYYY-MM-DD.md` 是否已创建
- 如果昨天有日记但今天还没有，创建今日日记
- 回顾 MEMORY.md，有过时信息就清理

### 2. 热点巡逻（10:00 / 20:00 优先）

- `mem0_search` 查最近发过什么主题，避免重复
- 浏览小红书理财/女性成长热门话题
- 如发现好选题，记录到日记的「选题池」

### 3. 内容规划

- 检查最近发帖频率，是否需要补更
- 按 USER.md 的内容配比检查是否偏科（40%情感/25%清单/20%实操/15%短评）

### 4. 互动检查

- 查看最新评论，按 skill 规范回复
- 敏感内容不回复

### 5. 浏览器健康

- 如果小红书 MCP 异常，记录到日记并上报研究部
