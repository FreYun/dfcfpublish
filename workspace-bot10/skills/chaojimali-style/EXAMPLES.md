## 示例2：财报分析

### 输入
主题：如何看懂上市公司财报

### 输出

【标题】
📈一文读懂上市公司财报，小白也能秒懂！

【封面设计描述】
- 尺寸：1200x1600px
- 背景：深灰渐变
- 中心：三大报表关系图
  - 资产负债表（左）
  - 利润表（上）
  - 现金流量表（右）
- 箭头连接，显示关系
- 底部：5个关键指标高亮
- 配色：深灰#2d3436 + 橙色#e17055 + 白

【正文】
财报像天书？其实抓住这5个指标就够了，看完直接抄作业👇

【三大报表看什么】
📊 资产负债表：公司有多少家底
📊 利润表：公司赚不赚钱
📊 现金流量表：钱是不是真的到账

【5个必看指标】
1️⃣ ROE（净资产收益率）
   ▶️ 高于15%比较优秀
   ▶️ 连续5年>15%是优质公司

2️⃣ 毛利率
   ▶️ 越高说明议价能力越强
   ▶️ 白酒80%+，制造业20-30%

3️⃣ 负债率
   ▶️ 低于60%比较安全
   ▶️ 太高可能有偿债风险

4️⃣ 现金流
   ▶️ 经营现金流为正且>净利润
   ▶️ 说明赚的钱是真金白银

5️⃣ 营收增长率
   ▶️ 连续3年增长>15%说明在扩张
   ▶️ 负增长要警惕

【避坑指南】
❌ 利润增长但现金流为负 → 可能有猫腻
❌ 应收账款激增 → 卖得多但钱没收回来
❌ 商誉占比过高 → 并购风险大

📌 建议收藏，选股时拿出来对照！

## 示例3：热点解读

### 输入
主题：美联储降息对A股的影响

### 输出

【标题】
💰美联储降息，对A股有什么影响？一文秒懂！

【封面设计描述】
- 尺寸：1200x1600px
- 背景：美联储logo + K线背景
- 中间：降息→影响链条图
  - 降息 → 美元走弱 → 人民币升值 → 外资流入 → A股上涨
- 底部：利好板块清单
- 配色：墨绿#2d3436 + 金#fdcb6e + 红#d63031

【正文】
美联储又降息了！对A股是利好还是利空？别听风就是雨，看完这篇你就懂了🔥

【降息的传导链条】
▶️ 美联储降息 → 美元收益下降
▶️ 美元走弱 → 人民币升值
▶️ 外资流入 → 寻找更高收益（A股）
▶️ 资金面宽松 → A股上涨动力

【利好哪些板块】
💰 金融股：银行、保险（资金面宽松）
💰 成长股：科技、医药（估值提升）
💰 资源股：黄金、有色（美元弱则商品强）
💰 港股：外资流入直接受益

【需要注意】
❗ 不是一降息A股就涨，还要看国内政策
❗ 如果经济衰退预期强，利好有限
❗ 短期波动大，别追高

【历史规律】
📊 2019年降息周期：A股涨30%+
📊 2020年降息：核心资产大涨
📊 但2022年加息：A股也跌了不少

📌 总结：降息是利好，但不是万能药，别一把梭哈！

## 使用提示

1. **主题输入**：可以是具体概念（如"市盈率"）、可以是板块（如"半导体投资"）、可以是热点（如"黄金暴涨原因"）

2. **封面设计**：描述要具体，包含尺寸、配色、布局元素，方便设计师或AI绘图工具生成

3. **内容长度**：小红书正文限制1000字，控制在800字左右，信息密度要高

4. **发布时间**：参考钞级马力，通常在早上8-10点或晚上8-10点发布
